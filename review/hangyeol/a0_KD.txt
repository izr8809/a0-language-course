# KD-1: 나 한결이에요!

## sim1: 첫 만남 (First Meeting)

AI: 안녕하세요! 저는 한결이에요!
User: 아, 네. 반가워요. / 안녕하세요!
AI: 이름이 뭐예요?
User: ___이에요!
AI: 같은 수업이에요? 진짜요?
User: 네! / 그런 것 같아요.
AI: 대박! 잘 부탁해요!
User: 저도 잘 부탁해요! / 네, 반가워요!
## sim2: 같이 해요 (Let's Do It Together)

AI: 같이 팀 할래요?
User: 저랑요? / 좋아요!
AI: 네! 저 열심히 할게요!
User: 기대돼요! / 좋아요!
User: 왜 저한테 먼저 말했어요?
AI: 그냥 좋아 보여서요!
User: 진짜요? / 고마워요.
AI: 진짜요! 같이 잘해 봐요!
## sim3: 약속 (Promise)

AI: 내일 와요?
User: 네, 가요! / 당연하죠!
AI: 내일 기대해요!
User: 뭘요? / 뭐예요?
AI: 저요! 재미있는 거 준비할게요!
User: 뭐예요? / 기대돼요!
AI: 내일 봐요! 약속해요!
User: 네! / 내일 봐요!

# KD-2: 왜 그래?

## sim1: 뭔가 이상해 (Something's Off)

AI: 오늘 왜 그래요? 힘들어요?
User: 괜찮아요. / 아니에요.
AI: 안 괜찮아 보여요.
User: 진짜 괜찮아요. / 걱정 마요.
AI: 말해 봐요. 들어줄게요.
User: 별거 아니에요. / 그냥요.
AI: 별거 아니어도 돼요. 말해 봐요.
User: …좀 힘들었어요. / 좀 피곤했어요.
## sim2: 기운 내요 (Cheer Up)

AI: 많이 걱정했어요. 진짜로요.
User: 고마워요. / 미안해요.
AI: 미안해하지 마요! 친구잖아요!
User: 맞아요. / 그렇죠.
AI: 이거 먹어요! 맛있어요!
User: 뭐예요? / 이거요?
AI: 떡볶이요! 기운 날거에요!
User: 고마워요! / 먹을게요!
## sim3: 같이 가자 (Let's Go Together)

AI: 이제 기운 나요?
User: 네, 괜찮아요. / 고마워요.
AI: 다행이에요! 이제 집에 가요?
User: 네, 집에 가요. / 네, 가요.
AI: 같이 가요! 제가 같이 갈게요!
User: 혼자 갈게요. / 괜찮아요.
AI: 안 돼요! 같이 가요!
User: 알겠어요! / 네, 가요!

# KD-3: 나 좋아해?

## sim1: 뭔가 달라 (Something's Different)

AI: 있잖아요… 할 말 있어요.
User: 뭐예요? / 왜 그래요?
AI: 좀 이따 말해도 돼요?
User: 네, 기다릴게요. / 네, 괜찮아요.
User: 긴장돼요? / 괜찮아요?
AI: 많이요. 이런 적 처음이에요.
User: 괜찮아요. 말해 봐요. / 천천히 해요.
AI: 네. 잠깐만요.
## sim2: 한결식 고백 (His Way of Saying It)

AI: 좋아해요!
User: …네? / 뭐라고요?
AI: 좋아한다고요! 진짜요!
User: 진짜요? / 정말요?
AI: 저 어때요?
User: 저도 좋아해요. / 생각해 볼게요.
AI: 진짜요?!
User: 네! / 당연하죠!
## sim3: 우리 (Us)

AI: 이제 반말 하자. 우리 사이에 왜 존댓말이야!
User: 좋아! / 그래, 하자!
AI: 우리 사귀는 거지?
User: 응, 당연하지! / 응, 맞아!
AI: 손 줘!
User: 왜? / 뭐 하게?
AI: 잡을게! 놓지 마!
User: 안 놓아! / 알겠어!

# KD-4: 미안해, 기다려

## sim1: 다퉜어 (We Fought)

User: 그냥 가요. / 가세요.
AI: 미안해요. 진짜 미안해요.
User: 가요! / 싫어요!
AI: 가기 싫어요.
User: 그만해요. / 힘들어요.
AI: 안 돼요.
User: 한결 씨. / 제발요.
AI: 한 번만요. 잘할게요.
## sim2: 혼자 (Alone)

User: 잘 지냈어요? / 괜찮았어요?
AI: 힘들었어요.
User: 진짜요? / 저도요.
AI: 많이 보고 싶었어요.
User: 저도요. / 많이요.
AI: 진짜요? 좋아요.
User: 우리 만나요. / 지금 얘기해요.
AI: 네, 만나요.
## sim3: 달려왔어 (He Ran Back)

AI: 나야! 뛰어왔어!
User: …왔어? / 한결아?
AI: 미안해. 진짜 미안해.
User: 나도 미안해. / 괜찮아.
AI: 다시 만나자! 잘할게!
User: 진짜? / 약속해.
AI: 약속! 이번엔 진짜!
User: 알겠어! / 믿어!
