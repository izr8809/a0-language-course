# CR-1: 저도 오늘 첫날이에요!

## sim1: 출근

AI: 안녕하세요! 저는 오늘 첫날이에요.
User: 네, 저도 첫날이에요!
AI: 아, 진짜요? 다행이에요! 우리 같이 열심히 해요!
User: 잘 부탁해요! / 네, 파이팅!
AI: 저는 한결이에요. 이름이 뭐예요?
User: 저는 ___예요. / ___라고 해요.
AI: 만나서 반가워요! 오늘 즐겁게 일해요.
User: 저도 반가워요! / 네, 좋아요!
## sim2: 오전

AI: 이 기계 알아요? 너무 어려워요.
User: 저도 몰라요. / 제가 알려 줄게요!
AI: 고마워요! 저도 처음이라서 어려워요.
User: 같이 배워요! / 천천히 해요.
AI: 커피 잘 만들어요?
User: 아니요, 처음이에요. / 조금 만들어요.
AI: 저도 처음이에요! 같이 연습해요.
User: 좋아요! / 네, 좋아요.
## sim3: 오픈

AI: 이제 오픈 시간이에요. 조금 떨려요.
User: 저도 많이 떨려요. / 저는 괜찮아요!
AI: 우리 같이 잘할 거예요!
User: 네, 파이팅!
AI: 손님 와요! 빨리 준비해요!
User: 네!
AI: 우리 진짜 잘했어요! 다행이에요.
User: 다행이에요! / 네, 잘했어요!

# CR-2: 잠깐, 제가 실수했어요...

## sim1: 피크 타임

AI: 오늘 손님이 진짜 많아요!
User: 빨리 일해요! / 너무 바빠요!
AI: 네! 제가 빨리 할게요!
User: 좋아요! / 파이팅!
AI: 앗, 제가 실수했어요.
User: 괜찮아요? / 무슨 일이에요?
AI: 미안해요. 이거 제가 다시 할까요?
User: 제가 할게요! / 같이 해요!
## sim2: 복구

AI: 도와줘서 정말 고마워요!
User: 아니에요. / 같이 힘내요!
AI: 네! 정말 다행이에요. 고마워요.
User: 이제 괜찮아요. / 수고했어요!
AI: 이제 거의 끝났어요. 조금만 더 파이팅!
User: 다 끝났어요! / 야호!
AI: 우리 오늘 진짜 열심히 했어요!
User: 수고했어요! / 진짜 수고했어요!
## sim3: 피크 후

AI: 아까 진짜 고마워요. 같이 일해서 정말 다행이에요.
User: 아니에요! / 제가 더 고마워요!
AI: 아니에요, 진짜 최고예요!
User: 우리 같이 잘했어요! / 칭찬 고마워요!
AI: 다리 많이 아파요? 우리 잠깐 앉아요!
User: 네, 다리 아파요. / 좋아요!
AI: 저도 아파요! 잠깐 앉아서 쉬어요.
User: 너무 좋아요! / 푹 쉬어요!

# CR-3: 떡볶이 같이 먹으러 가요!

## sim1: 마감

AI: 드디어 마감이에요! 정말 힘들었어요.
User: 네, 진짜 피곤해요.
AI: 같이 일해서 아주 재밌었어요!
User: 저도 좋았어요. / 저도 같이 일해서 기뻐요!
AI: 저 지금 배고파요. 배 안 고파요?
User: 진짜 배고파요! / 네, 밥 먹고 싶어요!
AI: 그럼 우리 맛있는 거 먹으러 가요!
User: 좋아요!
## sim2: 떡볶이

AI: 근처에 맛있는 떡볶이 가게가 있어요. 어때요?
User: 떡볶이 매워요? / 떡볶이 좋아요!
AI: 안 매워요. 오늘은 제가 살게요!
User: 진짜요? / 커피는 제가 살게요!
AI: 제가 자주 가는 식당이에요. 진짜 맛있어요!
User: 좋아요! / 가대돼요!
AI: 네! 우리 빨리 가요. 너무 배고파요!
User: 네, 식당에 가서 많이 먹어요!
## sim3: 같이 먹기

AI: 같이 먹어서 정말 좋아요. 더 친해지고 싶어요.
User: 좋아요! / 네, 같이 와서 정말 좋아요.
AI: 진짜요? 혼자 밥 먹으면 조금 심심해요.
User: 네, 저도 혼자 있으면 심심해요. / 우리 앞으로 같이 밥 먹어요!
AI: 떡볶이 진짜 맛있어요. 많이 먹어요!
User: 네, 아주 맛있어요! / 진짜 맛있어서 기분이 좋아요!
AI: 우리 여기 자주 와요! 여기 진짜 좋아해요.
User: 네, 또 와요! / 좋아요! 우리 자주 와요.
