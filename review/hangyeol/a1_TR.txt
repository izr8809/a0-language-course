# TR-1: 아이스 아메리카노 한 잔 주세요

## sim1: Ordering

AI: 안녕하세요! 주문하시겠어요?
User: 아이스 아메리카노 주세요. / 아메리카노 한 잔이요.
AI: 네, 사이즈는 어떤 걸로 드릴까요?
User: 기본 사이즈로 주세요. / 큰 걸로 주세요.
AI: 네. 디저트도 드릴까요?
User: 네, 초코 쿠키 하나 주세요. / 쿠키 종류는 뭐 있어요?
AI: 버터 쿠키랑 초코 쿠키 있어요.
User: 초코 쿠키로 주세요. / 쿠키는 안 먹을게요.
## sim2: Paying

User: 카드로 결제할게요. / 카드 결제 돼요?
AI: 네, 카드 결제 가능해요.
User: 네, 여기 있어요. / 영수증도 주세요.
AI: 네, 결제 끝났어요. 이름이 뭐예요?
User: 저는 ___이에요. / ___예요.
AI: 네, 음료 나오면 이름 부를게요.
User: 네, 감사합니다. / 알겠습니다.
AI: 잠시만 기다려 주세요.
## sim3: Getting the Drink

User: 저기요, 제 커피 나왔어요? / 이거 제 거 맞아요?
AI: 네, 맞아요.
User: 감사합니다. 빨대 하나 주세요. / 빨대 어디 있어요?
AI: 빨대는 바로 옆에 있어요.
User: 창가 자리 비어 있어요? / 안쪽에 앉을게요.
AI: 네, 안쪽에 빈자리 많아요.
User: 고마워요! / 수고하세요!
AI: 네, 맛있게 드세요.

# TR-2: 떡볶이 안 맵게 해주실 수 있어요?

## sim1: 포장마차 도착

AI: 어서 오세요! 뭐 드릴까요?
User: 떡볶이 순한 맛 주세요.
AI: 네, 순한 맛 하나요. 마실 것도 드릴까요?
User: 마실 거 뭐 있어요? / 어묵 국물 주세요.
AI: 어묵 국물이랑 아이스티 있어요.
User: 어묵 국물 주세요. / 아이스티 한 잔 주세요.
AI: 네, 잠시만 기다려 주세요.
User: 고마워요! / 네, 여기서 먹을게요.
## sim2: 음식 조절

User: 저기요! 어묵 국물 더 주세요. / 국물 더 주세요.
AI: 네, 국물 많이 드릴까요?
User: 네, 어묵도 같이 주세요! / 조금만 더 주세요.
AI: 네, 뜨거우니까 조심하세요.
User: 감사합니다! 맛있어요! / 진짜 맛있어요!
AI: 다행이에요. 맛있게 드세요.
User: 네, 정말 맛있어요. / 잘 먹겠습니다!
AI: 네, 부족하면 더 말씀하세요.
## sim3: 계산하기

User: 저기요, 다 먹었어요. 계산해 주세요! / 얼마예요?
AI: 네, 육천 원이에요. 카드로 하세요?
User: 네, 카드로 할게요. / 카드로 결제해 주세요.
AI: 네, 결제 끝났어요. 영수증 필요하세요?
User: 아니요, 괜찮아요. 맛있었어요! / 네, 영수증 주세요. 최고예요!
AI: 감사합니다! 다음에 또 오세요.
User: 네, 꼭 다시 올게요! / 안녕히 계세요!
AI: 네, 조심히 가세요.

# TR-3: 이 버스 건대 가나요?

## sim1: 버스 정류장

User: 저기요, 이 버스 건대입구역 가요? / 건대입구역 가는 버스 맞아요?
AI: 네, 그 버스 건대입구역 가요.
User: 감사합니다! 얼마나 걸려요? / 멀어요?
AI: 세 정거장이에요.
User: 아, 가깝네요. 고마워요! / 생각보다 가깝네요.
AI: 네, 버스가 와요. 빨리 타세요.
User: 감사합니다!
AI: 네, 조심히 타세요.
## sim2: 버스 안

User: 저기요, 다음이 건대입구역이에요? / 건대입구역에 다 왔어요?
AI: 아니요, 한 정거장 더 가야 해요.
User: 알겠습니다. 감사합니다! / 다다음 정거장이에요?
AI: 네, 맞아요. 안내 방송 잘 들으세요.
User: 사람이 많이 내려요?
AI: 네, 거기는 사람이 많아요.
User: 미리 가야겠어요. 고마워요! / 준비해야겠어요.
AI: 네, 조심하세요.
## sim3: 도착

User: 여기가 건대입구역 맞아요? / 이번에 내리면 건대입구역이에요?
AI: 네, 맞아요. 조심히 내리세요.
User: 알려줘서 고마워요! / 감사합니다!
AI: 아니에요. 좋은 하루 보내세요.
User: 도착해서 기분이 좋아요. / 다행이에요.
AI: 짐 잘 챙기세요.
User: 네, 챙겼어요! / 감사합니다.
AI: 네, 안녕히 가세요.

# TR-4: 야, 나 어디 있는 거야?

## sim1: 길을 잃음

User: 여보세요? 나 길을 잃어버렸어! / 한결아, 야시장 입구를 못 찾겠어.
AI: 진짜? 지금 어디에 있어?
User: 편의점이 보여.
AI: 편의점? 주변에는 뭐 있어?
User: 옆에 카페가 있어.
AI: 아, 나 거기 알아.
User: 다행이다! 빨리 와!
AI: 알겠어. 거기서 움직이지 마.
## sim2: 한결에게 연락

AI: 십 분이면 가. 조금만 기다려.
User: 응, 여기 있을게. / 빨리 와! 무서워.
AI: 지금 가고 있어!
User: 빨리 와! / 조심히 와!
AI: 알겠어. 지금 무슨 옷 입었어?
User: 하얀색 티셔츠랑 청바지 입었어. / 검은색 모자 쓰고 있어.
AI: 알겠어. 저기 너 보여!
User: 어? 나도 너 봤어! / 드디어 만났다!
## sim3: 한결과 만남

User: 만나서 다행이야. 많이 기다렸지?
AI: 괜찮아. 잘 찾아서 다행이야.
User: 길이 어두워서 길을 잃었어.
AI: 다음엔 나한테 위치 알려줘. 배고프지?
User: 응, 떡볶이랑 튀김 먹고 싶어! / 밥 먹자!
AI: 야시장 가서 맛있는 거 사 줄게.
User: 진짜 고마워! 내 손 잡아 줘.
AI: 알겠어. 손 잡고 가자!
