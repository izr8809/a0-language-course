# KP-1: 밥은 먹었어?

## sim1: Meeting Your Idol

User: 안녕하세요! 만나서 진짜 반가워요! / 안녕하세요! 드디어 만났어요!
AI: …어, 안녕하세요. 나도 반가워요.
User: 오늘 와줘서 정말 감사해요! / 진짜 보고 싶었어요!
AI: 고마워요. 멀리서 왔어요?
User: 밥은 먹었어요? 오늘 힘들지 않았어요? / 식사는 하셨어요? 오늘 많이 바빴죠?
AI: …어, 먹었어요. 괜찮아요.
User: 오늘 정말 멋있었어요. 보고 싶었어요! / 오늘 무대 진짜 좋았어요. 보고 싶었어요!
AI: …됐어요. 고마워요.
## sim2: Chatting with Idol

User: 한국어 아직 많이 부족한 것 같아요. / 한국어 잘 못해서 죄송해요.
AI: …아니에요. 생각보다 잘 하는데요.
User: 아니에요, 아직 많이 부족해요. / 아직 배우는 중이에요. 더 열심히 할게요.
AI: 잘 해요. 지금도 충분해요.
User: 노래 중에서 제일 좋아하는 게 있어요? / 혹시 제일 좋아하는 노래가 뭐예요?
AI: …딱히 하나는 모르겠는데요.
User: 오늘 공연에서 하나만 불러줄 수 있어요? / 그럼 오늘 무대에서 한 곡 불러줘요!
AI: …해볼게요. 어떤 거 듣고 싶어요?
## sim3: Goodbye

User: 이거 드리고 싶어요. 직접 준비했어요. / 선물 가져왔어요. 직접 만들었어요.
AI: …어, 뭐예요. 고마워요.
User: 편지예요. 꼭 읽어줘요. / 편지예요. 나중에 꼭 읽어주세요.
AI: 읽을게요. 진짜로.
User: 다음에 또 올게요. 꼭 또 봐요! / 또 올게요! 꼭 다시 만나요!
AI: 어. 또 와요.
User: 정말 고마워요. 앞으로도 응원할게요! / 감사해요! 계속 응원할게요!
AI: 나도 고마워요.

# KP-2: ...진짜 대박이다

## sim1: Before the Show

User: 안녕하세요! 혼자 오신 거예요? / 안녕하세요! 혼자 오셨어요?
AI: …어, 혼자 왔어요.
User: 저도 혼자예요. 누구 팬이에요? / 저도요! 누구 제일 좋아해요?
AI: 뭐, 다 좋아하는 편이에요.
User: 진짜 많이 떨려요. 당신은요? / 너무 떨려요! 설레지 않아요?
AI: …나도 좀 그래요. 별로 안 내색해서요.
User: 곧 시작해요! 빨리 보고 싶어요! / 이제 곧이에요! 빨리 시작했으면 좋겠어요!
AI: 어, 나도요. 빨리 시작했으면요.
## sim2: During the Show

User: 드디어 나왔어요! 진짜 멋있지 않아요? / 나왔어요! 진짜 멋져요, 그죠?
AI: …어, 진짜 대박이에요.
User: 노래도 진짜 잘 하고 있어요! / 목소리도 진짜 좋아요!
AI: 맞아요. 예상보다 더 좋은데요.
User: 앵콜 나올 것 같아요? 기대되요! / 앵콜 해줄까요? 너무 기대돼요!
AI: …나와야죠. 안 나오면 섭섭하죠.
User: 진짜 나왔어요! 너무 좋아요! / 앵콜 나왔어요! 진짜 대박이에요!
AI: …어, 좋아요. 진짜로.
## sim3: After the Show

User: 진짜 너무 좋았어요! 어떠셨어요? / 완전 좋았어요! 어땠어요?
AI: …어, 생각보다 훨씬 좋았어요.
User: 목소리도 다 나갔어요. 당신도요? / 너무 소리 질러서 목이 아파요. 당신도요?
AI: 나도요. 뭔가 다 쏟아냈어요.
User: 다음 콘서트도 같이 가요! 좋지 않아요? / 다음에도 같이 와요! 어때요?
AI: …어, 그거 좋겠어요.
User: 잘 가요! 오늘 즐거웠어요! / 잘 가세요! 오늘 진짜 재밌었어요!
AI: 나도요. 잘 가요.

# KP-3: 진짜 수고했어

## sim1: After a Performance

User: 잘 지냈어요? 보고 싶었어요! / 안녕하세요! 많이 보고 싶었어요!
AI: …어, 그냥 있었어요. 고마워요.
User: 오늘 공연 진짜 수고 많았어요! / 오늘 무대 정말 고생하셨어요!
AI: 고마워요. 덕분에 힘냈어요.
User: 지금 이 라이브 보고 있어요! / 지금 방송 보고 있어요!
AI: …알아요. 이름 보였어요.
User: 진짜요? 사랑해요! / 진짜요? 너무 좋아요!
AI: 나도요. 계속 봐주세요.
## sim2: Encouragement

User: 요즘 많이 힘들어 보여요. 괜찮아요? / 요즘 많이 피곤해 보여요. 괜찮으세요?
AI: …좀 그래요. 근데 괜찮아질 거예요.
User: 괜찮아요! 항상 응원하고 있어요! / 걱정하지 마세요! 저 항상 응원해요!
AI: 고마워요. 그 말이 도움이 돼요.
User: 항상 열심히 하는 거 잘 알아요. 고마워요! / 늘 노력하는 거 다 알아요. 감사해요!
AI: …알아줘서 고마워요.
User: 힘내요! 항상 여기 있을게요! / 파이팅! 저 항상 기다리고 있을게요!
AI: 근데 먼저 본인도 챙겨요. 나도 응원할게요.
## sim3: Sending Love

User: 오늘 올라온 영상 봤어요. 너무 좋았어요. / 오늘 올린 영상 봤어요. 진짜 좋았어요.
AI: …그래요? 봐줘서 고마워요.
User: 정말 좋았어요. 자주 올려줘요! / 진짜 좋았어요. 더 많이 올려주세요!
AI: 노력해볼게요. 고마워요.
User: 이제 자야 하지 않아요? 많이 피곤하죠. / 늦었는데 이제 쉬어야 하지 않아요?
AI: …어, 이제 잘 것 같아요.
User: 잘 자요! 좋은 꿈 꿔요! / 푹 쉬세요! 좋은 꿈 꾸세요!
AI: 여러분도요. 잘 자요.
