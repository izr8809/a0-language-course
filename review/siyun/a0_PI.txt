# PI-1: 안녕하세요! 저는 ___이에요

## sim1: Nice to Meet You

AI: 안녕하세요. 시윤이에요.
User: 안녕하세요! / 반가워요!
AI: 이름이 뭐예요?
User: 저는 ___이에요!
AI: 어디에서 왔어요?
User: ___에서 왔어요!
AI: 멀리서 왔네요. 한국 좋아해요?
User: 네, 좋아해요! / 진짜 좋아해요!
## sim2: Getting Closer

User: 시윤 씨, 취미 뭐예요? / 뭐 좋아해요?

User: 저도요! / 오, 저도 좋아해요!
AI: 취향 똑같네요.
AI: 밥 먹었어요?
User: 아직이요! / 아니요!
AI: 그럼 밥 먹으러 가요.
User: 같이 먹어요! / 좋아요!
## sim3: Let's Hang Out

AI: 편하게 해요. 반말 해요.
User: 네! / 좋아요!
User: 주말에 뭐 해? / 이번 주말 시간 있어?

User: 같이 놀래? / 나랑 놀자!

# PI-4: 카톡 해!

## sim1: First Text

User: 뭐 해? / 지금 뭐 하고 있어?

User: 나도 집! 심심해. / 나도 집이야!

AI: 오늘 뭐 했어?
User: 영화 봤어. / 드라마 봤어.
AI: 뭐 봤어?
User: 새로 나온 거! / 비밀!
## sim2: Chatting More
