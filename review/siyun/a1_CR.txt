# CR-1: ...어, 잘 왔네

## sim1: Morning

User: 안녕하세요! / 왔습니다!
AI: 어, 왔어요?
User: 오늘 날씨 좋아요! / 기분 좋아요!
AI: 그래요? 몰랐어요.
User: 오늘 회의 있어요? / 10시에 회의죠?
AI: 네. 10시에 회의실이에요.
User: 알겠습니다! / 준비할게요!
AI: 늦으면 안 돼요. 먼저 갈게요.
## sim2: During the Day

User: 점심 먹었어요? / 배고프지 않아요?
AI: 아직이요. 뭐 먹을 거예요?
User: 근처에 새로 생긴 데 있어요! / 맛집 알아요!
AI: 그래요? 이따 가봐요.
User: 이거 봐주세요. / 확인해 주세요.
AI: 어, 잠깐요.
User: 이 부분 맞아요? / 이거 괜찮아요?
AI: 여기 고치면 돼요.
## sim3: Leaving

User: 오늘 수고하셨습니다! / 고생하셨습니다!
AI: 네, 수고했어요.
User: 먼저 갈게요! / 퇴근하겠습니다!
AI: 네, 가요. 바람 불어요.
User: 좋은 저녁 보내세요! / 내일 봬요!
AI: 네. 내일 일찍 와요.
User: 네! / 알겠어요!
AI: 조심해서 가요.

# CR-2: 알았어, 해볼게

## sim1: Getting Instructions

User: 이거 어떻게 하면 좋을까요? / 이거 어떻게 해요?
AI: 그냥 해봐요. 막히면 말해요.
User: 알겠습니다. / 네, 해보겠습니다.
AI: 네, 오늘까지요.
User: 다 했습니다. / 끝났습니다.
AI: 어, 잠깐 볼게요.
User: 어떤가요? / 괜찮아요?
AI: 네, 잘 했어요.
## sim2: Meeting

User: 이 부분 의견 있으세요? / 어떻게 생각하세요?
AI: 특별히 없어요. 그냥 해요.
User: 이 방향으로 가면 어떨까요? / 이렇게 할까요?
AI: 나쁘지 않아요.
User: 이렇게 마무리할까요? / 이걸로 끝낼까요?
AI: 네, 그렇게 해요.
User: 수고하셨습니다. / 감사합니다.
AI: 네, 다들 수고했어요.
## sim3: End of Day

User: 다 끝났습니다. 확인해 주세요. / 끝났습니다. 봐주세요.
AI: 네, 잠깐 볼게요.
User: 감사합니다. / 고맙습니다.
AI: 잘 했어요. 빨리 했네요.
User: 오늘 수고하셨습니다. / 감사했습니다.
AI: 네, 수고했어요.
User: 먼저 가보겠습니다. / 먼저 갈게요.
AI: 네, 조심해서 가요

# CR-3: ...뭐라고 했어?

## sim1: I Don't Understand

User: 죄송한데, 다시 말해 주세요. / 뭐라고 하셨어요?
AI: 오늘까지 이메일로 보내요.
User: 알겠습니다. 오늘 안에 보내겠습니다. / 네, 보낼게요.
AI: 네. 늦으면 말해요.
User: 이 부분이 맞는 거죠? / 이거 맞아요?
AI: 네, 맞아요.
User: 감사합니다. / 알겠습니다.
AI: 됐어요. 모르면 또 물어봐요.
## sim2: Asking for Help

User: 잠깐 물어봐도 될까요? / 질문 있습니다.
AI: 뭐예요? 말해요.
User: 이 부분을 어떻게 해야 할지 모르겠습니다. / 이거 어떻게 해요?
AI: 이 순서대로 해요.
User: 이해됐습니다. / 이제 알겠습니다.
AI: 네, 됐어요. 별거 아니에요.
User: 많이 도움됐습니다. / 고맙습니다.
AI: 다음에 또 물어봐요.
## sim3: Apologizing

User: 오늘까지는 어려울 것 같습니다. / 오늘은 힘들 것 같아요.
AI: 얼마나 더 필요해요?
User: 하루만 더 주시면 됩니다. / 내일까지 할게요.
AI: 알겠어요. 내일까지 해요.
User: 감사합니다. 열심히 하겠습니다. / 열심히 할게요.
AI: 네. 걱정 마요, 할 수 있어요.
User: 수고하세요. 감사합니다. / 내일 뵙겠습니다.
AI: 네, 내일 봐요.
