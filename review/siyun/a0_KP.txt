# KP-1: 이름 알아

## sim1: First Encounter

User: 안녕하세요! / 와, 시윤이다!
AI: 어, 안녕하세요. 와줘서 고마워요.
User: 반가워요! / 보고 싶었어요!
AI: 저도요. 줄 길었죠?
User: 괜찮아요. 사인 해주세요! / 조금요. 사인 부탁해요!
AI: 이름 뭐예요?
User: ___이에요!
AI: 알았어요. 잠깐만요.
## sim2: He Knows You

AI: 또 왔어요? 멀리서 왔죠?
User: 네! / 항상 와요!
AI: 무리하지 마세요. 건강이 먼저예요.
User: 저 기억해요? / 저 알아요?
AI: 당연하죠, 아직도 저 좋아해요?
User: 네, 좋아해요! / 네, 많이요!
AI: 고마워요. 밥 잘 먹어요.
User: 앞으로도 올게요! / 계속 응원할게요!
## sim3: Goodbye (For Now)

User: 이거요. / 받아 주세요.
AI: 뭐예요? 선물이에요?
User: 편지예요! / 직접 썼어요!
AI: 고마워요. 잘 읽을게요.
AI: 조심히 가세요. 뛰지 마세요.
User: 네, 감사해요! / 다음에 봐요!
AI: 다음에도 오세요. 밥 먹고 오세요.
User: 꼭이요! / 또 올게요!

# KP-2: 나만 봤어

## sim1: Before the Show

AI: 안녕하세요! 다들 제 콘서트 왔어요?
User: 네! / 네, 정말 좋아해요!
AI: 고마워요! 콘서트 너무 기대돼요
User: 맞아요! / 저도 너무 기대돼요!
AI: 이제 곧 만나요! 떨려요?
User: 네, 많이요! / 진짜 떨려요!
AI: 괜찮아요. 제가 더 멋있게 할게요! 
User: 빨리 보고 싶어요! / 기대돼요!
## sim2: During the Show

AI: 저 나왔어요! 오래 기다렸죠?
User: 와, 멋있어요! / 대박이에요!
AI: 지금 저기 이쪽 보고 있어요!
User: 진짜요?! / 설레요!
AI: 한번 더 할까요?
User: 앵콜! 앵콜! / 더 보고 싶어요!
AI: 좋아요! 그럼 다시 시작할게요!
User: 최고예요! / 와, 대박!
## sim3: After the Show

User: 진짜 좋았어요! / 최고였어요!
AI: 고마워요. 저도 오늘 진짜 행복했어요!
User: 저 봤어요? / 눈 마주쳤어요!
AI: 네, 당연하죠!
AI: 오늘 재밌었어요!
User: 네, 저도요! / 진짜 좋았어요!
AI: 다음에 또 봐요!
User: 네, 꼭이요! / 약속이에요!

# KP-3: …봤어

## sim1: He Went Live

AI: 다들 왔어요?
User: 네! / 왔어요!
AI: 오늘 일이 많았어요. 정말 바빴어요.
User: 수고했어요! / 고생했어요!
AI: 오늘 뭐 했어요?
User: 시윤 생각했어요! / 라이브 기다렸어요!
AI: 안 돼요! 할 일하면서 봐요.
User: 네, 사랑해요! / 네, 알겠어요
## sim2: The Reply

AI: 메시지 봤어요.
User: 진짜요?! / 읽었어요?
AI: 네. 고마워요.
User: 진짜 기뻐요! / 너무 좋아요!
AI: 뭐가 좋았어요?
User: 다 좋아요! / 전부 다요!
AI: 그래요? 고마워요. 무리하지 마세요.
User: 항상 응원할게요! / 계속 볼게요!
## sim3: Sending Love

User: 오늘 영상 봤어요! / 영상 잘 봤어요!
AI: 그래요? 늦게까지 보지 마세요.
User: 진짜 좋았어요! / 너무 멋있었어요!
AI: 고마워요. 일찍 자요.
AI: 이제 자요. 늦었어요.
User: 잘 자요! / 푹 쉬세요!
AI: 네. 잘자요.
User: 좋은 꿈 꿔요! / 내일 봐요!
