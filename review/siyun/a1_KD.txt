# KD-1: ...진짜라고?

## sim1: Surprise!

User: 나 어제 BTS 콘서트 갔다 왔어! / 나 어제 완전 대박인 곳 갔어!
AI: …진짜라고? 어땠는데?
User: 완전 좋았어! / 자리도 진짜 좋았어!
AI: …그렇구나. 좋았겠다.
User: 사진 찍어왔어. 한번 봐봐. / 영상도 찍었어. 볼래?
AI: …어, 볼래.
User: 이거 봐, 대박이지? / 이거 가까이서 찍은 거야!
AI: 진짜 대박이네. 좋았겠다.
## sim2: Reacting to News

User: 나 여자친구 생겼어! / 나 만나는 사람 생겼어!
AI: …진짜? 어떻게 됐어?
User: 어제 고백 받았어! / 갑자기 고백 받았어!
AI: 잘됐다. 진짜 축하해.
User: 3년 동안 나 좋아했대. / 오래 좋아했대.
AI: …헐, 진짜? 몰랐어?
User: 응, 전혀 몰랐어. / 하나도 몰랐어.
AI: …그렇구나. 오래 기다렸네.
## sim3: Bad News

User: 나 핸드폰 잃어버렸어. / 핸드폰 어디 갔는지 모르겠어.
AI: …어? 어디서 잃어버렸어?
User: 택시에 두고 내렸어. / 버스에 두고 내렸어.
AI: …어떡해. 빨리 연락해봐.
User: 어떡하지, 큰일 났어... / 진짜 어떡해...
AI: 찾을 수 있어. 너무 걱정 마.
User: 그렇게 생각해? 고마워. / 고마워, 좀 나아졌어.
AI: 응. 잘 될 거야.

# KD-2: ...짜증나, 진짜

## sim1: Frustration

User: 왜 그래? 무슨 일 있었어? / 괜찮아? 무슨 일이야?
AI: 버스 놓쳤어. 진짜 짜증나.
User: 어떡해. 다음 버스 언제야? / 다음 거 금방 와?
AI: …10분 뒤야. 하필 오늘.
User: 오늘 힘들었어? / 피곤해 보여.
AI: 좀 그랬어. 일도 많고.
User: 그렇구나. 수고 많았어. / 오늘 고생했어.
AI: …어, 그래. 고마워.
## sim2: Happy

User: 나 오늘 시험 합격했어! / 나 시험 붙었어!
AI: …진짜? 방금 연락 받았어?
User: 응! 방금 결과 나왔어! / 응! 아까 확인했어!
AI: 잘했네. 진짜 고생 많았어.
User: 고마워. 네가 도와줬잖아. / 네 덕분이야.
AI: …아니야. 네가 한 거야.
User: 그래도 고마워. / 네가 있어서 할 수 있었어.
AI: 됐어. 네가 잘한 거야.
## sim3: Sad

User: 나 오늘 좀 슬퍼. / 나 오늘 기분이 안 좋아.
AI: 왜, 무슨 일 있었어?
User: 친구가 한국 떠났어. / 친한 친구가 돌아갔어.
AI: …그렇구나. 많이 친했어?
User: 응, 많이 친했어. / 응, 벌써 보고 싶어.
AI: 괜찮아. 또 볼 수 있어.
User: 고마워. / 그 말 들으니까 좀 나아.
AI: 응. 힘내.
# KD-3: 보고 싶다고

## sim1: Missing You

User: 뭐 해? 요즘 안 보여. / 요즘 뭐 해?.
AI: …그냥 있어. 왜?
User: 그냥, 보고 싶어서. / 별일 아니야, 보고 싶었어.
AI: …나도 보고 싶어.
User: 나올 수 있어? 잠깐 보자. / 지금 시간 돼? 만나자.
AI: …어, 잠깐 나갈 수 있어.
User: 그럼 빨리 나와! 기다릴게. / 좋아! 먼저 나가 있을게.
AI: 알았어. 조금만 기다려.
## sim2: Together

User: 왔어! 어디 있어? / 나 왔어! 어디야?
AI: 어, 여기. 왔어.
User: 오래 기다렸어? / 많이 기다렸어?
AI: …아니. 방금 왔어.
User: 진짜 오랜만이다. 반가워! / 오랜만이다. 보고 싶었어!
AI: …나도. 잘 지냈어?
User: 응! 같이 있으니까 좋다. / 응! 너 보니까 좋다.
AI: …나도 좋아.
## sim3: Don't Go

User: 나 이제 가봐야 해. / 나 슬슬 가야 할 것 같아.
AI: …벌써? 좀 더 있어.
User: 미안해. / 진짜 가야 해.
AI: 가지 마. 조금만 더.
User: 다음에 또 보자. / 금방 또 만나자.
AI: 응. 내일 볼 수 있어?
User: 응, 조심히 가. / 응, 내일 보자.
AI: 너도 조심해서 가.

# KD-4: ...진짜 장난해?

## sim1: Argument

User: 미안해. 늦었어. / 늦어서 미안해.
AI: …1시간이야.
User: 알아, 진짜 미안해. / 사정이 있었어.
AI: 전화도 안 받고.
User: 핸드폰 무음이었어. / 소리가 안 울렸어.
AI: …항상 이래. 매번.
User: 미안해. 다음엔 안 그럴게. / 다음부터는 꼭 확인할게.
AI: 세 번째야. 이번엔 진짜야.
## sim2: Making Up

User: 아직도 화났어? / 아직 화 안 풀렸어?
AI: …아직 좀 그래.
User: 미안해. 밥 사줄게. / 미안해. 밥 내가 살게.
AI: …진짜 사줘?
User: 응, 뭐 먹고 싶어? / 뭐 먹을래? 다 사줄게.
AI: 삼겹살 먹고 싶어.
User: 그래, 가자! 맛집 있어. / 좋아, 가자! 여기 맛있어.
AI: 같이 가.
## sim3: All Good

User: 맛있다. 기분 좀 풀렸어? / 맛있지? 기분 나아졌어?
AI: …어, 좀. 고기가 좋네.
User: 그치. 아까 진짜 미안했어. / 맞아. 아까 일은 진짜 미안해.
AI: 됐어. 다음엔 전화 받아.
User: 약속할게. 꼭 받을게. / 약속해. 무조건 받을게.
AI: …진짜 약속이야?
User: 응, 약속. 진심이야. / 응, 진짜야. 믿어.
AI: 알겠어. 믿어볼게.
