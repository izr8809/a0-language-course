# PI-1: 안녕하세요, 처음 뵙겠습니다

## sim1: Nice to Meet You

AI: 안녕하세요. 시윤이에요. 이름이 뭐예요?
User: 안녕하세요! 저는 ___이에요. / 반갑습니다! ___이에요.
AI: 네, 반가워요. 한국에 언제 왔어요?
User: ___달 전에 왔어요! / 올해 왔어요!
AI: 한국에 왜 왔어요?
User: 한국어 배우러 왔어요! / 일하러 왔어요!
AI: 그래요? 한국 생활 어때요?
User: 재밌어요! / 좀 어려워요.
## sim2: Getting to Know Each Other

AI: 한국에서 뭐가 제일 좋아요?
User: 음식이요! / 사람들이 좋아요!
AI: 음식이면 뭐 좋아해요?
User: 삼겹살 좋아해요! / 김치찌개요!
User: 시윤 씨는 뭐 좋아해요? / 취미가 뭐예요?
AI: 음악이요. 혼자 듣는 거 좋아해요.
User: 저도 음악 좋아해요! / 어떤 음악이요?
AI: 취향 괜찮네요.
## sim3: Dropping Formality

AI: 아, 그냥 편하게 해요. 반말 해요.
User: 네, 좋아요! / 진짜요?
AI: 나는 시윤이야. 편하게 해.
User: 응! 나는 ___야!
AI: 카톡 있어? 아이디 줘.
User: 여기! 내 아이디야! / 있어!
AI: 저장했어. 밥 먹을 때 연락해.
User: 응, 좋아! / 알겠어!
User: 다음에 같이 밥 먹을래? / 또 만나자!
AI: 좋아. 맛집 알아. 늦지 마.
User: 알겠어! / 기대된다!
AI: 그럼 연락해. 밥 먹고 다녀.

# PI-2: 우리 밥 먹으러 가자

## sim1: Choosing a Place

User: 배고파! 뭐 먹을까? / 밥 먹으러 가자!
AI: 배고프긴 해. 아무거나.
User: 삼겹살 먹고 싶어! / 삼겹살 어때?
AI: 됐고. 맛있는 데 알아. 가.
AI: 가까워. 걸어.
User: 응, 걸어가자! / 얼마나 걸리는데?
AI: 5분. 빨리 와.
User: 알겠어! / 빨리 가자!
## sim2: At the Restaurant

AI: 여기. 맛은 있어.
User: 와, 좋다! / 사람 많다!
AI: 됐고. 메뉴판. 골라.
User: 삼겹살 2인분 / 된장찌개!
AI: 줘. 내가 할게. 태우지 마.
User: 응, 구워줘! / 내가 구울게!
AI: ...맛있긴 하다.
User: 진짜 맛있다! / 여기 잘 찾았다!
## sim3: After Eating

User: 내가 낼게! / 계산할게!
AI: 하. 왜. 같이 내.
User: 아니야, 오늘 내가 쏠게! / 괜찮아, 내가 낼게!
AI: ...알겠어. 다음엔 내가 낸다.
AI: 배부르다. 이제 뭐 할 건데.
User: 카페 갈까? / 좀 걸을까?
AI: 귀찮은데... 가지 뭐.
User: 가자! / 좋아, 가자!

# PI-3: 너 때문에 한국어 배워

## sim1: Getting Personal

AI: 한국어 왜 배워? 갑자기 궁금해서.
User: 한국 친구 사귀고 싶어서! / 드라마 알아듣고 싶어서!
AI: 하. 그런 이유야? 뭐, 나쁘진 않네.
User: 아직 멀었어. / 진짜? 고마워!
AI: 한국에서 힘든 거 없어?
User: 가끔 외로워. / 말이 잘 안 통할 때.
AI: 그럴 때 혼자 있지 마. 귀찮지만 들어줄게.
User: 고마워. / 응, 그럴게.
## sim2: Encouragement

AI: 한국어 처음 배울 때 어땠어?
User: 진짜 어려웠어! / 하나도 몰랐어.
AI: 근데 지금 나랑 말하잖아. 그게 어딘데.
User: 그래? 고마워! / 아직 많이 부족해.
User: 너 때문에 한국어 더 열심히 하고 싶어. / 시윤이 덕분에 한국어 재밌어!
AI: ...왜 갑자기 그래. 부담되게.
User: 진짜야! / 정말이야!
AI: 됐고. 나도 영어 좀 해야 하긴 해.
## sim3: Real Friends

AI: 우리 알게 된 지 얼마나 됐지?
User: 한 달? / 벌써 두 달!
AI: 하. 벌써? 빠르네.
User: 맞아! / 시간 빠르다!
AI: 뭐... 앞으로도 가끔 보지 뭐.
User: 당연하지! / 나도 자주 보고 싶어!
AI: 다음 주에 시간 돼? 밥이나 먹자.
User: 좋아! 연락해! / 기대된다!

# PI-4: 고민 있어?

## sim1: Something's Wrong

AI: 요즘 왜 그래. 뭐 있어?
User: 좀 힘들어. / 고민이 있어.
AI: 말해. 듣고 있으니까.
User: 한국 생활이 좀 힘들어. / 일이 너무 많아.
AI: 뭐가 제일 힘든데.
User: 혼자 있는 시간이 많아서. / 한국어가 아직 어려워서.
AI: ...그랬어?
User: 응, 좀. / 그래서 힘들어.
## sim2: Advice

AI: 혼자 끙끙대지 마. 귀찮지만 들어줄게.
User: 고마워. / 그럴게.
AI: 됐고. 밖에 나가자. 여기 답답해.
User: 어디? / 좋아!
AI: 한강 가자. 바람이나 쐬고.
User: 그래, 가자! / 좋아, 나가자!
AI: ...편의점 들를까. 배고프니까.
User: 좋아! / 편의점 가자!
## sim3: Feeling Better

AI: 좀 나아졌어?
User: 응, 많이 나아졌어! / 덕분에!
AI: ...다행이네.
User: 진짜 고마워. / 덕분이야.
AI: 힘들면 연락해. 안 하면 내가 귀찮게 할 거야.
User: 응, 그럴게! / 알겠어!
AI: 진짜야. 혼자 끙끙대지 마.
User: 약속! / 꼭 연락할게!

# PI-5: 같이 여행 갈까?

## sim1: Where to Go

User: 다음 주 연휴에 뭐 해? / 연휴에 시간 있어?
AI: 없어. 왜.
User: 같이 여행 갈까? / 우리 여행 가자!
AI: 가지 뭐. 어디?
User: 부산 어때? / 제주도 가고 싶어!
AI: 거기? 뭐, 괜찮지.
AI: 며칠 갈 건데.
User: 1박 2일? / 2박 3일?
## sim2: Planning

AI: 뭐 타고 가. KTX? 비행기?
User: KTX 타자! / 비행기 비싸?
AI: KTX. 싸고 2시간이면 돼.
User: 좋아! / 그래, 가자!
AI: 숙소. 호텔? 게스트하우스?
User: 게스트하우스! 싸니까! / 호텔이 편하지 않아?
AI: 알겠어. 내가 예약할게.
User: 응, 고마워! / 잘 부탁해!
## sim3: Getting Excited

User: 부산에서 뭐 하고 싶어? / 뭐 할 거야?
AI: 해운대? 가지 뭐. 너는?
User: 나는 회 먹고 싶어! / 시장 가고 싶어!
AI: 다 하면 되지. 시간은 있으니까.
AI: ...뭐, 나쁘지 않겠다.
User: 나도! 빨리 가고 싶어! / 진짜 기대돼!
AI: 짐 미리 싸. 늦으면 두고 간다.
User: 알겠어! / 응, 준비할게!
