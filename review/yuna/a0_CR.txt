# CR-1: 카페 알바 첫날

## sim1: 첫 인사

AI: 안녕하세요! 저 유나예요. 오늘부터 같이 일해요?
User: 안녕하세요! / 잘 부탁드립니다!
AI: 저도요! 같이 일해요. 준비됐어요?
User: 네! / 잘 부탁드립니다!
User: 여기 처음입니다. / 긴장됩니다.
AI: 저도 얼마 안 됐어요. 걱정 마세요!
User: 다행입니다. / 좋았습니다.
AI: 같이 잘 해요! 뭐부터 알려줄까요?
## sim2: 업무 파악

AI: 이게 메뉴판이에요.
User: 어렵네요. / 많네요.
AI: 금방 외워요. 같이 해봐요!
User: 해볼게요. / 열심히 할게요.
User: 이건 뭐예요? / 인기 메뉴가 뭐예요?
AI: 라떼예요. 많이 팔려요.
User: 알겠습니다. / 감사합니다.
AI: 잘 할 거예요. 한번 만들어 볼래요?
## sim3: 쉬는 시간 (반말 전환)

User: 힘들어? / 괜찮아?

AI: 잘 됐다! 내일도 같이 하자!

User: 다행입니다. / 감사합니다.

User: 여기 얼마나 됐어요? / 여기 오래 됐어요?

User: 좋겠습니다. / 저도 좋습니다.
AI: 같이 하면 더요. 오래 하실 거예요?

# CR-3: 알바 마무리

## sim1: 마감 청소

User: 알겠습니다. / 네!
User: 다 닦았습니다. / 끝났습니다.
AI: 빠르다! 고마워요. 잘하네요!
User: 뭐 더 있습니까? / 또 뭐 합니까?
AI: 바닥만요. 금방 끝나요!
## sim2: 퇴근 준비

User: 집 어디예요? / 여기 가까워요?

User: 또 보자. / 내일 봐.
