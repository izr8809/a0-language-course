# KD-1: 우연한 만남

## sim1: 신촌 거리에서

AI: 앗, 죄송해요!
User: 괜찮아요. / 아, 네.
AI: 많이 놀랐죠?
User: 아니에요. / 조금요.
User: 여기 자주 와요?
AI: 네, 자주요.
User: 저도요. / 저도 자주요.
AI: 그럼 또 봐요.
## sim2: 편의점 앞에서

AI: 또 만났네요.
User: 진짜요? / 어, 정말요?
AI: 동네가 좁네요.
User: 그러네요. / 맞아요.
User: 연락해도 돼요? / 번호 줄 수 있어요?
AI: 좋아요, 네.
User: 번호 알려줘요. / 연락처 주세요.
AI: 잠깐만요.
## sim3: 동네 공원에서 (반말 전환)

AI: 또 왔어?
User: 응, 산책하러. / 응, 그냥 왔어.
AI: 나도 자주 와.
User: 몰랐어. / 진짜?
User: 오늘 뭐 했어? / 오늘 어땠어?
AI: 수업 들었어.
User: 힘들었겠다. / 고생했어.
AI: 고마워.

# KD-2: 설레는 고백

## sim1: 도서관에서

AI: 여기 자리 있어요?
User: 네, 앉아요. / 네, 여기요.
AI: 감사해요.
User: 같이 공부해요. / 같이 해요.
User: 시험 있어요? / 공부하러 왔어요?
AI: 네, 맞아요
User: 저도요. / 저도 내일이에요.
AI: 같이 해요, 그럼.
## sim2: 카페에서

AI: 있잖아요.
User: 네? / 왜요?
AI: 드릴 말 있어요.
User: 말해요. / 들을게요.
User: 뭔데요? / 무슨 말이에요?
AI: 저, 좋아해요.
User: 진짜요? / 정말요?
AI: 네, 진짜요.
## sim3: 학교 앞에서 (반말 전환)

AI: 어젯밤에 생각했어.
User: 나도. / 나도 그랬어.
AI: 어떻게 생각해?
User: 나도 좋아해. / 설레
User: 오늘 뭐 해? / 오늘 시간 있어?
AI: 너랑 있고 싶어.
User: 나도. / 나도 그래.
AI: 그럼 가자.

# KD-3: 화해와 재회

## sim1: 학교 복도에서

AI: 안녕하세요.
User: 안녕하세요. / 오랜만이에요.
AI: 잘 지냈어요?
User: 네, 잘 지냈어요. / 네, 괜찮았어요.
User: 시간 있어요? / 잠깐 돼요?
AI: 네, 가능해요.
User: 얘기할게요. / 할 말 있어요.
AI: 저도요.
## sim2: 버스 정류장에서

AI: 미안해요.
User: 저도요. / 저도 미안해요.
AI: 많이 서운했죠?
User: 조금요. / 네, 좀요.
User: 괜찮아요. / 이제 됐어요.
AI: 다행이에요.
User: 다시 친하게 지내요. / 다시 잘 지내요.
AI: 네, 그래요.
## sim3: 집 앞에서 (반말 전환)

AI: 오늘 고마워.
User: 나도. / 나도 고마워.
AI: 많이 보고 싶었어.
User: 나도 그랬어. / 나도 보고 싶었어.
User: 이제 자주 봐. / 이제 자주 만나.
AI: 당연하지.
User: 약속해. / 꼭이야.
AI: 응, 약속할게.
