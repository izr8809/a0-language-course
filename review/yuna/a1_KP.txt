# KP-1: 팬미팅 심층 대화

## sim1: 사인회

AI: 오래 기다려줘서 정말 고마워요!
User: 만나서 좋아요. / 드디어 만나서 너무 좋아요.
AI: 와줘서 정말 기분이 좋아요.
User: 저도요, 이 순간을 기다렸어요. / 저야말로요, 꿈같아요.
User: 이번 앨범 노래 전부 좋았어요. / 이번 앨범 진짜 좋았어요.
AI: 어떤 곡이 제일 좋았어요?
User: 세 번째 곡이요. / 세 번째 노래를 제일 많이 들었어요.
AI: 그 가사 쓸 때 오래 걸렸는데 다행이에요.
## sim2: 포토타임

AI: 어떤 포즈로 찍을까요?
User: 같이 하트 만들면 어때요? / 하트 포즈 어때요?
AI: 저도 그 포즈 좋아해요!
User: 그럼 같이 해요! / 좋아요, 찍어요!
User: 사진 매일 볼 것 같아요. / 예쁘게 나왔으면 좋겠어요.
AI: 맞아요, 저도 이 사진 갖고 싶어요.
User: 오늘 오래 기억할 것 같아요. / 오늘 잊지 못할 것 같아요.
AI: 저도요, 팬분들 오면 항상 힘이 나요.
## sim3: 개인 토크

AI: 이름이 뭐야? 편하게 얘기해.
User: 나는 ___야, 오늘 최고였어. / ___야, 오늘 진짜 좋았어.
AI: 어떻게 팬이 됐어?
User: 데뷔곡 듣고 바로 팬이 됐어. / 첫 노래 듣고 팬이 됐어.
User: 앞으로도 계속 응원할게! / 계속 응원하고 있을게!
AI: 고마워, 그 말 들으니까 더 열심히 해야겠다.
User: 오늘 가까이서 얘기해서 좋았어. / 직접 만나서 얘기해서 행복했어.
AI: 나도 진짜 좋았어, 다음에도 꼭 와줘.

# KP-2: 콘서트 후 백스테이지

## sim1: 공연 직후

AI: 공연 어땠어요?
User: 너무 좋았어요. / 정말 좋았어요.
AI: 그 말 들으니까 피곤함이 사라지는 것 같아요.
User: 마지막 무대가 멋있었어요. / 마지막 노래가 대단했어요.
User: 무대에서 긴장하지 않았어요? / 공연 전에 긴장 많이 했어요?
AI: 처음엔 긴장했는데 노래 시작하면 괜찮아요.
User: 표정이 자연스럽고 자신감 있었어요. / 무대에서 정말 여유로워 보였어요.
AI: 무대에서 팬들 보면 힘이 나거든요.
## sim2: 공연 준비 이야기

AI: 이번 콘서트 준비하면서 힘든 게 있었어요.
User: 뭐가 힘들었어요? / 어떤 부분이 힘들었어요?
AI: 안무 연습을 많이 해서 발이 아팠어요.
User: 너무 잘해서 전혀 몰랐어요. / 힘들었겠어요.
User: 다음 활동도 기대돼요. / 다음에 뭐 할지 궁금해요.
AI: 다음엔 더 좋은 무대 보여드리고 싶어요.
User: 힌트 줄 수 있어요? / 조금만 알려줘요.
AI: 아직 비밀인데 기대해도 좋아요.
## sim3: 솔직한 대화

AI: 오늘 특별히 더 잘하고 싶었어.
User: 왜 오늘이 더 특별했어? / 오늘 뭐가 달랐어?
AI: 오래 응원해준 팬들이 많이 와서 그랬어.
User: 나도 이 공연 오래 기다렸어. / 나도 진짜 오래 기다렸거든.
User: 다음 콘서트도 꼭 올게! / 다음에도 무조건 올 거야!
AI: 그 말 들으니까 더 열심히 해야겠다.
User: 오늘 가까이서 얘기해서 좋았어. / 직접 만나서 이야기해서 행복했어.
AI: 나도 진짜 좋았어, 다음에 꼭 또 만나자.

# KP-3: 라이브 방송

## sim1: 방송 시작

AI: 오늘 와줘서 고마워요!
User: 바로 들어왔어요! / 기다렸어요!
AI: 기분 좋아요, 힘이 나요!
User: 오늘 뭐 해요? / 뭐 준비했어요?
User: 특별한 거 있어요? / 이벤트 있어요?
AI: 네, 깜짝 이벤트 있어요!
User: 힌트 주세요! / 뭐예요?
AI: 비밀이에요, 곧 알게 될 거예요!
## sim2: Q&A 시간

AI: 질문 있으면 댓글로 보내주세요!
User: 제일 좋아하는 곡이 뭐예요? / 어떤 노래 제일 좋아요?
AI: 마지막 곡이요, 제일 열심히 만들었어요.
User: 그 노래 진짜 좋아요! / 저도 제일 좋아해요!
User: 가사 보고 울었어요. / 감동받았어요.
AI: 진짜요? 그런 말 들으면 힘이 나요.
User: 다음 앨범도 기대할게요! / 응원할게요!
AI: 고마워요, 더 열심히 할게요!
## sim3: 방송 마무리

AI: 방송 끝나기 전에 편하게 얘기하자.
User: 좋아! / 나도 그러고 싶었어.
AI: 오늘 많이 와줘서 행복했어.
User: 나도 재밌었어! / 나도 행복했어!
User: 다음 방송도 할 거지? / 또 해줘!
AI: 그럼, 자주 할게!
User: 기다릴게! / 꼭 봐야지!
AI: 응, 항상 여기서 기다리고 있을게.
