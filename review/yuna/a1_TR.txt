# TR-3: 신촌 분식집

## sim1: 분식집 입장

AI: 오늘 여기서 떡볶이 먹을까요?
User: 좋아요, 오늘 떡볶이가 딱 먹고 싶었어요. / 네, 저도 마침 떡볶이 먹고 싶었어요.
AI: 저도요, 여기 국물이 정말 맛있거든요.
User: 그럼 빨리 들어가서 자리 잡아요. / 얼른 들어가요, 배고파요.
User: 어묵이랑 순대도 같이 시킬 거예요? / 어묵이랑 순대도 추가할까요?
AI: 당연하죠, 셋 다 같이 먹어야 맛있잖아요!
User: 맞아요, 그럼 같이 시켜요. / 그렇죠, 같이 주문해요.
AI: 여기는 항상 양이 많아서 진짜 좋아요.
## sim2: 분식 타임

AI: 이 떡볶이 진짜 맛있지 않아요?
User: 네, 너무 맛있어요, 국물이 최고예요. / 진짜 맛있어요, 특히 국물이 너무 좋아요.
AI: 저는 여기 어묵도 진짜 좋아해요.
User: 저도 어묵 너무 맛있어요, 자꾸 손이 가요. / 어묵 진짜 맛있어요, 계속 먹게 돼요.
User: 떡볶이 한 그릇 더 시킬까요? / 떡볶이 하나 더 주문할까요?
AI: 저는 좋은데요, 배가 좀 차오르긴 했어요.
User: 그럼 반만 더 시켜도 될 것 같아요. / 그러면 조금만 더 시켜요.
AI: 좋아요, 저는 국물만 더 마셔도 좋아요.
## sim3: 식사 후 수다

AI: 오늘 진짜 너무 많이 먹었다, 배 터질 것 같아.
User: 나도, 근데 너무 맛있었어. / 나도 배불러, 근데 진짜 맛있었어.
AI: 맞아, 여기 진짜 자주 와야겠어.
User: 나도 또 오고 싶어, 너무 좋았어. / 나도 다음에 또 오자, 진짜 좋았어.
User: 다음에 또 같이 올 수 있어? / 우리 다음에도 여기 올까?
AI: 당연하지, 언제든지 같이 오자!
User: 그래, 다음엔 다른 메뉴도 도전해보자. / 좋아, 다음엔 새로운 거 먹어보자.
AI: 좋아, 다음엔 김밥도 같이 먹어보자.

# TR-1: 신촌 노래방

## sim1: 노래방 가기 전

AI: 저기 노래방 어때요, 같이 가요!
User: 좋아요, 오늘 노래 부르고 싶었어요. / 네, 저도 마침 노래방 가고 싶었어요.
AI: 여기 노래방 진짜 좋거든요, 기대돼요!
User: 그럼 빨리 들어가요, 저도 기대돼요. / 얼른 가요, 저도 신나요.
User: 어떤 노래 부를 거예요? / 무슨 노래 부를 건지 정했어요?
AI: 요즘 좋아하는 노래가 엄청 많아요!
User: 저도 같이 부를 수 있는 노래 골라봐요. / 우리 둘이 부를 수 있는 노래도 찾아봐요.
AI: 같이 부를 수 있는 노래로 골라볼게요.
## sim2: 노래방 안에서

AI: 이 노래 진짜 좋아해요, 같이 불러요!
User: 알겠어요, 같이 불러볼게요. / 좋아요, 같이 해봐요.
AI: 하이라이트 부분이 진짜 너무 좋아요.
User: 맞아요, 거기서 목소리 잘 나온 것 같아요. / 그 부분에서 노래 진짜 잘 부른 것 같아요.
User: 점수가 얼마나 나왔어요? / 몇 점 나왔어요?
AI: 88점이에요, 꽤 잘 나온 것 같아요!
User: 저도 한번 해볼게요, 이번엔 질 수 없죠. / 이번엔 제 차례예요, 지지 않을 거예요.
AI: 저는 응원할게요, 힘내서 불러봐요!
## sim3: 노래방 나오면서

AI: 목이 좀 쉬어버렸어, 너무 많이 불렀나봐.
User: 나도, 근데 진짜 재밌었어. / 나도 목 아파, 근데 완전 재밌었어.
AI: 응, 이렇게 신난 건 오랜만이야.
User: 맞아, 스트레스가 다 풀린 것 같아. / 응, 기분이 진짜 좋아졌어.
User: 다음에도 노래방 같이 올 수 있어? / 우리 다음에도 노래방 가자.
AI: 당연하지, 나 노래방 진짜 좋아해!
User: 그래, 다음엔 더 많은 노래 연습해올게. / 좋아, 다음엔 새 노래도 준비해올게.
AI: 좋아, 나도 연습해서 점수 더 높일 거야.

# TR-2: 신촌 편의점

## sim1: 편의점 들어가기

AI: 배고픈데 편의점에서 뭔가 살까요?
User: 좋아요, 저도 뭔가 먹고 싶었어요. / 네, 저도 배고팠어요, 들어가요.
AI: 여기 신상 과자가 엄청 많아 보여요!
User: 그래요? 어떤 거 있는지 같이 봐요. / 진짜요? 뭐가 있는지 구경해봐요.
User: 음료도 하나 살 거예요? / 마실 것도 하나 고를 거예요?
AI: 네, 달달한 걸로 하나 골라볼게요.
User: 저는 탄산 마시고 싶어요, 같이 고를까요? / 저는 탄산음료 먹고 싶어요, 같이 골라요.
AI: 좋아요, 같이 골라서 하나씩 사요.
## sim2: 편의점 안에서

AI: 이 새우깡이랑 이 과자 중에 뭐가 더 맛있을까요?
User: 저는 새우깡이 더 좋을 것 같아요. / 새우깡이 더 맛있을 것 같아요.
AI: 그럼 새우깡으로 살게요, 한번 믿어볼게요.
User: 제 선택을 믿어줘서 고마워요. / 믿어줘서 감사해요, 후회 안 할 거예요.
User: 제가 먼저 계산할게요. / 오늘은 제가 낼게요.
AI: 아니에요, 오늘은 제가 살게요!
User: 그래요? 그럼 다음엔 제가 꼭 낼게요. / 정말요? 그러면 다음엔 제가 살게요.
AI: 알겠어요, 그럼 다음엔 부탁할게요.
## sim3: 편의점 나오면서

AI: 아 이거 진짜 맛있어, 새우깡 진짜 잘 골랐어.
User: 그렇지, 내가 잘 골랐지. / 맞지, 역시 내 눈이 좋지.
AI: 맞아, 역시 믿고 따라오길 잘했어.
User: 앞으로도 내 선택 믿어도 돼. / 다음에도 내가 고를게, 믿어.
User: 오늘 하루 어땠어? / 오늘 재밌었어?
AI: 진짜 재밌었어, 신촌에서 이렇게 놀기 딱이야.
User: 나도, 오늘 같이 나와서 잘됐어. / 나도 재밌었어, 같이 와서 좋았어.
AI: 맞아, 다음에도 또 이렇게 놀자!
