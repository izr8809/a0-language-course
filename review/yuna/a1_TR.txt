# TR-3: 신촌 분식집

## sim1: 분식집 입장

AI: 오늘 여기서 떡볶이 먹을까요?
User: 좋아요! / 네, 먹고 싶었어요!
AI: 저도요, 여기 국물이 맛있어요.
User: 빨리 들어가요! / 배고파요!
User: 어묵도 시킬까요? / 순대도 먹을까요?
AI: 당연하죠, 같이 먹어야 맛있어요!
User: 맞아요! / 같이 시켜요!
AI: 여기 양이 많아서 좋아요.
## sim2: 분식 타임

AI: 이 떡볶이 진짜 맛있지 않아요?
User: 네, 너무 맛있어요! / 국물이 최고예요!
AI: 저는 여기 어묵도 좋아해요.
User: 저도요! / 어묵 맛있어요!
User: 떡볶이 더 시킬까요? / 하나 더 먹을까요?
AI: 좋아요, 근데 배가 좀 불러요.
User: 그럼 조금만 더요. / 반만 더 시켜요.
AI: 좋아요, 국물만 더 마실게요.
## sim3: 식사 후 수다

AI: 오늘 너무 많이 먹었다, 배불러.
User: 나도 배불러. / 근데 맛있었어.
AI: 맞아, 여기 자주 와야겠어.
User: 나도 또 오고 싶어. / 다음에 또 오자.
User: 다음에도 같이 올래? / 또 올까?
AI: 당연하지, 언제든 같이 오자!
User: 좋아! / 다음엔 다른 것도 먹자.
AI: 응, 다음엔 김밥도 먹어보자.

# TR-1: 신촌 노래방

## sim1: 노래방 가기 전

AI: 저기 노래방 어때요? 같이 가요!
User: 좋아요! / 저도 가고 싶었어요!
AI: 여기 노래방 진짜 좋아요!
User: 빨리 들어가요! / 기대돼요!
User: 어떤 노래 부를 거예요? / 뭐 부를 거예요?
AI: 아직 모르겠어요!
User: 같이 부를 노래 골라요! / 둘이 부를 노래 찾아요!
AI: 네, 같이 부를 노래 찾아볼게요.
## sim2: 노래방 안에서

AI: 이 노래 좋아해요, 같이 불러요!
User: 알았어요! / 같이 해봐요!
AI: 하이라이트 부분이 진짜 좋아요.
User: 맞아요! / 잘 부른 것 같아요!
User: 점수 얼마 나왔어요? / 몇 점이에요?
AI: 88점이에요! 잘 나왔죠?
User: 이번엔 제 차례예요! / 저도 해볼게요!
AI: 응원할게요! 힘내요!
## sim3: 노래방 나오면서

AI: 목이 좀 쉬었어, 너무 많이 불렀나봐.
User: 나도 목이 아파! / 근데 재밌었어.
AI: 진짜 신났어.
User: 스트레스 다 풀렸어. / 기분 좋아졌어.
User: 다음에도 같이 가자! / 또 오자!
AI: 당연하지, 노래방 좋아해!
User: 다음엔 새 노래 연습해올게. / 다른 노래 연습해야지!
AI: 좋아, 나도 점수 더 높일 거야.

# TR-2: 신촌 편의점

## sim1: 편의점 들어가기

AI: 배고픈데 편의점 갈까요?
User: 좋아요! / 저도 배고팠어요!
AI: 여기 신상 과자 많아 보여요!
User: 그러네요? / 같이 봐요!
User: 음료도 살 거예요? / 마실 것도요?
AI: 네, 달달한 거 고를게요.
User: 저는 탄산이요! / 같이 골라요!
AI: 좋아요, 같이 사요.
## sim2: 편의점 안에서

AI: 새우깡이랑 이거 중에 뭐가 맛있을까요?
User: 새우깡이요! / 새우깡이 더 맛있어요!
AI: 그럼 새우깡으로 할게요!
User: 좋은 선택이에요! / 맛있을 거예요!
User: 제가 계산할게요! / 오늘은 제가 낼게요!
AI: 아니에요, 오늘은 제가 살게요!
User: 진짜요? 다음엔 제가요! / 그럼 다음엔 제가!
AI: 네, 다음에 부탁해요.
## sim3: 편의점 나오면서

AI: 이거 진짜 맛있어, 잘 골랐어.
User: 그치? / 내가 잘 골랐지!
AI: 맞아, 믿고 사길 잘했어.
User: 앞으로도 내 선택 믿어! / 다음에도 내가 고를게!
User: 오늘 어땠어? / 재밌었어?
AI: 진짜 재밌었어.
User: 나도 재밌었어! / 같이 와서 좋았어.
AI: 그래, 다음에도 또 놀자!
