# CR-1: 바쁜 주말

## sim1: 오픈 준비

AI: 오늘 주말이라 바쁠 거예요!
User: 그래요? / 많이요?
AI: 네, 줄이 문 밖까지 설 수도 있어요.
User: 알겠어요! / 열심히 할게요!
User: 저는 뭐 하면 돼요? / 뭐 도와줄까요?
AI: 음료 만드는 거 해줄 수 있어요?
User: 네, 할 수 있어요! / 네, 할게요!
AI: 고마워요! 같이 힘내요!
## sim2: 러시 타임

AI: 주문이 많이 밀렸어요, 서둘러요!
User: 네, 빨리 만들게요! / 네, 알겠어요!
AI: 아이스 아메리카노 세 잔 먼저요!
User: 알겠어요! / 바로 만들게요!
User: 잠깐, 주문 맞아요? / 다음 주문 뭐예요?
AI: 라떼 두 잔에 녹차 한 잔이에요!
User: 고마워요! / 알겠어요!
AI: 잘 하고 있어요.
## sim3: 마감 정리

AI: 오늘 진짜 힘들었지.
User: 응, 피곤해. / 괜찮아.
AI: 같이 청소하면 금방 끝나!
User: 맞아! / 같이 하자!
User: 오늘 나 잘한 것 같아? / 어땠어?
AI: 진짜 잘했어.
User: 고마워! / 다행이다!
AI: 다음 주도 같이 힘내자!

# CR-2: 알바 중 실수

## sim1: 실수 발생

AI: 저 음료 잘못 만든 것 같아요!
User: 어떻게요? / 뭐가 잘못됐어요?
AI: 핫으로 줘야 하는데 아이스로 만들었어요.
User: 빨리 다시 만들어요! / 괜찮아요, 다시 해요!
User: 손님 많이 기다려요? / 급해요?
AI: 네, 그래서 지금 긴장돼요.
User: 빨리 만들면 괜찮아요! / 걱정 마세요!
AI: 고마워요, 서두를게요!
## sim2: 사과와 수정

AI: 손님한테 죄송하다고 해야 할 것 같아요.
User: 솔직하게 말해요. / 그게 좋아요.
AI: 새로 만들었다고 설명할게요.
User: 네, 손님도 이해할 거예요! / 잘 될 거예요!
User: 손님이 이해해줘서 다행이에요! / 잘됐어요!
AI: 네, 정말 다행이에요!
User: 앞으로 주문 다시 확인해요! / 다음엔 확인해요!
AI: 맞아요, 꼭 확인하고 만들게요!
## sim3: 반성과 다짐

AI: 아직도 아까 실수 때문에 속상해?
User: 응, 조금. / 좀 그래.
AI: 괜찮아, 누구나 실수해.
User: 고마워. / 그래도 미안해.
User: 다음엔 더 잘할 수 있을까? / 괜찮을까?
AI: 당연하지, 이번에 많이 배웠잖아!
User: 그렇게 생각하니까 좀 나아. / 고마워.
AI: 맞아, 실수하면서 더 잘하는 거야.

# CR-3: 알바 마지막 날

## sim1: 마지막 날 아침

AI: 오늘이 마지막 날이에요, 실감 나요?
User: 아직 모르겠어요. / 이상해요.
AI: 벌써 6개월이에요, 아쉬워요.
User: 저도요. / 맞아요.
User: 좋은 경험이었어요! / 많이 배웠어요!
AI: 맞아요! 새로운 시작이에요!
User: 응원해 주세요! / 감사해요!
AI: 새로운 곳에서도 잘할 거예요!
## sim2: 마지막 근무

AI: 오늘도 열심히 하는 모습 멋있어요!
User: 마지막이니까요. / 잘하고 싶어요.
AI: 그 마음이 느껴져요!
User: 진짜요? / 고마워요!
User: 같이 일한 분들한테 인사했어요? / 인사 다 했어요?
AI: 아직이요, 조금 있다가 할 거예요.
User: 지금 해두는 게 좋아요! / 나중에 후회해요!
AI: 맞아요, 지금 인사하고 올게요!
## sim3: 작별 인사

AI: 이제 진짜 끝났어, 어때?
User: 시원해. / 서운해.
AI: 여기서 진짜 많이 성장했어.
User: 응, 좋은 경험이었어. / 나도 그래.
User: 앞으로도 연락하자! / 가끔 봐!
AI: 당연하지, 언제든 연락해!
User: 고마워! / 너도 화이팅!
AI: 잘 가, 앞으로도 잘 될 거야!
