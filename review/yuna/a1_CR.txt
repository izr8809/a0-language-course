# CR-1: 바쁜 주말

## sim1: 오픈 준비

AI: 오늘 주말이라 엄청 바쁠 것 같아요!
User: 그렇습니까? 얼마나 바빠질 것 같습니까?
AI: 줄이 문 밖까지 설 수도 있어요.
User: 알겠습니다, 최선을 다해서 해보겠습니다.
User: 저는 오늘 뭘 맡으면 되겠습니까?
AI: 음료 만드는 거 담당해줄 수 있어요?
User: 네, 제가 할 수 있을 것 같습니다.
AI: 잘 부탁해요! 같이 열심히 해봐요.
## sim2: 러시 타임

AI: 주문이 너무 많이 밀렸어요, 서둘러요!
User: 네, 최대한 빨리 만들어 보겠습니다.
AI: 아이스 아메리카노 세 잔이 지금 급해요.
User: 알겠습니다, 바로 만들기 시작하겠습니다.
User: 잠깐, 저 주문을 제대로 들은 거 맞습니까?
AI: 라떼 두 잔에 녹차 한 잔이었어요!
User: 감사합니다, 덕분에 실수 안 했습니다.
AI: 잘 하고 있어요, 조금만 더 힘내봐요!
## sim3: 마감 정리

AI: 오늘 진짜 힘든 하루였지, 괜찮아?
User: 응, 많이 피곤하지만 그래도 괜찮아.
AI: 같이 청소하면 금방 끝날 것 같아.
User: 맞아, 같이 하면 훨씬 더 빠를 거야.
User: 오늘 나 꽤 잘한 것 같지 않아?
AI: 진짜 잘했어, 네 덕분에 잘 버텼어!
User: 다음 주도 이렇게 바쁠까?
AI: 모르겠어, 그냥 같이 또 열심히 해보자.

# CR-2: 알바 중 실수

## sim1: 실수 발생

AI: 저 음료를 잘못 만든 것 같아요, 어떡하죠!
User: 어떻게 잘못 만들었습니까, 뭐가 문제입니까?
AI: 뜨거운 걸로 줘야 하는데 차갑게 만들었어요.
User: 아, 그러면 빨리 다시 만들어야 할 것 같습니다.
User: 손님이 많이 기다리고 있는 거 아닙니까?
AI: 네, 그래서 지금 엄청 긴장되고 있어요.
User: 빨리 만들면 괜찮을 겁니다, 너무 걱정 마십시오.
AI: 고마워요, 서두르다가 또 실수할 뻔했어요.
## sim2: 사과와 수정

AI: 손님한테 솔직하게 죄송하다고 해야 할 것 같아요.
User: 맞습니다, 솔직하게 말하는 게 제일 좋을 것 같습니다.
AI: 음료 새로 만들어드렸다고 직접 설명할게요.
User: 네, 그러면 손님도 이해해줄 거라고 생각합니다.
User: 손님이 이해해줘서 진짜 다행이었습니다, 그렇지 않습니까?
AI: 네, 정말 다행이었어요, 식은땀이 너무 났어요.
User: 앞으로는 주문을 다시 한 번 확인하는 게 좋을 것 같습니다.
AI: 맞아요, 앞으로는 꼭 확인하고 나서 만들게요.
## sim3: 반성과 다짐

AI: 아직도 아까 실수 때문에 속상해 보여.
User: 응, 더 꼼꼼하게 확인했어야 했는데.
AI: 처음에는 누구든지 실수할 수 있어, 너무 자책하지 마.
User: 그래도 손님한테 불편을 드린 것 같아서 마음에 걸려.
User: 다음에는 내가 더 잘할 수 있을까?
AI: 당연하지, 이번 경험이 정말 좋은 교훈이 됐을 거야.
User: 그렇게 생각하니까 조금 마음이 나아진 것 같아.
AI: 맞아, 실수를 통해서 더 많이 성장하는 거야.

# CR-3: 알바 마지막 날

## sim1: 마지막 날 아침

AI: 오늘이 진짜 마지막 날이에요, 실감이 나요?
User: 아직 잘 모르겠습니다, 좀 이상한 느낌입니다.
AI: 여기서 일한 게 꽤 오랜 시간이 됐잖아요.
User: 맞습니다, 벌써 6개월이나 지났다는 게 믿기지 않습니다.
User: 마지막이라고 생각하니까 좀 슬프지 않습니까?
AI: 네, 저도 아쉬운 마음이 정말 많이 들어요.
User: 그래도 새로운 시작이기도 하지 않습니까.
AI: 맞아요, 새로운 곳에서도 분명 잘할 수 있을 거예요.
## sim2: 마지막 근무

AI: 오늘도 이렇게 열심히 하는 모습이 정말 멋있어요.
User: 마지막 날이니까 더 잘하고 싶었습니다.
AI: 그 마음이 음료에도 느껴지는 것 같아요.
User: 정말입니까? 그런 말 들으니까 기분이 많이 좋습니다.
User: 같이 일했던 분들한테 인사는 다 했습니까?
AI: 아직 다 못했어요, 조금 있다가 할 거예요.
User: 나중에 후회하지 않게 지금 해두는 게 좋을 것 같습니다.
AI: 맞아요, 잠깐 시간 내서 지금 인사하고 올게요.
## sim3: 작별 인사

AI: 이제 진짜 끝났어, 어때?
User: 음, 이상하게 시원하면서도 서운해.
AI: 여기서 진짜 많이 성장한 것 같아.
User: 응, 나도 그렇게 생각해, 좋은 경험이었어.
User: 앞으로도 자주 연락하면서 지낼 수 있어?
AI: 당연하지, 언제든지 연락해도 돼!
User: 고마워, 여기서 일한 거 하나도 후회 없어.
AI: 잘 가, 앞으로도 분명 잘 될 거야!
