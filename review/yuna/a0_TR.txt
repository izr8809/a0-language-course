# TR-1: 떡볶이 주세요

## sim1: 야시장 음식 주문

AI: 어서오세요! 뭐 드릴까요?
User: 떡볶이 주세요. / 순대 주세요.
AI: 네! 몇 인분이요?
User: 일 인분이요. / 이 인분이요.
AI: 계란이나 치즈 추가하실래요?
User: 치즈 추가요. / 계란 추가요.
AI: 네, 알겠습니다! 좀만 기다려 주세요.
User: 네! / 알겠어요!
## sim2: 결제 + 추가주문

AI: 다 됐어요! 오천 원이에요.
User: 카드 돼요? / 현금이요.
AI: 네, 됩니다!
User: 여기요. / 네!
User: 저기요, 음료도 있어요? / 음료 뭐 있어요?
AI: 네! 매실차랑 사이다 있어요.
User: 매실차 주세요. / 사이다 주세요.
AI: 네, 천 원이요!
## sim3: 마무리 + 인사

AI: 맛있게 드셨어요?
User: 네, 맛있었어요! / 네, 정말 좋았어요!
AI: 감사합니다! 또 오세요!
User: 네, 또 올게요! / 감사합니다!
User: 포장도 돼요? / 하나 더 주세요.
AI: 네, 돼요! 잠깐만요!
User: 감사합니다! / 잘 먹었어요!
AI: 네, 맛있게 드세요! 안녕히 가세요!

# TR-2: 아이스 아메리카노 주세요

## sim1: 카페 주문

AI: 안녕하세요! 뭐 드릴까요?
User: 아이스 아메리카노 주세요. / 아이스 라떼 주세요.

AI: 딸기 케이크랑 티라미수 있어요!
User: 딸기 케이크 주세요. / 티라미수 주세요.

# TR-3: 여기서 내릴게요

## sim1: 버스 타기

User: 저기요, 이 버스 홍대 가요? / 이거 몇 번 버스예요?
AI: 네, 홍대역 가요! 타세요!
User: 감사합니다! / 네!
AI: 카드 찍으세요!
User: 홍대역까지 몇 정거장이에요? / 여기서 멀어요?
AI: 세 정거장이에요!
User: 알겠어요! / 감사합니다!
AI: 네!
## sim2: 이동 중

User: 저기요, 다음이 홍대역이에요? / 여기 어디예요?
AI: 아직이에요! 하나 더 가세요.
User: 아, 네! / 알겠어요!
AI: 다음에 내리세요!
AI: 다음 정류장 홍대역이에요!
User: 감사합니다! 여기서 내릴게요. / 알겠어요!
AI: 네, 조심히 내리세요!
User: 네! / 감사합니다!
## sim3: 하차 + 택시

User: 저기요, 택시 어디서 타요? / 택시 정류장 어디예요?
AI: 저쪽에 있어요! 횡단보도 건너세요.
User: 감사합니다! / 저기요?
AI: 네, 저쪽이요!
User: 신촌이요. / 신촌 가 주세요.
AI: 네, 알겠습니다! 십 분 정도 걸려요.
User: 네! / 감사합니다!
AI: 안전벨트 매 주세요!
