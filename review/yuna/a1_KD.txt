# KD-3: 오해와 해명

## sim1: 오해 발생

AI: 저 드릴 말씀이 있어요, 잠깐 시간 있어요?
User: 무슨 일이에요, 표정이 안 좋아 보여요. / 왜 그래요, 무슨 일 있어요?
AI: 친구한테 이야기 들었는데 좀 속상해요.
User: 무슨 이야기를 들으셨는지 말해줄 수 있어요? / 어떤 이야기를 들었는지 알려줄 수 있어요?
User: 혹시 제가 한 말을 오해하신 것 아닐까요? / 제 말이 잘못 전해진 것 같아요.
AI: 직접 들은 거라서 오해가 아닌 것 같아요.
User: 그 상황을 전부 알고 계신 건지 궁금해요. / 혹시 그때 상황을 다 들으신 건가요?
AI: ...그러고 보니 전부는 모르는 것 같아요.
## sim2: 해명과 화해

AI: 그날 있었던 일 자세히 설명해줄 수 있어요?
User: 네, 사실 그때 상황이 좀 복잡했어요. / 네, 그때 일이 좀 복잡한 상황이었어요.
AI: 복잡했어도 솔직하게 말해주면 좋겠어요.
User: 제가 그런 의도로 한 말이 아니었어요. / 저는 그런 뜻으로 말한 게 아니었어요.
User: 이제 제 말을 믿어줄 수 있을 것 같아요? / 이제 좀 이해가 됐을까요?
AI: 처음부터 이야기해줬으면 더 좋았을 텐데요.
User: 미안해요, 그때 제대로 설명했어야 했어요. / 죄송해요, 그때 바로 말했어야 했어요.
AI: 저도 너무 단정 지은 것 같아요, 미안해요.
## sim3: 진짜 화해

AI: 이제 오해 다 풀렸으니까 진짜 다행이야.
User: 나도 솔직하게 말했어야 했어, 미안해. / 나도 바로 얘기했어야 했는데, 미안해.
AI: 다음엔 바로 말해줘, 나 진짜 답답했잖아.
User: 알겠어, 다음엔 바로 말할게. / 응, 다음부터는 숨기지 않을게.
User: 우리 앞으로는 더 솔직하게 지내자. / 앞으로 우리 서로 숨기지 말자.
AI: 좋아, 그 대신 나도 솔직하게 말할 거야.
User: 그게 더 좋지, 우리잖아. / 당연하지, 우리 사이에 그게 맞지.
AI: 맞아, 이런 일 다시는 없을 거야.

# KD-1: 감정 고백

## sim1: 마음의 시작

AI: 저 요즘 뭔가 이상한 것 같아요.
User: 어떻게 이상한지 말해줄 수 있어요? / 뭐가 이상한 건지 알려줄 수 있어요?
AI: 설명하기 좀 어려운데, 뭔가 달라진 것 같아요.
User: 혹시 저랑 관련이 있는 건지 궁금해요. / 혹시 그게 저 때문인 건가요?
User: 얼굴이 빨개진 것 같아요, 혹시 괜찮아요? / 얼굴이 많이 붉어졌어요, 괜찮으세요?
AI: 그냥 더워서 그래요, 신경 쓰지 않아도 돼요.
User: 별로 더운 날씨가 아닌데요, 솔직하게 말해봐요. / 오늘 덥지 않은데요, 진짜 이유를 말해줘요.
AI: ...솔직하게 말하기가 좀 무서워요.
## sim2: 고백 직전

AI: 저 물어봐도 되는 말이 하나 있어요.
User: 뭐든지 물어봐요, 다 대답할 수 있어요. / 네, 편하게 물어봐요, 다 말해줄게요.
AI: 저를 어떻게 생각하는지 정말 궁금해요.
User: 어떤 의미로 물어보는 건지 알려줄 수 있어요? / 그게 어떤 뜻인지 좀 더 말해줄 수 있어요?
User: 저한테 특별한 감정이 생긴 건 아닐까요? / 혹시 저를 좋아하게 된 건 아니에요?
AI: 저도 잘 모르겠는데, 자꾸 신경이 많이 써져요.
User: 저도 사실 비슷한 감정이 있었어요. / 사실 저도 같은 마음이었어요.
AI: 정말요? 지금 심장이 너무 빨리 뛰고 있어요.
## sim3: 고백

AI: 있잖아, 나 솔직하게 말해도 괜찮아?
User: 응, 말해봐. 나 듣고 있어. / 당연하지, 편하게 말해.
AI: 나 너한테 마음이 생긴 것 같아.
User: 나도 그 말 기다리고 있었어. / 나도 그 말 듣고 싶었어.
User: 그럼 우리 이제 어떻게 되는 거야? / 그러면 우리 이제 사귀는 거야?
AI: 나 오래 기다렸어, 먼저 말하고 싶었거든.
User: 나도 좋아해, 이제 말할 수 있어서 좋아. / 나도 같은 마음이야, 솔직해져서 다행이야.
AI: 잘됐다, 이제 계속 같이 있고 싶어.

# KD-2: 이별 후 재회

## sim1: 우연한 재회

AI: 어... 여기서 만날 줄은 전혀 몰랐어요.
User: 저도 몰랐어요, 정말 오랜만이에요. / 저도 깜짝 놀랐어요, 진짜 오래됐네요.
AI: 맞아요, 벌써 이렇게 오래됐네요.
User: 그동안 어떻게 지냈는지 너무 궁금했어요. / 그동안 잘 지냈는지 정말 궁금했어요.
User: 잠깐이라도 같이 얘기할 수 있을까요? / 시간 괜찮으면 잠깐 이야기할까요?
AI: 저도 그러고 싶었어요, 시간 좀 있어요?
User: 네, 근처 카페에 들어가도 괜찮을까요? / 네, 저기 카페에서 앉아서 이야기해요.
AI: 그럼요, 같이 가요.
## sim2: 과거를 나누다

AI: 예전에 여기 근처 같이 온 적 있었잖아요.
User: 맞아요, 그때가 벌써 옛날 일이 됐네요. / 그러게요, 시간이 정말 빨리 갔네요.
AI: 저는 가끔 그때 생각이 나요, 솔직히요.
User: 저도 그랬어요, 뭔가 아쉬운 마음이 있었어요. / 저도요, 항상 좀 아쉬웠어요.
User: 그때 제대로 끝내지 못한 것 같아서 아쉬웠어요. / 그때 잘 마무리하지 못해서 마음에 남았어요.
AI: 저도 그 마음 알아요, 너무 갑작스러웠잖아요.
User: 이제는 솔직하게 얘기할 수 있을 것 같아요. / 지금은 편하게 이야기할 수 있을 것 같아요.
AI: 저도 같은 마음이에요, 다행이에요.
## sim3: 다시 시작

AI: 있잖아, 이제 그냥 편하게 얘기해도 돼?
User: 응, 나도 그게 훨씬 더 편할 것 같아. / 그래, 나도 편하게 말하는 게 좋아.
AI: 솔직히 오늘 다시 만나서 너무 좋았어.
User: 나도 그랬어, 진짜 반가웠어. / 나도, 오늘 다시 만나서 진짜 좋았어.
User: 우리 이제 계속 연락하고 지내자. / 우리 이번엔 자주 연락하자.
AI: 그러자, 우리 이번엔 끊지 말자.
User: 이번엔 꼭 연락하기로 약속해. / 이번엔 진짜 연락 끊지 말자, 약속이야.
AI: 응, 나도 진짜로 약속할게.
