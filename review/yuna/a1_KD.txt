# KD-3: 오해와 해명

## sim1: 오해 발생

AI: 저 할 말 있어요, 시간 있어요?
User: 무슨 일이에요? / 왜 그래요?
AI: 친구한테 이야기 들었는데 속상해요.
User: 무슨 이야기요? / 뭘 들었어요?
User: 제가 한 말 오해하신 것 같아요. / 제 말이 잘못 전해진 것 같아요.
AI: 직접 들은 거라서 오해가 아닌 것 같아요.
User: 그때 상황 다 들었어요? / 전부 알고 계세요?
AI: 전부는 모르는 것 같아요.
## sim2: 해명과 화해

AI: 그날 무슨 일이었어요?
User: 그때 상황이 복잡했어요. / 그때 좀 복잡한 상황이었어요.
AI: 복잡해도 솔직하게 말해주세요.
User: 저는 그런 뜻으로 말한 게 아니었어요. / 그런 의도가 아니었어요.
User: 이제 제 말 믿어요? / 이해됐어요?
AI: 처음부터 말해줬으면 좋았을 텐데요.
User: 미안해요, 바로 설명했어야 했어요. / 죄송해요, 그때 말했어야 했어요.
AI: 저도 너무 빨리 판단했어요, 미안해요.
## sim3: 진짜 화해

AI: 이제 오해 풀렸어, 다행이야.
User: 응, 다행이야. / 미안했어.
AI: 다음엔 바로 말해줘, 진짜 답답했어.
User: 알겠어, 다음엔 바로 말할게. / 응, 다음엔 숨기지 않을게.
User: 앞으로 더 솔직하게 지내자. / 서로 숨기지 말자.
AI: 좋아, 나도 솔직하게 말할게.
User: 그게 좋지. / 당연하지.
AI: 맞아, 이런 일 다시 없을 거야.

# KD-1: 감정 고백

## sim1: 마음의 시작

AI: 저 요즘 뭔가 이상한 것 같아요.
User: 어떻게 이상해요? / 뭐가 달라요?
AI: 뭔가 달라진 것 같아요.
User: 혹시 저 때문이에요? / 저랑 관련 있어요?
User: 얼굴이 빨개진 것 같아요, 괜찮아요? / 얼굴 많이 붉어졌어요.
AI: 그냥 더워서 그래요, 괜찮아요.
User: 오늘 안 더운데요, 솔직하게 말해요. / 덥지 않은데요, 진짜 이유가 뭐예요?
AI: ...솔직하게 말하기가 무서워요.
## sim2: 고백 직전

AI: 저 물어봐도 돼요?
User: 네, 물어봐요. / 뭐든지 물어봐요.
AI: 저를 어떻게 생각해요?
User: 어떤 의미예요? / 무슨 뜻이에요?
User: 저한테 특별한 감정 생겼어요? / 혹시 저 좋아해요?
AI: 잘 모르겠는데, 자꾸 신경 써져요.
User: 저도 비슷한 감정 있었어요. / 사실 저도 같은 마음이에요.
AI: 정말요? 지금 심장이 빨리 뛰고 있어요.
## sim3: 고백

AI: 있잖아, 나 솔직하게 말해도 돼?
User: 응, 말해봐. / 당연하지, 편하게 말해.
AI: 나 너 좋아해.
User: 진짜야? / 정말이야?
AI: 응, 진짜야. 오래 기다렸어.
User: 나도 좋아해. / 나도 같은 마음이야.
AI: 진짜? 너무 좋다!
User: 나도 좋아! / 우리 이제 사귀는 거야.

# KD-2: 이별 후 재회

## sim1: 우연한 재회

AI: 여기서 만날 줄 몰랐어요.
User: 저도요, 진짜 오랜만이에요. / 저도 깜짝 놀랐어요.
AI: 오랜만이네요.
User: 그동안 어떻게 지냈어요? / 잘 지냈어요?
User: 잠깐 이야기할 수 있어요? / 시간 괜찮아요?
AI: 저도 그러고 싶었어요.
User: 네, 저기 카페 갈까요? / 네, 카페에서 이야기해요.
AI: 좋아요, 같이 가요.
## sim2: 과거를 나누다

AI: 예전에 여기 같이 온 적 있었잖아요.
User: 맞아요, 벌써 옛날 일이네요. / 그러게요, 시간 빠르네요.
AI: 저는 가끔 그때 생각나요.
User: 저도요, 좀 아쉬웠어요. / 저도 그랬어요.
User: 그때 잘 끝내지 못해 아쉬웠어요. / 마무리를 잘 못해서 마음에 남았어요.
AI: 저도 그 마음 알아요, 너무 갑자기였잖아요.
User: 이제는 솔직하게 얘기할 수 있을 것 같아요. / 지금은 편하게 말할 수 있어요.
AI: 저도 같은 마음이에요, 다행이에요.
## sim3: 다시 시작

AI: 있잖아, 이제 편하게 얘기해도 돼?
User: 응, 그게 더 편해. / 그래, 편하게 말하자.
AI: 솔직히 오늘 다시 만나서 좋았어.
User: 나도 그랬어, 진짜 반가웠어. / 나도, 오늘 만나서 좋았어.
User: 우리 계속 연락하자. / 이번엔 자주 연락하자.
AI: 그러자.
User: 이번엔 꼭 약속해. / 진짜 연락하자, 약속이야.
AI: 응, 진짜 약속할게.
