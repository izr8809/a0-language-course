# KD-1: 처음 뵙겠습니다

## sim1: 첫 만남 (First Meeting)

User: 처음 뵙겠습니다. 잘 부탁드립니다. / 안녕하세요. 만나 뵙게 되어 반갑습니다.
AI: …네. 저도요.
User: 여기 자주 오시나요? / 여기 자주 오세요?
AI: 가끔 옵니다.
User: 저도 가끔 오는데, 처음 뵙는 것 같아요. / 저도 가끔 오거든요. 처음 뵈는 거죠?
AI: …그렇군요. 몰랐어요.
User: 앞으로 자주 뵐 것 같아요. / 또 뵐 수 있으면 좋겠어요.
AI: 그럴 수도 있겠네요.
## sim2: 어색한 시간 (Awkward Silence)

User: 뭐 드세요? 저는 아메리카노요. / 뭐 마시세요? 저는 아메리카노요.
AI: 저도 아메리카노 마십니다.
User: 취향이 비슷하네요. / 저랑 같은 거 드시네요.
AI: …그런 것 같네요.
AI: 여기 전에도 오신 적 있으세요?
User: 네, 몇 번 왔었어요. / 네, 전에도 온 적 있어요.
AI: 저도 봤어요. 기억해요.
User: …진짜요? 몰랐어요. / 정말요? 전혀 몰랐어요.
## sim3: 묘한 느낌 (Strange Feeling)

User: 오늘 좋은 시간이었습니다. / 오늘 즐거웠습니다.
AI: …저도 그렇게 생각해요.
User: 또 뵐 수 있을까요? / 다음에 또 만날 수 있을까요?
AI: 모르겠어요. 아마도요.
User: 연락드려도 될까요? / 연락해도 괜찮으세요?
AI: …괜찮습니다. 드세요.
User: 감사해요. 기다릴게요. / 고마워요. 연락 기다릴게요.
AI: 네. 확인할게요.

# KD-2: …흔들려요

## sim1: 참는 현준 (Holding It In)

User: 괜찮으세요? 좀 달라 보여요. / 무슨 일 있으세요? 평소랑 달라요.
AI: …괜찮습니다. 신경 쓰지 마세요.
User: 진짜 괜찮으신 거 맞아요? / 정말 괜찮으세요? 걱정돼요.
AI: 네. 그냥 피곤한 거예요.
User: 하실 말씀 있으시면 말씀해 주세요. / 말하고 싶은 거 있으시면 해주세요.
AI: 아무것도 아니에요. 괜찮아요.
User: 그래도 말씀해 주시면 좋겠어요. / 그래도 이야기해 주세요. 듣고 싶어요.
AI: …그냥 생각이 좀 많아요.
## sim2: 감정 장면 (The Moment)

AI: 왜 연락을 안 하셨어요. 기다렸어요.
User: …죄송해요. 몰랐어요. / 미안해요. 정말 몰랐어요.
AI: 기다렸다고요. 오래.
User: 정말 죄송합니다. 몰랐어요. / 진심으로 미안해요. 몰랐어요.
User: 화나신 거예요? 많이 화나셨어요? / 혹시 화났어요? 저한테 화난 거예요?
AI: 화난 거 아니에요. 그냥요.
User: 그럼 뭔데요. 말씀해 주세요. / 그럼 뭐예요? 이야기해 주세요.
AI: …저도 잘 모르겠어요.
## sim3: 수습 (Aftermath)

User: 이제 좀 괜찮으세요? / 지금은 좀 나아지셨어요?
AI: …조금요. 앉아 있으니까요.
User: 솔직하게 말씀해 주셔서 감사해요. / 솔직하게 이야기해 주셔서 고마워요.
AI: 솔직하게 말한 건 처음이에요.
User: 옆에 있어도 될까요? 있고 싶어요. / 같이 있어도 돼요? 가고 싶지 않아요.
AI: …됐어요. 있으세요.
User: 감사해요. 여기 있을게요. / 고마워요. 안 갈게요.
AI: 네. 가지 마세요.

# KD-3: …좋아해요

## sim1: 말 못하는 현준 (He Can't Say It)

User: 왜 말씀이 없으세요? 오늘 좀 달라요. / 왜 조용하세요? 오늘 뭔가 달라요.
AI: 그냥요. 별거 아니에요.
User: 뭔가 하실 말씀 있으신 것 같아요. / 무슨 말 하고 싶은 거 있는 것 같아요.
AI: …아니에요. 그냥 걸어요.
User: 하실 말씀 있으시면 지금 하셔도 돼요. / 말하고 싶으시면 지금 말씀하세요.
AI: …있어요. 근데 어떻게 말해야 할지.
User: 천천히 말씀하셔도 괜찮아요. / 급하지 않아요. 천천히 하세요.
AI: 조금만 기다려 주세요.
## sim2: 돌려서 (Indirect)

AI: 좋아해요. 오래 전부터요.
User: …뭐라고요? 지금 뭐라고 하셨어요? / …네? 방금 뭐라고 하신 거예요?
AI: 좋다고요. 분명히 말했어요.
User: 진짜요? 저한테요? / 정말요? 저를 좋아해요?
User: 저도요. 저도 좋아해요. / 나도요. 나도 좋아해요.
AI: …그래요? 진짜예요?
User: 네. 진짜예요. / 네. 정말이에요.
AI: 다행이에요. 정말로요.
## sim3: 결국엔 (Finally)

User: 저희 이제 사귀는 건가요? / 우리 이제 사귀는 거예요?
AI: …그렇게 생각해도 돼요.
User: 좀 더 확실하게 말씀해 주세요. / 확실하게 대답해 주세요.
AI: 네. 사귀는 거예요.
User: 손 잡아도 될까요? 잡고 싶어요. / 손 잡아도 돼요? 잡고 싶어요.
AI: …잡으세요. 괜찮아요.
User: 손이 차가우시네요. 따뜻하게 해드릴게요. / 손이 차갑네요. 제가 녹여드릴게요.
AI: 네. 고마워요.

# KD-4: …가지 마세요

## sim1: 이별 (Goodbye)

User: 그냥 가세요. 더 이상 못 하겠어요. / 가세요. 이제 힘들어요.
AI: …네. 알겠어요.
User: 가세요! 왜 안 가세요? / 빨리 가세요! 왜 서 있어요?
AI: 못 가겠어요. 그냥은요.
User: 이제 그만해요. 우리 그만해요. / 여기서 끝내요. 우리 그만해요.
AI: …그렇게 하고 싶지 않아요.
User: 안녕히 계세요. 잘 지내세요. / 잘 지내세요. 안녕히 계세요.
AI: 가지 마세요. 제발요.
## sim2: 혼자 (Alone)

User: 잘 지내셨어요? 오래간만이에요. / 오랜만이에요. 잘 지냈어요?
AI: …네. 그냥 지냈어요.
User: 진짜 괜찮으신 거예요? / 정말 괜찮은 거예요?
AI: 모르겠어요. 솔직히요.
User: 죄송해요. 제가 잘못했어요. / 미안해요. 제 잘못이에요.
AI: …됐어요. 괜찮아요.
User: 보고 싶었어요. 많이요. / 많이 보고 싶었어요. 진짜요.
AI: 저도요. 저도 그랬어요.
## sim3: 다시 (Return)

AI: 저예요. 왔어요.
User: …오셨어요. 어떻게 오셨어요? / …왔어요? 왜 온 거예요?
AI: 그냥 왔어요. 보고 싶어서요.
User: 들어오세요. 기다렸어요. / 어서 오세요. 기다리고 있었어요.
User: 다시 할 수 있을까요? 우리요. / 우리 다시 시작할 수 있어요?
AI: …네. 할 수 있어요.
User: 진짜요? 다시 잘할 수 있어요? / 정말요? 이번엔 잘할 수 있어요?
AI: 네. 잘할게요. 믿어주세요.
