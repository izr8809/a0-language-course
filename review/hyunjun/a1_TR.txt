# TR-1: 예약 확인하고 싶은데요

## sim1: 입장

User: 안녕하세요, 예약 확인하고 싶은데요. / 안녕하세요, 예약한 사람인데요.
AI: 성함이 어떻게 되세요?
User: ___이에요. 두 명으로 예약했어요.
AI: 네, 확인됐어요. 이쪽으로 안내해 드릴게요.
User: 혹시 창가 자리로 앉을 수 있을까요? / 창가 쪽 자리 가능할까요?
AI: 잠깐만요, 확인해 볼게요.
User: 네, 감사합니다. / 네, 부탁드려요.
AI: 이쪽 창가 자리 괜찮으세요?
## sim2: 주문

User: 저기요, 스테이크로 주문하고 싶어요. / 저기요, 스테이크 하나 주문할게요.
AI: 굽기는 어떻게 해드릴까요?
User: 미디엄으로 해주세요. / 미디엄으로 부탁드려요.
AI: 네, 알겠습니다. 잠시만 기다려 주세요.
User: 물도 한 잔 주실 수 있어요? / 물 한 잔도 부탁드려요.
AI: 탄산수, 일반 중 어떤 걸로 드릴까요?
User: 일반 물로 주세요. / 그냥 물로 부탁드려요.
AI: 네, 바로 가져다 드릴게요.
## sim3: 마무리

User: 저기요, 계산해 주실 수 있어요? / 저기요, 계산 부탁드려요.
AI: 카드로 하시겠어요, 현금으로 하시겠어요?
User: 카드로 할게요. / 카드로 결제할게요.
AI: 네, 여기 영수증이에요.
User: 잘 먹었습니다. 정말 맛있었어요. / 잘 먹었어요. 너무 맛있었어요.
AI: 감사합니다. 또 오세요!
User: 네, 꼭 다시 올게요. / 네, 다음에 또 올게요.
AI: 안녕히 가세요.

# TR-2: 이거 다른 사이즈 있어요?

## sim1: 입장

User: 안녕하세요, 이 제품 혹시 있어요? / 안녕하세요, 이거 재고 있나요?
AI: 네, 있어요. 어떤 걸 찾고 계세요?
User: 이 모델로 보고 싶어요. / 이 모델 좀 보여주세요.
AI: 이쪽으로 오세요. 보여드릴게요.
User: 혹시 다른 사이즈나 색상도 있어요? / 다른 사이즈나 색깔도 있나요?
AI: 네, 검정이랑 네이비 있어요.
User: 검정으로 보여주실 수 있어요? / 검정색으로 볼 수 있을까요?
AI: 네, 잠깐만요.
## sim2: 계산

User: 이걸로 할게요. 카드 되나요? / 이걸로 살게요. 카드 결제 돼요?
AI: 네, 카드 됩니다. 여기 대주세요.
User: 네, 감사합니다. / 네, 고맙습니다.
AI: 영수증 드릴까요?
User: 네, 영수증 주세요. / 네, 영수증 부탁드려요.
AI: 네, 여기 있어요.
User: 혹시 교환은 어디서 하나요? / 교환하려면 어디로 가야 해요?
AI: 3층 고객 서비스 센터로 가시면 돼요.
## sim3: 교환

User: 안녕하세요, 이거 교환하고 싶은데요. / 안녕하세요, 이거 교환 가능할까요?
AI: 영수증 가지고 계세요?
User: 네, 여기 있어요. / 네, 여기요.
AI: 잠깐만요, 확인해 드릴게요.
User: 이 색상으로 바꿀 수 있을까요? / 이 색깔로 교환할 수 있어요?
AI: 네, 바꿔드릴게요. 잠시만요.
User: 감사합니다. 정말 도움이 됐어요. / 고맙습니다. 덕분에 잘 됐어요.
AI: 감사합니다. 또 오세요.

# TR-3: 아이스 아메리카노 한 잔 주세요

## sim1: 주문

User: 안녕하세요, 아이스 아메리카노 한 잔 주세요. / 안녕하세요, 아이스 아메리카노 하나요.
AI: 사이즈는 어떻게 해드릴까요?
User: 라지로 주세요. / 큰 사이즈로 주세요.
AI: 네, 알겠습니다.
User: 혹시 케이크 종류가 어떻게 돼요? / 케이크는 뭐가 있어요?
AI: 치즈 케이크랑 초코 케이크 있어요.
User: 치즈 케이크 하나 주세요. / 치즈 케이크로 할게요.
AI: 네, 같이 준비해 드릴게요.
## sim2: 결제

User: 카드로 결제해도 될까요? / 카드로 해도 돼요?
AI: 네, 카드 됩니다. 여기 대주세요.
User: 네, 감사합니다. / 네, 알겠습니다.
AI: 이름이 어떻게 되세요?
User: ___이에요.
AI: 나오면 불러 드릴게요.
User: 네, 감사합니다. / 네, 고맙습니다.
AI: 잠깐만 기다려 주세요.
## sim3: 자리

User: 혹시 자리가 어디 있어요? / 빈자리가 어디 있나요?
AI: 안쪽에 자리 있어요. 이쪽으로 오세요.
User: 네, 감사합니다. / 네, 알겠어요.
AI: 네!
User: 저기요, 이게 제 주문 맞나요? / 저기요, 이거 제 거 맞아요?
AI: 네, 아이스 아메리카노 라지예요.
User: 감사합니다! / 고맙습니다!
AI: 맛있게 드세요.

# TR-4: 강남역으로 가주세요

## sim1: 탑승

User: 안녕하세요, 강남역으로 가주세요. / 안녕하세요, 강남역까지 부탁드려요.
AI: 네, 타세요. 출발할게요.
User: 감사합니다. / 네, 고맙습니다.
AI: 안전벨트 매주세요.
User: 혹시 많이 막혀요? 얼마나 걸릴까요? / 길이 많이 막히나요? 시간 얼마나 걸려요?
AI: 지금 좀 막히는데, 한 15분 정도 걸려요.
User: 아, 괜찮아요. 감사합니다. / 아, 네. 알겠습니다. 감사해요.
AI: 네.
## sim2: 이동 중

User: 저기요, 저기서 세워 주실 수 있어요? / 저기요, 저기서 내려주실 수 있어요?
AI: 여기서 세워 드릴까요?
User: 네, 저기 앞에서요. / 네, 저 앞쪽에서요.
AI: 네, 알겠습니다.
User: 다 왔나요? 거의 다 온 것 같은데요. / 거의 다 왔어요? 다 온 것 같은데요.
AI: 네, 다 왔어요. 조심히 내리세요.
User: 감사합니다! / 고맙습니다!
AI: 네.
## sim3: 하차

User: 카드로 결제해도 될까요? / 카드로 해도 될까요?
AI: 네, 카드 됩니다. 여기 단말기예요.
User: 네, 감사합니다. / 네, 고맙습니다.
AI: 감사합니다.
User: 감사합니다. 수고하셨어요. / 고맙습니다. 수고하세요.
AI: 네, 감사합니다.
User: 안녕히 가세요! / 조심히 가세요!
AI: 네, 조심히 들어가세요.
