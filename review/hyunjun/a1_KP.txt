# KP-1: 왔어요?

## sim1: First Sign

User: 안녕하세요! 만나서 정말 반가워요! / 안녕하세요! 드디어 만났어요!
AI: …네, 앉아요. 기다렸어요?
User: 네, 오래 기다렸어요. 드디어 왔어요! / 네, 많이 기다렸어요. 진짜 왔어요!
AI: 수고했어요. 멀리서 왔어요?
User: 사인 해주세요! 이름도 써줄 수 있어요? / 사인 부탁해요! 이름도 적어줄 수 있어요?
AI: 이름 말해요.
User: ___이에요! 잘 부탁드려요!
AI: …됐어요. 천천히 썼어요.
## sim2: He Noticed

AI: 전에도 온 적 있어요?
User: 네! 항상 와요. 기억해요? / 네! 매번 와요. 알아봤어요?
AI: …알아요. 봤어요.
User: 진짜요? 기억해줘서 감사해요! / 정말요? 기억해줘서 고마워요!
AI: 좋아해줘서 고마워요.
User: 저야말로요! 앞으로도 계속 올게요! / 저도 감사해요! 다음에도 꼭 올게요!
AI: …그래요. 기다릴게요.
User: 정말요? 꼭 또 올게요! / 진짜요? 다음에 또 올게요!
## sim3: Goodbye (For Now)

User: 이거 드리고 싶어요. 직접 준비했어요. / 이거 선물이에요. 직접 만들었어요.
AI: …뭐예요. 고마워요.
User: 편지예요. 꼭 읽어줘요! / 편지 써왔어요. 나중에 읽어줘요!
AI: …받을게요. 읽을게요.
AI: 조심히 가요. 멀리서 왔잖아요.
User: 네, 감사해요! 정말 행복했어요! / 네, 고마워요! 오늘 너무 좋았어요!
AI: …저도요. 또 봐요.
User: 꼭이요! 다음에 또 올게요! / 약속이에요! 다음에 꼭 올게요!

# KP-2: 무대 위에서

## sim1: Before the Show

AI: 왔어요? 오래 기다렸어요?
User: 네! 많이 기다렸어요. 드디어 왔어요! / 네! 오래 기다렸어요. 진짜 설레요!
AI: …수고했어요. 잘 봐줘요.
User: 물론이죠! 정말 기대돼요! / 당연하죠! 너무 기대돼요!
User: 떨리지 않아요? 저는 엄청 떨려요! / 긴장 안 돼요? 저는 너무 떨려요!
AI: …아니요. 별로 안 떨려요.
User: 진짜요? 대단해요. 잘 할 거예요! / 정말요? 멋있어요. 응원할게요!
AI: 알아요. 잘 볼게요.
## sim2: During the Show

AI: 왔어요? 잘 보이는 자리예요?
User: 네! 잘 보여요. 최고예요! / 네! 다 보여요. 너무 좋아요!
AI: …다행이에요. 계속 봐줘요.
User: 당연하죠! 눈 못 떼겠어요! / 물론이죠! 계속 보고 있어요!
User: 앵콜! 앵콜! 한 곡 더 해줘요! / 앵콜! 앵콜! 한 번 더 해줘요!
AI: …네. 알겠어요.
User: 진짜 나왔어요! 너무 좋아요! / 정말 나왔어요! 진짜 최고예요!
AI: 봐줘서 고마워요.
## sim3: After the Show

User: 진짜 너무 좋았어요! 어떠셨어요? / 정말 대박이었어요! 기분 어때요?
AI: …그래요. 잘 봐줘서 고마워요.
User: 수고 많았어요! 정말 멋있었어요! / 고생했어요! 진짜 멋있었어요!
AI: 네. 고마워요.
AI: 조심히 가요. 멀리서 왔잖아요.
User: 네, 감사해요! 다음에 또 올게요! / 네, 고마워요! 꼭 다시 올게요!
AI: …또 와요. 기다릴게요.
User: 꼭이요! 잘 들어가요! / 약속이에요! 조심히 가요!

# KP-3: …봤어요

## sim1: He Went Live

AI: …왔어요? 이름 보였어요.
User: 네! 기다렸어요. 보고 싶었어요! / 네! 기다리고 있었어요. 반가워요!
AI: 그래요. 수고했어요.
User: 오늘 공연 진짜 수고 많았어요! / 오늘 무대 정말 고생했어요!
AI: 뭐 했어요? 오늘 바빴어요?
User: 라이브 기다렸어요! 계속 기다렸어요! / 라이브 기다리고 있었어요! 너무 반가워요!
AI: …그래요. 고마워요.
User: 사랑해요! 항상 응원하고 있어요! / 사랑해요! 언제나 응원할게요!
## sim2: The Reply

AI: …봤어요. 다 읽었어요.
User: 진짜요? 읽어줘서 정말 감사해요! / 정말요? 읽어줘서 너무 고마워요!
AI: 네. 잘 썼어요.
User: 감사해요! 앞으로도 응원할게요! / 고마워요! 계속 응원할게요!
AI: 뭐가 제일 좋았어요? 궁금해요.
User: 다 좋아요! 다 좋았어요! / 전부 좋았어요! 하나도 안 빠져요!
AI: …그래요. 고마워요.
User: 항상 열심히 해줘서 감사해요! / 언제나 열심히 해줘서 고마워요!
## sim3: Sending Love

User: 오늘 라이브 정말 좋았어요. 고마워요! / 오늘 라이브 너무 좋았어요. 감사해요!
AI: …네. 봐줘서 고마워요.
User: 자주 해줘요! 항상 기다리고 있어요! / 자주 켜줘요! 매번 기다릴게요!
AI: 노력해볼게요. 고마워요.
AI: 이제 자요. 많이 피곤하죠?
User: 네, 잘 자요! 좋은 꿈 꿔요! / 네, 푹 쉬어요! 좋은 꿈 꿔요!
AI: …네. 너도 잘 자요.
User: 감사해요! 내일도 응원할게요! / 고마워요! 내일도 응원할게요!
