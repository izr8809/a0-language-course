# KD-1: 처음 뵙겠습니다

## sim1: 첫 만남 (First Meeting)

User: 처음 뵙겠습니다.
AI: 현준이에요. 반가워요. 여기 앉으세요.
User: 잘 부탁드립니다. / 감사합니다.
AI: 뭐 드릴까요? 커피 괜찮아요?
User: 여기 자주 오세요? / 여기 처음이에요.
AI: 자주 와요. 조용해서 좋아요. 어떠세요?
User: 좋아요. / 분위기 좋네요.
AI: 다행이에요. 편하게 계세요.
## sim2: 어색한 시간 (Awkward Silence)

User: 뭐 하세요? / 무슨 일 하세요?
AI: 회사 일이에요. 왜요?
User: 궁금해서요. / 그냥요.
AI: 궁금해요? 좋아요. 더 물어보세요.
AI: 추우세요? 여기 바람 들어와요.
User: 괜찮아요. / 조금요.
AI: 잠깐만요. 자리 바꿀게요.
User: …감사합니다.
## sim3: 묘한 느낌 (Strange Feeling)

User: 오늘 좋았습니다. / 즐거웠어요.
AI: 저도요. 차 불러드릴까요?
User: 괜찮아요. / 감사합니다.
AI: 조심히 가세요. 도착하면 연락 주세요.
User: 또 만날 수 있어요? / 또 봬요.
AI: 제가 연락할게요. 언제 편해요?
User: 언제든지요. / 주말이요.
AI: 좋아요. 연락할게요.

# KD-2: …흔들려요

## sim1: 차가운 현준 (Cold Distance)

User: 괜찮으세요? / 오늘 왜 그러세요?
AI: 괜찮아요. 일이 좀 많아요.
User: 진짜요? / 그래요?
AI: 진짜예요. 신경 쓰지 마세요.
AI: 아까 그 사람 누구예요?
User: 친구예요. / 왜요?
AI: …그래요. 알겠어요.
User: 왜 물어보세요? / 신경 쓰여요?
## sim2: 감정 장면 (The Crack)

AI: 저 기분 이상해요.
User: 뭐가요? / 무슨 말이에요?
AI: 모르겠어요. 저도 몰라요.
User: 천천히 말씀하세요. / 괜찮아요.
User: 솔직하게 말씀해 주세요. / 저한테 말해 주세요.
AI: 신경 쓰여요. 자꾸요.
User: 저요? / 제가요?
AI: 네. 이유는 모르겠어요. 
## sim3: 수습 (Aftermath)

User: 현준 씨. / 괜찮으세요?
AI: 괜찮아요.  앉으세요.
User: 여기요? / 네 고마워요.
AI: 좀 같이 있어도 돼요?
User: 옆에 있을게요. / 같이 있어요.
AI: 고마워요. 처음이에요.
User: 뭐가요? / 처음이요?
AI: 이렇게 편한 거요. 사람이랑.

# KD-3: …좋아해요

## sim1: 보여드릴 게 있어요 (Something to Show You)

AI: 왔어요? 따라오세요.
User: 어디 가요? / 무슨 일이에요?
AI: 보여드릴 게 있어요. 가요.
User: 네 가요. / 알겠어요.
AI: 여기예요. 어떠세요?
User: 와, 예뻐요. / 여기 처음이에요.
AI: 제가 좋아하는 곳이에요. 처음 보여주는 거예요.
User: 진짜요? / 그래요?
## sim2: 돌려서 (The Confession)

AI: 여기 앉으세요. 드릴 게 있어요.
User: 뭐예요? / 선물이에요?
AI: 선물이에요. 열어보세요.
User: 감사합니다. / 열어도 돼요?
User: 왜 이렇게 잘해주세요? / 감사해요.
AI: 처음이에요. 이런 마음.
User: 무슨 마음이요? / …현준 씨.
AI: 좋아해요. 제가 할 수 있는 말은 이게 다예요.
## sim3: 결국엔 (Finally)

User: 저도 좋아해요. / 저도요.
AI: …진짜예요? 확인하고 싶어요.
User: 진짜예요. / 네, 진심이에요.
AI: 다행이에요. 제가 잘 할게요.
AI: 추워요? 들어갈까요?
User: 조금 더 있고 싶어요. / 괜찮아요.
AI: 그래요. 가까이 오세요.
User: 괜찮아요 / 네 알았어요.

# KD-4: …돌아올게요

## sim1: 떠남 (Departure)

AI: 할 말이 있어요. 앉으세요.
User: 네. / 뭔데요?
AI: 해외 출장이에요. 한동안 못 봐요.
User: …언제요? / 갑자기요?
User: 얼마나요? / 오래 가요?
AI: 한 달이요. 어쩔 수 없어요.
User: 알겠어요. / 아쉽네요
AI: 기다려 줄 수 있어요?
## sim2: 혼자 (Apart)

AI: 잘 지내요? 거기 날씨 어때요?
User: 보고 싶어요. / 잘 지내요.
AI: 저도요. 바빠서 연락 못 했어요. 죄송해요.
User: 괜찮아요. / 바쁘죠.
User: 언제 와요? / 많이 남았어요?
AI: 곧이요. 조금만 기다려 주세요.
User: 기다릴게요. / 빨리 오세요.
AI: 꼭 갈게요. 약속이에요.
## sim3: 다시 (Return)

AI: 왔어요.
User: 오랜만이요 / 돌아왔어요?
AI: 네. 돌아왔어요. 보고 싶었어요.
User: 저도요. / 많이 보고 싶었어요.
User: 이제 안 가요? / 또 가요?
AI: 안 가요. 여기 있을게요.
User: 진짜요? / 약속이에요?
AI: 약속이에요. 이번엔 안 가요.
