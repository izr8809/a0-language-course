# KD-1: 나 요즘 좋아하는 사람 있어

## sim1: Confession to a Friend

User: 나 비밀 하나 있어. / 나 할 말 있어.
AI: 뭔데? 말해 봐!
User: 나 요즘 좋아하는 사람 있어. / 나 누구 좋아하게 됐어.
AI: 헐, 진짜? 누구?!
AI: 혹시 내가 아는 사람이야?
User: 응. / 아니, 몰라.
AI: 어떤 사람인데?
User: 진짜 착해. / 웃을 때 예뻐.
## sim2: Details

AI: 어디에서 만났어?
User: 카페에서! / 수업에서!
AI: 로맨틱하다! 그래서 어떻게 됐어?
User: 아직 아무 말도 못 했어. / 번호는 받았어!
AI: 고백 안 할 거야?
User: 하고 싶은데 무서워. / 아직 일러.
AI: 너 좋은 사람이야! 자신 있게 해!
User: 진짜? 고마워. / 그 말 듣으니까 힘 나!
## sim3: Encouragement

AI: 고백하면 알려 줘! 응원할게!
User: 알겠어! / 떨리는데 해 볼게.
AI: 파이팅!
User: 고마워! / 진짜 든든하다!
AI: 잘 될 거야!
User: 그랬으면 좋겠다. / 나도 그렇게 생각해.
AI: 당연하지! 연락해!
User: 응! / 알겠어!

# KD-2: 오해하지 마

## sim1: Misunderstanding

AI: 어제 왜 다른 사람이랑 있었어?
User: 뭐? 무슨 말이야? / 오해하지 마!
AI: 나 봤어. 카페에서.
User: 그 사람 그냥 친구야! / 아, 그때?
AI: 진짜 그냥 친구야?
User: 응, 진짜야. 같은 수업 듣는 애야. / 의심하지 마.
AI: 미안. 나 좀 질투했나 봐.
User: 괜찮아. / 그럴 수 있어.
## sim2: Talking It Out

User: 왜 바로 물어보지 않았어? / 그냥 나한테 말하지.
AI: 물어보기가 좀 그랬어.
User: 다음에는 바로 물어봐. 오해하지 말고. / 다음부터는 나한테 먼저 말해.
AI: 알겠어, 미안해.
AI: 나 진짜 미안해.
User: 괜찮아. 근데 나 믿어 줘. / 됐어. 다음부터 잘하자.
AI: 응, 믿어. 다시는 안 그럴게.
User: 알겠어. / 약속이다.
## sim3: Making Up

AI: 화 풀렸어?
User: 풀렸어. / 거의.
AI: 맛있는 거 먹으러 가자. 내가 살게.
User: 좋아! / 뭐 먹으러 갈까?
AI: 다음엔 바로 물어볼게!
User: 그래! / 꼭 그렇게 해!
AI: 좋아해. 알지?
User: 나도. / 알지.

# KD-4: 요즘 왜 이래?

## sim1: Something Changed

User: 요즘 왜 연락 안 해? / 요즘 왜 이래?
AI: 미안, 요즘 좀 바빠서.
User: 바쁜 건 알겠는데, 너무 안 하잖아. / 그래도 한마디는 할 수 있잖아.
AI: 미안해.
User: 나 신경 안 쓰는 것 같아서 속상해. / 솔직히 좀 서운했어.
AI: 아니야, 진짜 일 때문에 그래. 미안해.
User: 알겠어. / 그래도 연락은 해 줘.
## sim2: Honest Talk

AI: 요즘 진짜 힘들었어. 너한테 잘 못했어.

User: 바보야. 나한테는 얘기해도 돼. / 왜 혼자 참아? 나한테 말해.
AI: 고마워. 앞으로 잘할게.
User: 나도 미안해, 좀 심하게 말했어. / 나도 너무 화냈어. 미안.
AI: 아니야, 네 말이 맞아.
User: 앞으로 서로 솔직하자. / 우리 숨기지 말고 다 얘기하자.
## sim3: Better

User: 아무거나! 같이 있으면 돼. / 너랑 먹으면 다 맛있어.
