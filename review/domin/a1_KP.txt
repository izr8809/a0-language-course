# KP-1: 오빠 생일 축하해요!

## sim1: Birthday Message

User: 생일 축하해요! / 오빠, 생일 축하해요!
AI: 감사해요! 고마워요!
User: 오늘 뭐 했어요? / 생일에 뭐 했어요?
AI: 멤버들이랑 케이크 먹었어요!
User: 선물 보냈어요! 받았어요? / 저 선물 보냈는데 도착했어요?
AI: 아직 못 받았어요! 뭔데요?
User: 비밀이에요! / 편지랑 선물이요!
AI: 뭔지 진짜 궁금해요! 힌트 주세요!
## sim2: Fan Event

AI: 다들 생일 축하해 줘서 감사해요!
User: 축하해요! / 사랑해요!
AI: 팬분들 덕분에 행복해요!
User: 저도요. 행복하세요! / 저도 행복해요! 
AI: 소원 빌었어요! 뭔지 알아요?
User: 뭔데요? / 말해 주세요!
AI: 팬분들이랑 오래오래 함께하고 싶어요!
User: 감동이에요! / 저도요!
## sim3: Ending

AI: 오늘 진짜 행복했어요!
User: 저도 행복해요! / 저도요! 최고예요!
AI: 내년 생일에도 같이 해요!
User: 당연하죠! / 약속이에요!
AI: 이제 잘 거예요! 잘 자요!
User: 잘 자요! / 좋은 꿈 꾸세요!
AI: 사랑해요!
User: 저도 사랑해요! / 저도요! 많이 사랑해요!
