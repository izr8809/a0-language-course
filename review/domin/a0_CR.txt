# CR-1: 안녕하십니까

## sim1: Morning

User: 안녕하십니까! / 안녕하세요!
AI: 안녕하세요! 오늘 날씨 좋죠?
User: 네, 좋습니다! / 그러네요!
AI: 오늘도 화이팅이에요!
AI: 오늘 회의 있어요. 알고 있죠?
User: 네, 알고 있습니다. / 네, 들었습니다.
AI: 좋아요. 10시에 회의실이에요.
User: 네, 알겠습니다. / 네, 감사합니다.
## sim2: During the Day

AI: 점심 먹었어요?
User: 네, 먹었습니다. / 아직입니다.
AI: 같이 먹을래요?
User: 네, 좋습니다. / 그럴까요?
AI: 이거 좀 봐 주실 수 있어요?
User: 네, 보겠습니다. / 네, 알겠습니다.
AI: 감사합니다!
User: 아닙니다. / 별말씀을요.
## sim3: Leaving

AI: 오늘 수고 많으셨습니다!
User: 수고하셨습니다! / 고생하셨습니다!
User: 먼저 가보겠습니다. / 퇴근하겠습니다.
AI: 네, 조심히 가세요!
User: 내일 뵙겠습니다! / 감사합니다!
AI: 네, 내일 봐요!
User: 안녕히 가세요! / 조심히 가세요!
AI: 안녕히 가세요!

# CR-2: 네, 알겠습니다

## sim1: Getting Instructions

AI: 이거 오늘까지 해 주세요.
User: 네, 알겠습니다. 
AI: 질문 있으면 말해 주세요.
User: 알겠습니다. / 지금은 없습니다.
AI: 다 했어요?
User: 거의 다 했습니다. / 조금 더 걸립니다.
AI: 알겠어요. 천천히 하세요.
User: 감사합니다. / 네, 알겠습니다.
## sim2: Meeting

AI: 의견 있으면 말해 주세요.
User: 아직 없습니다. / 네, 알겠습니다.
AI: 이 부분 어떻게 생각해요?
User: 좋은 것 같습니다. / 좋습니다.
AI: 그러면 이렇게 하겠습니다.
User: 좋습니다. / 네, 알겠습니다.
AI: 수고하셨습니다!
User: 수고하셨습니다! / 고생하셨습니다!
## sim3: End of Day

User: 다 끝났습니다. 확인해 주세요. / 다 했습니다.
AI: 네, 감사합니다! 바로 볼게요.
AI: 잘 했어요!
User: 감사합니다. / 더 할 거 있을까요?
AI: 오늘도 수고하셨습니다!
User: 수고하셨습니다! / 오늘도 고생하셨습니다!
User: 먼저 가보겠습니다. / 먼저 갈게요.
AI: 네, 수고하세요!

# CR-3: 다시 말씀해 주시겠어요?

## sim1: I Don't Understand

AI: 이거 오늘까지 이메일로 보내주세요.
User: 다시 말씀해 주시겠어요? / 죄송합니다, 못 들었습니다.
AI: 네, 이거 오늘까지 이메일로 보내주세요.
User: 감사합니다. / 네, 알겠습니다.
AI: 이해되셨어요?
User: 네, 알겠습니다. / 이 부분이 잘 모르겠습니다.
AI: 이 부분은 이렇게 하시면 돼요.
User: 네. 알겠습니다. / 이해했습니다.
## sim2: Asking for Help

AI: 이렇게 하시면 됩니다!
User: 그렇군요! / 감사합니다!
AI: 또 모르는 거 있으면 말해 주세요!
