# PI-1: 안녕하세요! 저는 ___이에요

## sim1: Nice to Meet You

AI: 안녕하세요. 서연이에요.
User: 저는 ___이에요!
AI: 어디에서 왔어요?
User: ___에서 왔어요!
AI: 멀리서 왔네요. 한국 좋아해요?
User: 네, 좋아해요! / 많이요!
AI: 뭐가 제일 좋아요?
User: 드라마요! / 음식이요!
## sim2: Getting Closer

User: 서연 씨, 취미 뭐예요? / 뭐 좋아해요?
AI: 커피요. 마시는 거 좋아해요.
User: 저도요! / 오, 진짜요?
AI: 괜찮네요. 다음에 같이 마셔요.
AI: 밥 먹었어요?
User: 아직이요! / 조금요!
AI: 같이 먹어요. 제가 정해줄게요.
User: 네! / 좋아요!
## sim3: Let's Hang Out

AI: 우리 말 놓을까요?
User: 네! / 좋아요!
AI: 나는 서연이야. 편하게 해.
User: 응! 나는 ___야!
User: 주말에 뭐 해? / 이번 주말 시간 있어?
AI: 아직 안 정했어. 왜?
User: 같이 놀래? / 놀러 가자!
AI: 좋아. 시간은 내가 정할게.

# PI-3: 성수 갈래?

## sim1: This Weekend

User: 주말에 뭐 해? / 이번 주말 일정 있어?
AI: 아직. 왜?
User: 같이 놀래? / 같이 밥 먹으러 갈래?
AI: 좋아. 내가 정할게.
User: 샐러드 먹으러 갈래? / 카페 갈래?
AI: 성수에 좋은 데 알아. 거기 가자.
User: 좋아! / 가자!
AI: 아침은 먹었어? 또 안 먹었지.
## sim2: When and Where

AI: 몇 시에 만날까?
User: 2시? / 오후에!
AI: 성수역 2번 출구. 늦으면 안 돼.
User: 응! / 알겠어!
AI: 그럼 토요일 2시, 성수역.
User: 응! / 맞아!
AI: 10분 전에 와. 늦으면 나 간다.
User: 알겠어! / 안 늦어!
## sim3: Day of

AI: 도착했어?
User: 거의 다 왔어! / 나가고 있어!
AI: 나는 도착했어. 2번 출구 앞이야.
User: 금방 가! / 기다려!
User: 어디야? / 어디에 있어?
AI: 여기야. 손 흔들었어.
User: 아, 봤어! / 찾았어!
AI: 일단 밥. 내가 골라놨어.

# PI-4: 카톡 해!

## sim1: First Text

User: 뭐 해? / 지금 바빠?
AI: 카페 있어. 너는?
User: 나는 집! 심심해. / 나는 집이야!
AI: 심심하면 나와. 밥 사줄게.
AI: 오늘 뭐 했어?
User: 영화 봤어. / 드라마 봤어.
AI: 뭐 봤어? 재밌으면 나도 알려줘.
User: 새로 나온 거! / 비밀!
## sim2: Chatting More

AI: 나 지금 여기야.
User: 어디야? / 와, 좋다!
AI: 성수 카페. 다음에 같이 가자.
User: 가자! / 응, 가고 싶어!
User: 이거 봤어? / 이거 알아?
AI: 뭐야? 안 봤어.
User: 진짜 웃겨! 봐봐! / 재밌어! 봐!
AI: 알겠어. 볼게.
## sim3: Good Night

AI: 밥 먹었어?
User: 응, 먹었어. / 아직!
User: 너는? / 뭐 먹었어?
AI: 나는 샐러드 먹었어.
AI: 나 잘 거야. 너도 자.
User: 잘 자! / 좋은 밤!
User: 내일 봐! / 내일 연락해!
AI: 응. 내일 봐.
