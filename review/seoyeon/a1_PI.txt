# PI-1: 안녕하세요, 처음 뵙겠습니다

## sim1: Nice to Meet You

AI: 안녕하세요. 서연이에요. 이름이 뭐예요?
User: 안녕하세요! 저는 ___이에요. / 반갑습니다! ___이에요.
AI: 네, 반가워요. 한국에 언제 왔어요?
User: ___달 전에 왔어요! / 올해 왔어요!
AI: 한국에 왜 왔어요?
User: 한국어 배우러 왔어요! / 일하러 왔어요!
AI: 그래요? 한국 생활 괜찮아요?
User: 재밌어요! / 좀 어려워요.
## sim2: Getting to Know Each Other

AI: 한국에서 뭐가 제일 좋아요?
User: 음식이요! / 카페가 좋아요!
AI: 그래요? 저는 카페 좋아해요.
User: 진짜요? / 여기도 좋아요!
User: 서연 씨는 취미가 뭐예요? / 취미 있으세요?
AI: 커피 마시면서 책 읽는 거요.
User: 좋아요! 저도 책 좋아해요! / 멋있어요!
AI: 고마워요.
## sim3: Dropping Formality

AI: 우리 편하게 말 놓을까요?
User: 네, 좋아요! / 그럴까요?
AI: 나는 서연이야. 편하게 해.
User: 응! 나는 ___야!
AI: 카톡 있어? 아이디 줘.
User: 여기! 내 아이디야! / 있어! 여기
AI: 저장했어. 다음에 또 보자.
User: 연락해! / 응, 좋아.

# PI-2: 우리 밥 먹으러 가자

## sim1: Choosing a Place

User: 배고파! 뭐 먹을까? / 밥 먹으러 갈까?

User: 삼겹살 먹고 싶어! / 삼겹살 어때?

AI: 가까워. 10분이면 돼. 가자.
User: 응, 걸어가자! / 얼마나 걸리는데?
AI: 걸어가면 돼. 따라와.
User: 알겠어! / 좋아, 가자!
## sim2: At the Restaurant

AI: 여기야. 내가 자주 오는 데야.
User: 와, 좋다! / 사람 많다!
AI: 뭐 먹을지 정해. 메뉴판 여기.
User: 삼겹살 2인분 / 된장찌개!
AI: 내가 구울게. 앉아 있어.
User: 응, 구워줘! / 도와줄게!
AI: 잘 익었어. 먹어.
User: 진짜 맛있다! / 여기 잘 찾았다!
## sim3: After Eating

User: 내가 낼게! / 오늘 내가 살게!
AI: 됐어. 내가 낼게. 다음에 네가 사.
User: 아니야, 오늘 내가 차례야! / 괜찮아, 내가 낼게!
AI: ...알겠어. 다음엔 내가 낸다.
AI: 배불러. 이제 뭐 할 거야?
User: 카페 갈까? / 좀 걸을까?
AI: 좋아. 가자.
User: 가자! / 기대된다.

# PI-3: 너 때문에 한국어 배워

## sim1: Getting Personal

AI: 한국어 왜 배워?
User: 한국 친구 사귀고 싶어서! / 드라마 알아듣고 싶어서!
AI: 그래? 좋은 이유네.
User: 그치? / 진짜? 고마워!
AI: 한국에서 힘든 거 없어?
User: 가끔 외로워. / 말이 잘 안 통할 때.
AI: 힘들면 말해. 혼자 고민하지 마.
User: 고마워. / 응, 그럴게.
## sim2: Encouragement

AI: 한국어 처음 배울 때 어땠어?
User: 진짜 어려웠어! / 하나도 몰랐어.
AI: 근데 지금 잘하잖아.
User: 그래? 고마워! / 아직 많이 부족해.
User: 너 때문에 한국어 더 열심히 하고 싶어. / 너랑 더 많이 이야기하고 싶어.
AI: ...고마워. 그런 말 처음 들어.
User: 진짜야! / 정말이야!
AI: 나도 너 때문에 영어 배우고 싶어.
## sim3: Real Friends

AI: 우리 알게 된 지 얼마나 됐지?
User: 한 달? / 벌써 두 달!
AI: 벌써? 빠르네.
User: 맞아! / 시간 빠르다!
AI: 앞으로도 자주 봐.
User: 당연하지! / 나도 그러고 싶어!
AI: 다음 주 토요일 비어? 잡아 놓을게.
User: 좋아! 연락해! / 응! 토요일 좋아!

# PI-4: 고민 있어?

## sim1: Something's Wrong

AI: 말해. 듣고 있을게.

User: 혼자 있는 시간이 많아서. / 한국어가 아직 어려워서.
AI: 그랬구나. 알겠어.
User: 응, 좀. / 많이 힘들었어.
## sim2: Advice

AI: 혼자 고민하지 마.
User: 고마워. / 그럴게.

AI: 한강 가자. 바람 쐬면 나아져.
User: 그래, 가자! / 좋아, 나가자!
AI: 편의점 들려. 내가 살게.
User: 좋아! / 편의점 가자!
## sim3: Feeling Better

AI: 좀 나아졌어?
User: 응, 많이 나아졌어! / 덕분에!
AI: 그럼 됐어.
User: 진짜 고마워. / 덕분이야.
AI: 다음에 힘들면 바로 연락해.
User: 응, 그럴게! / 고마워!
AI: 약속이야. 안 지키면 안 돼.
User: 약속! / 응, 알겠어!

# PI-5: 같이 여행 갈까?

AI: 며칠 갈 거야? 정하자.
