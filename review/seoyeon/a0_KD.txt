# KD-1: 처음이야

## sim1: 첫 인사 (First Greeting)

User: 처음 만나네요. / 안녕하세요.
AI: 어. 여기 앉아.
User: 잘 부탁드려요. / 반가워요.
AI: 밥 먹었어?
User: 여기 자주 와요? / 자주 오세요?
AI: 가끔. 너는?
User: 저도요. / 가끔 와요.
AI: 혼자 다니지 마. 다음엔 같이 가.
## sim2: 어색한 시간 (Awkward Time)

User: 뭐 마셔요? / 뭐 마실 거예요?
AI: 아메리카노. 너는?
User: 저도요. / 같은 거요.
AI: 따뜻한 거 마셔. 시켜줄게.
AI: 안 추워?
User: 괜찮아요. / 안 추워요.
AI: 추울 줄 알았어. 이거.
User: …감사해요. / 고마워요.
## sim3: 묘한 느낌 (Strange Feeling)

User: 오늘 좋았어요. / 재밌었어요.
AI: 그래? 조심히 가.
User: 네. / 잘 가요.
AI: 도착하면 카톡 해. 꼭.
User: 또 만나요? / 다음에 또 봐요?
AI: 당연하지. 내가 연락할게.
User: 연락할게요. / 기다릴게요.
AI: 내가 먼저 할 거야. 기다려.

# KD-2: 왜 그래

## sim1: 서연이 이상해 (Something's Off)

User: 괜찮아? / 무슨 일이야?
AI: 괜찮아. 별일 아니야.
User: 진짜? / 정말?
AI: 진짜. 신경 쓰지 마.
User: 말해봐. / 얘기해.
AI: 이따가 말할게.
User: 말해봐. / 지금 해.
AI: 지금은 아니야. 이따가.
## sim2: 감정 (Feelings)

AI: 연락 안 하면 어떡해.
User: …미안해. / 미안.
AI: 기다렸잖아. 힘들었어.
User: 몰랐어. 미안. / 진짜 미안해.
User: 화났어? / 나한테 화났어?
AI: 화 안 났어.
User: 그럼 뭔데. / 그럼 왜?
AI: 걱정했다고. 다음엔 꼭 해.
## sim3: 수습 (Aftermath)

User: 괜찮아? / 지금 어때?
AI: …괜찮아. 너부터 말해.
User: 솔직하게. / 진짜로 말해.
AI: …좀 힘들었어. 근데 괜찮아.
User: 있어줄게. / 안 갈게.
AI: 가지 마. 좀 더 있어.
User: 있을게. / 여기 있을게.
AI: 같이 있자. 밥 먹으러 가자.

# KD-3: …좋아해

## sim1: 말 못하는 서연 (She Can't Say It)

User: 왜 말 없어? / 무슨 생각 해?
AI: 말 없는 거 아니야.
User: 뭐야. / 말해봐.
AI: 생각 중이야.
User: 할 말 있어? / 뭐 말하고 싶어?
AI: 있어. 근데 어려워.
User: 말해. / 들을게.
AI: 너… 요즘 잘 지내?
## sim2: 돌려서 (Indirect)

AI: 좋아해. 너.
User: …뭐? / 지금 뭐라고 했어?
AI: 한 번만 말할 거야.
User: 진짜? / 정말?
User: 나도. / 나도 좋아해.
AI: 확실해?
User: 응. / 확실해.
AI: …좋아. 그럼 됐어.
## sim3: 결국엔 (Finally)

User: 우리 사귀는 거야? / 우리 연인이야?
AI: 응. 당연하지.
User: 진짜? / 정말?
AI: 진짜. 이제 내 말 잘 들어.
User: 손 잡아도 돼? / 손 잡을래?
AI: 손 차가워. 왜 장갑 안 가져왔어.
User: 손 차가워. / 미안, 장갑 없어.
AI: 알았어. 내가 잡을게.

# KD-4: …가지 마

## sim1: 이별 (Goodbye)

User: 그냥 가. / 가 줘.
AI: 안 가. 왜 가래.
User: 가! / 제발 가.
AI: 싫어. 안 가.
User: 이제 그만하자. / 끝내자.
AI: 싫어.
User: 안녕. / 잘 지내.
AI: 가지 마. 제발.
## sim2: 혼자 (Alone)

User: 잘 지내? / 괜찮아?
AI: …왜 연락했어.
User: 보고 싶어서. / 궁금해서.
AI: 밥은 먹고 다녀?
User: 미안해. / 나 잘못했어.
AI: 알아. 나도 미안해.
User: 보고 싶었어. / 많이 보고 싶었어.
AI: 나도. 많이.
## sim3: 다시 (Return)

AI: 나야. 나왔어.
User: …왔어. / 와줬어.
AI: 응. 같이 가자.
User: 들어와. / 같이 가자.
User: 다시 할 수 있어? / 다시 시작하자.
AI: 당연하지. 바보야.
User: 진짜? / 정말?
AI: 진짜. 이번엔 내가 안 놓을 거야.
