# KP-1: 이름 알아

## sim1: First Encounter

User: 안녕하세요! / 반갑습니다!
AI: 어, 안녕하세요. 밥 먹었어요?
User: 반가워요! / 네, 먹었어요!
AI: 줄 길었죠? 힘들었어요?
User: 사인 해주세요! / 사인 부탁드려요!
AI: 이름이 뭐예요?
User: ___이에요!
AI: 알았어요. 잠깐만요.
## sim2: She Knows You

AI: 또 왔어요? 멀리서 왔죠?
User: 네! / 항상 와요!
AI: 무리하지 마세요. 건강이 먼저예요.
User: 기억해요? / 저 알아요?
AI: 좋아해요?
User: 네, 좋아해요! / 네, 많이요!
AI: 고마워요. 밥 잘 먹어요.
User: 앞으로도 올게요! / 또 올게요!
## sim3: Goodbye (For Now)

User: 이거요. / 이거 드릴게요.
AI: 뭐예요? 선물이에요?
User: 편지예요! / 제가 쓴 거예요!
AI: 고마워요. 잘 읽을게요.
AI: 조심히 가세요. 뛰지 마세요.
User: 네, 감사해요! / 네, 고마워요!
AI: 다음에도 오세요. 밥 먹고 오세요.
User: 꼭이요! / 꼭 올게요!

# KP-2: 서연 콘서트

## sim1: Before the Show

AI: 안녕하세요! 서연 팬이에요?
User: 네! / 네, 좋아해요!
AI: 저도요! 잔소리하는 거 너무 좋아요!
User: 맞아요! / 진짜 귀여워요!
AI: 떨려요?
User: 네, 많이요! / 진짜 떨려요!
AI: 서연이 팬한테 밥 먹었냐고 물어봐요!
User: 빨리 보고 싶어요! / 언제 나와요?
## sim2: During the Show

AI: 서연 나왔어요!
User: 와, 예뻐요! / 대박이에요!
AI: 이쪽 봤어요!
User: 진짜요?! / 설레요!
AI: 앵콜! 앵콜!
User: 다시 나올까요? / 더 보고 싶어요!
AI: 진짜 나왔어요!
User: 최고예요! / 와, 대박!
## sim3: After the Show

User: 진짜 좋았어요! / 최고였어요!
AI: 맞아요! 서연 진짜 예뻤어요!
User: 서연이 봤어요! / 이쪽 봤어요!
AI: 나도 봤어요! 걱정해주는 거 최고예요!
AI: 오늘 재밌었어요!
User: 네, 저도요! / 진짜 재밌었어요!
AI: 다음에 또 봐요!
User: 네, 꼭이요! / 꼭 만나요!

# KP-3: …봤어요

## sim1: She Went Live

AI: 왔어요? 늦었어요.
User: 네! / 왔어요!
AI: 그래요. 밥 먹었어요?
User: 수고했어요! / 잘 봤어요!
AI: 오늘 뭐 했어요?
User: 기다렸어요! / 라이브 기다렸어요!
AI: 기다리지 마세요. 할 거 하세요.
User: 사랑해요! / 응원해요!
## sim2: The Reply

AI: 메시지 봤어요.
User: 진짜요?! / 읽었어요?
AI: 네. 고마워요.
User: 감사해요! / 너무 좋아요!
AI: 뭐가 좋았어요?
User: 다 좋아요! / 다 좋았어요!
AI: 그래요? 고마워요. 무리하지 마세요.
User: 항상 응원할게요! / 계속 좋아할게요!
## sim3: Sending Love

User: 오늘 영상 봤어요! / 영상 잘 봤어요!
AI: 그래요? 늦게까지 보지 마세요.
User: 진짜 좋았어요! / 너무 좋았어요!
AI: 고마워요. 일찍 자요.
AI: 이제 자요. 늦었어요.
User: 잘 자요! / 안녕히 주무세요!
AI: 네. 내일 밥 먹어요.
User: 좋은 꿈 꿔요! / 내일 봐요!
