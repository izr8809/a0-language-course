# TR-1: 이 빵 하나 주세요

## sim1: 주문하기

User: 안녕하세요! 이 빵 하나 주세요. / 안녕하세요! 빵 하나 살게요.
AI: 네, 어떤 빵으로 드릴까요?
User: 크루아상 하나 주세요. / 크루아상으로 할게요.
AI: 알겠어요. 잠깐만 기다려 주세요.
User: 혹시 아메리카노도 있어요? / 아메리카노도 되나요?
AI: 네, 아이스랑 따뜻한 거 다 있어요.
User: 아이스 아메리카노 하나 주세요. / 아이스로 한 잔 주세요.
AI: 네, 같이 준비해 드릴게요.
## sim2: 계산하기

User: 혹시 다 해서 얼마예요? / 전부 얼마예요?
AI: 크루아상이랑 아메리카노 해서 팔천 원이에요.
User: 카드로 결제해도 될까요? / 카드로 해도 되나요?
AI: 네, 카드 되세요.
User: 여기요, 감사합니다. / 여기 있어요. 감사해요.
AI: 감사합니다. 영수증 드릴까요?
User: 아니요, 괜찮아요. / 괜찮아요, 안 주셔도 돼요.
AI: 네, 맛있게 드세요!
## sim3: 자리 잡기

User: 혹시 안에 자리 있어요? / 안에서 먹을 수 있어요?
AI: 네, 창가 쪽에 자리 있어요.
User: 거기 앉아도 될까요? / 저기 앉을 수 있어요?
AI: 네, 편하게 앉으세요.
User: 크루아상이 정말 맛있네요! / 여기 빵 진짜 맛있어요!
AI: 감사해요. 자주 오세요!
User: 네, 또 올게요. / 네, 다음에 또 올게요.
AI: 기다릴게요. 즐거운 시간 되세요!

# TR-2: 공원 가는 길 좀 알려주세요

## sim1: 길 묻기

User: 저기요, 공원 가는 길 좀 알려주세요. / 실례합니다, 공원이 어디에 있어요?
AI: 저쪽으로 쭉 가시면 나와요.
User: 걸어서 얼마나 걸려요? / 여기서 멀어요?
AI: 한 오 분 정도면 돼요.
User: 이 길로 쭉 가면 되는 거죠? / 이쪽으로 계속 가면 돼요?
AI: 네, 맞아요. 큰 나무 보이면 다 온 거예요.
User: 아, 알겠어요. 감사합니다! / 네, 알려주셔서 고마워요!
AI: 네, 즐거운 산책 하세요.
## sim2: 공원에서

User: 저기요, 이 자리 비어 있어요? / 실례합니다, 여기 앉아도 돼요?
AI: 네, 앉으셔도 돼요.
User: 감사합니다. 여기 자주 오세요? / 고마워요. 여기 많이 오세요?
AI: 네, 날씨 좋을 때 자주 와요.
User: 혹시 산책로가 어디에 있어요? / 산책할 수 있는 길이 어디예요?
AI: 저쪽 나무 옆으로 가시면 있어요.
User: 한 바퀴 도는 데 얼마나 걸려요? / 다 돌면 시간이 얼마나 걸려요?
AI: 천천히 걸으면 한 이십 분 정도 돼요.
## sim3: 공원 나가기

User: 저기요, 출구가 어느 쪽이에요? / 실례합니다, 나가는 길이 어디예요?
AI: 저쪽 방향으로 나가시면 돼요.
User: 감사합니다. 근처에 시장도 있어요? / 고마워요. 이 근처에 시장이 있나요?
AI: 네, 나가시면 바로 보여요.
User: 오늘 날씨가 정말 좋네요. / 날씨가 너무 좋아요, 오늘.
AI: 그러게요. 산책하기 딱 좋은 날이에요.
User: 네, 감사합니다. 좋은 하루 되세요! / 맞아요. 감사해요, 좋은 하루 보내세요!
AI: 네, 즐거운 시간 되세요.

# TR-3: 이거 좀 깎아주실 수 있어요?

## sim1: 구경하기

User: 저기요, 이거 얼마예요? / 실례합니다, 이거 가격이 어떻게 돼요?
AI: 그거 오천 원이에요.
User: 생각보다 싸네요. 뭐로 만든 거예요? / 와, 싸다. 이거 뭐로 만들었어요?
AI: 도자기로 만든 빈티지 컵이에요.
User: 이거 오래된 거예요? / 이거 옛날 거예요?
AI: 네, 한 삼십 년 됐어요.
User: 와, 진짜요? 예쁘다. / 정말요? 너무 예뻐요.
AI: 감사해요. 하나밖에 없는 거예요.
## sim2: 흥정하기

User: 혹시 좀 깎아주실 수 있어요? / 조금 싸게 해주실 수 있어요?
AI: 얼마 생각하세요?
User: 사천 원에 해주실 수 있어요? / 사천 원이면 될까요?
AI: 네, 그렇게 해 드릴게요.
User: 카드로 결제해도 될까요? / 카드로 낼 수 있어요?
AI: 죄송해요, 여기는 현금만 돼요.
User: 아, 알겠어요. 현금 있어요. / 네, 그럼 현금으로 할게요.
AI: 감사합니다. 잘 쓰세요!
## sim3: 마무리

User: 혹시 봉투 하나 주실 수 있어요? / 봉투 있으면 하나 주세요.
AI: 네, 잠깐만요. 여기 있어요.
User: 감사합니다. 깨지지 않게 싸주실 수 있어요? / 고마워요. 조심히 포장해 주실 수 있어요?
AI: 네, 제가 잘 싸 드릴게요.
User: 오늘 장사 잘 되셨어요? / 오늘 손님 많았어요?
AI: 네, 덕분에 잘 됐어요. 감사해요.
User: 다음 주에 또 올게요! / 다음에 또 놀러올게요!
AI: 네, 기다릴게요. 또 오세요!

# TR-4: 연남동까지 가주세요

## sim1: 버스 타기

User: 저기요, 이 버스 홍대 가요? / 실례합니다, 이 버스 홍대 지나가요?
AI: 네, 홍대 지나가요.
User: 몇 정류장이나 가야 해요? / 홍대까지 몇 정거장이에요?
AI: 두 정류장만 가시면 돼요.
User: 요금이 얼마예요? / 버스비가 얼마예요?
AI: 천이백 원이에요. 카드 되세요.
User: 네, 카드로 할게요. 감사합니다. / 카드로 낼게요. 고마워요.
AI: 네, 안쪽으로 들어가세요.
## sim2: 버스 안

User: 저기요, 홍대 정류장이 다음이에요? / 실례합니다, 다음이 홍대예요?
AI: 네, 다음 정류장이에요.
User: 내릴 때 버튼 눌러야 해요? / 벨을 눌러야 돼요?
AI: 네, 저기 버튼 누르시면 돼요.
User: 여기서 내리면 되는 거죠? / 여기가 맞죠?
AI: 네, 여기서 내리세요.
User: 감사합니다. 안녕히 가세요! / 고마워요. 좋은 하루 되세요!
AI: 네, 조심히 내리세요.
## sim3: 택시 귀가

User: 안녕하세요! 연남동으로 가주세요. / 안녕하세요! 연남동까지 부탁해요.

User: 연남동 공원 근처로 가주세요. / 연남동 공원 앞에서 내려주세요.
AI: 네, 알겠어요. 출발할게요.
User: 저기 골목 앞에서 세워주실 수 있어요? / 저 앞 골목에서 내려주세요.
AI: 네, 잠깐만요. 여기서 세워 드릴게요.
User: 감사합니다. 얼마예요? / 고마워요. 요금이 얼마예요?
AI: 팔천 원이에요. 카드 되세요.
