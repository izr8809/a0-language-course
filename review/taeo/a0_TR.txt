# TR-1: 이거 주세요

## sim1: 주문

User: 이거 주세요. / 이 빵 주세요.
AI: 어떤 거요?
User: 이 빵이요. / 이거요.
AI: 네, 잠깐만요. 따뜻하게 해드릴까요?
User: 네, 커피 있어요? 
AI: 네, 있어요. 드릴까요?
User: 아메리카노 주세요. / 라테 주세요.
AI: 알겠습니다. 준비할게요.
## sim2: 계산

User: 얼마예요? / 전부 얼마예요?
AI: 육천 원이에요.
User: 카드 돼요? / 카드로 할게요.
AI: 네, 카드 돼요.
User: 여기요. / 감사합니다.
AI: 감사합니다. 봉투 드릴까요?
User: 봉투 있어요? / 봉투 주세요.
AI: 여기 있어요. 드릴게요.
## sim3: 자리

User: 자리 있어요? / 창가 자리 있어요?
AI: 안쪽에 있어요. 
User: 감사합니다. / 여기 앉을게요.
AI: 편하게 앉으세요.
User: 맛있어요. / 여기 빵 좋네요.
AI: 감사합니다. 다 직접 만들어요.
User: 감사합니다! / 또 올게요.
AI: 또 오세요. 감사합니다.

# TR-2: 공원 어디예요?

## sim1: 길 묻기

User: 공원 어디예요? / 공원 어디있나요?.
AI: 저쪽이에요.
User: 멀어요? / 걸어서 돼요?
AI: 아니요, 가까워요.
User: 이 길이요? / 이쪽이요?
AI: 네, 맞아요.
User: 감사합니다! / 알겠습니다.
AI: 조금만 더 가세요.
## sim2: 공원에서

User: 자리 있어요? / 여기 앉아도 돼요?
AI: 네, 앉으세요. 그늘도 있어요.
User: 감사합니다. / 여기 좋네요.
AI: 여기 좋은 자리예요.
User: 산책로 있어요? / 걷기 좋은 데 있어요?
AI: 네, 저쪽이에요.
User: 감사합니다! / 가볼게요.
AI: 천천히 걸어 가세요.
## sim3: 공원 나가기

User: 출구 어디예요? / 나가는 길 어디에요?
AI: 저기 보이세요? 저쪽으로 가세요.
User: 감사합니다. / 알겠습니다.
AI: 조심히 가세요.
User: 동네에 시장 있어요? / 근처에 시장 있어요?
AI: 네, 근처에 있어요.
User: 감사합니다! / 가볼게요.
AI: 즐거운 시간 되세요.

# TR-3: 이거 얼마예요?

## sim1: 구경

User: 이거 얼마예요? / 이거 볼 수 있어요?
AI: 오천 원이에요. 한번 보세요.
User: 비싸요? / 좀 깎아주세요.
AI: 괜찮은 가격이에요.
User: 이거 뭐예요? / 이거 보여주세요.
AI: 빈티지 컵이에요.
User: 예뻐요. / 마음에 들어요.
AI: 감사합니다. 골라 보세요.
## sim2: 흥정

User: 깎아주실 수 있나요? / 좀 싸게 해주세요.
AI: 조금만 깎아줄게요. 얼마가 좋아요?
User: 사천 원이요 / 사천 원 돼요?
AI: 네, 좋아요.
User: 카드 돼요? / 카드로 할게요.
AI: 현금만 돼요.
User: 현금 있어요. / 현금으로 할게요.
AI: 감사합니다. 잘 쓰세요.
## sim3: 마무리

User: 봉투 있어요? / 봉투 주세요.
AI: 네, 여기 드릴게요. 넣어드릴까요?
User: 감사합니다. / 잘 담아주세요.
AI: 잘 담았어요. 조심하세요.
User: 감사합니다. / 잘 쓸게요.
AI: 좋은 거 고르셨어요. 잘 쓰세요.
User: 또 올게요! / 많이 파세요!
AI: 기다릴게요. 또 오세요.

# TR-4: 여기서 세워주세요

## sim1: 버스 타기

User: 서울역 가요? / 홍대 가요?
AI: 어디 가세요?
User: 서울역이요. / 홍대입구요.
AI: 네, 이 버스 맞아요. 타세요.
User: 얼마예요? / 카드 돼요?
AI: 천이백 원이에요.
User: 카드 돼요? / 카드로 할게요.
AI: 네, 카드 돼요. 찍으세요.
## sim2: 버스 안

User: 홍대 멀어요? / 얼마나 걸려요?
AI: 두 정류장이에요. 금방이에요.
User: 감사합니다. / 알겠습니다.
AI: 곧 도착해요.
User: 여기서 내려요? / 다음이에요?
AI: 여기에요.
User: 감사합니다! / 알겠습니다.
AI: 조심히 내리세요.
## sim3: 택시 귀가

User: 연남동이요. / 연남동 가주세요.
AI: 네, 알겠습니다.
User: 빨리 가주세요. / 출발해주세요.
AI: 네, 바로 출발할게요.
User: 여기서 세워주세요. / 여기서 내릴게요.
AI: 네, 잠깐만요.
User: 감사합니다! / 수고하세요.
AI: 조심히 들어가세요.
