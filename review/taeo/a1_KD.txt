# KD-1: 비가 오면 솔직해져

## sim1: 첫 만남 (First Meeting)

AI: 비 맞고 있어. 우산 없어?
User: 응, 없어. / 어, 깜빡했어.
AI: 같이 가. 어디 가?
User: 저쪽 카페. / 연남동 쪽으로.
AI: 비 좋아. 소리가 좋잖아.
User: 그래? / 맞아, 좋다.
AI: 나 태오야. 이름이 뭐야?
User: 나는 ___.
## sim2: 걷는 중 (Walking Together)

AI: 천천히 가자.
User: 왜? / 알겠어.
AI: 비 소리 들어봐.
User: 그러네, 조용하다. / 맞아, 좋다.
AI: 잘 됐다.
User: 뭐가? / 잘 됐다고?
AI: 같이 걷게 돼서.
User: 그러네. / 나도 잘 됐다.

## sim3: 비 그친 후 (After the Rain)
User: 비 그쳤어. / 비 멈췄다.
AI: 아직 오는데?
User: 거의 그쳤는데. / 근데 거의 안 와.
AI: 다행이네.
User: 고마워. 덕분에 안 맞았어. / 고마워. 우산 고마워.
AI: 다음에 또 봐.
User: 응, 또 보자. / 잘 가.
AI: 비 오면 또 봐.

# KD-2: 색이 달라 보여

## sim1: 눈치 (Noticing)

AI: 오늘 좀 달라 보여.
User: 달라? 어떻게? / 진짜?
AI: 표정이. 평소랑 달라.
User: 그렇게 보여? / 진짜 달라 보여?
AI: 괜찮아?
User: 어떻게 알아? / 말 안 했는데.
AI: 그냥 알아.
User: 신기하다. / 고마워.
## sim2: 옆에 (Beside You)

AI: 옆에 있을게. 말 안 해도 돼.
User: 왜? 불편하지 않아? / 괜찮아? 힘들지 않아?
AI: 아니. 그냥 있고 싶어.
User: 고마워. 진짜로. / 정말 고마워. 많이.
AI: 나가자. 바람 좋아.
User: 좋아, 나가자. / 어디?
AI: 기분 나아질 거야.
User: 응, 가자. / 고마워.
## sim3: 말 없이도 (Without Words)

User: 말 안 해도 괜찮아? / 조용히 있어도 돼?
AI: 응. 말 없어도 알아.
User: 그래도 돼? 진짜로? / 정말? 괜찮아?
AI: 응. 그냥 있어.
User: 알아줘서 고마워. 진짜로. / 알아줘서 고마워.
AI: 나는 이해해.
User: 그래? 이해해? / 진짜?
AI: 그럼. 그게 나야.

# KD-3: 좋아해

## sim1: 말 못하는 태오 (He Can't Say It)

User: 왜 말 없어? 오늘 이상해. / 오늘 왜 그래?
AI: 생각 중이야.
User: 무슨 생각? / 뭔 생각?
AI: 너 생각.
User: 할 말 있어? / 뭔가 있지?
AI: 있어. 근데 아직이야.
User: 언제 말해줄 거야? / 그럼 언제?
AI: 곧 말할게.
## sim2: 태오식 고백 (His Way)

AI: 좋아해. 너를.
User: 진짜? / 나를?
AI: 응, 진짜야.
User: 언제부터? / 왜?
AI: 처음부터.
User: 진짜야? / 장난 아니지?
AI: 응. 진짜야.
User: 나도 좋아해. / 나도 그래.
## sim3: 결국엔 (Finally)

User: 나도 좋아해. / 나도야.
AI: 알아. 느꼈어.
User: 알았어? / 어떻게 알았어?
AI: 그냥 알아.
User: 우리 사귀는 거야? / 우리 뭐야?
AI: 응. 같이 있자.
User: 진짜? / 좋아.
AI: 우리 잘 맞잖아.

# KD-4: 같이 있으면 돼

## sim1: 일상

AI: 오늘 뭐 했어?
User: 집에 있었어. / 바빴어.
AI: 보고 싶었어.
User: 나도. / 진짜?
AI: 뭐 먹고 싶어?
User: 아무거나. / 네가 골라.
AI: 같이 먹으러 가자.
User: 좋아. / 어디 가?
## sim2: 좋아

AI: 너랑 있으면 좋아.
User: 왜? / 나도 좋아.
AI: 그냥. 편해.
User: 나도 편해. / 나도 그래.
AI: 자주 보자.
User: 응. / 얼마나?
AI: 매일.
User: 매일? / 좋아.
## sim3: 우리

User: 우리 잘 맞는 것 같아. / 너랑 있으면 좋아.
AI: 나도 그래.
User: 진짜? / 처음부터?
AI: 응. 처음부터.
User: 나도. / 나도 그랬어.
AI: 앞으로도 같이 있자.
User: 응, 같이 있자. / 좋아.
AI: 잘 됐다.
