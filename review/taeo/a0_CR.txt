# CR-1: 오늘 하늘 색 좋다

## sim1: 하늘
AI: 오늘 하늘 색 좋아요.
User: 어? 맞네요. / 진짜 좋아요.
AI: 커피 냄새 좋아요.
User: 진짜 좋아요! / 향기롭네요!
AI: 저 태오예요. 반가워요.
User: 저는 ___예요.
AI: 이름 예뻐요. 잘 부탁해요.
User: 잘 부탁해요. / 저도 반가워요.

## sim2: 자리
AI: 이 자리 좋아요.
User: 왜요? / 왜 좋아요?
AI: 빛이 따뜻해요.
User: 그러네요, 맞아요. / 진짜 따뜻해요.
AI: 뭐 만들어요?
User: 카드 만들어요. / 디자인 작업 중이에요.
AI: 멋져요. 보고 싶어요.
User: 열심히 하고 있어요. / 감사해요, 보여줄게요.
## sim3: 하루 끝
AI: 오늘 잘 됐어요?
User: 그럭저럭이에요. / 좀 힘들었어요.
AI: 내일 더 잘 될 거예요.
User: 어떻게 알아요? / 진짜요?
AI: 그냥 느낌이에요.
User: 뭔 소리예요. / 웃겨요.
AI: 내일 또 봐요.
User: 네, 내일 봐요. / 그래요, 내일!

# CR-2: 소리로 생각해봐

## sim1: 막혔을 때

AI: 일 막혔어요?
User: 어, 조금요. / 네, 막혔어요.
AI: 눈 감아봐요.
User: 왜요? / 뭐 하게요?
AI: 잠깐요. 무슨 소리 들려요?
User: 소리요? / 무슨 소리요?
AI: 잘 들어봐요.
User: 바람 소리인가요? / 뭔가 들려요.
## sim2: 함께

AI: 편하게 있어봐요.
User: 어떻게요? / 뭘 해야 해요?
AI: 눈을 감아봐요.
User: 해볼게요. / 알겠어요.
AI: 어때요?
User: 차분해졌어요. / 편해졌어요!
AI: 나쁘지 않죠?
User: 맞아요, 감사해요. / 진짜 감사해요.
## sim3: 계속
AI: 커피 식었어요.
User: 그러네요. / 진짜 식었어요.
AI: 따뜻할 때 마셔야 해요.
User: 맞아요. / 그래야죠.
AI: 잘 될 것 같아요.
User: 진짜요? / 그래요?
AI: 좋은 느낌이에요.
User: 믿을게요. / 알겠어요.

# CR-3: 빵 먹으면 아이디어 나와

## sim1: 야근

AI: 빵 사왔어요.
User: 어, 감사해요. / 배고팠는데요!
AI: 단 거 먹어요. 힘이 나요.
User: 맞아요, 그렇죠. / 맞는 말이에요.
AI: 빵 먹으면 아이디어 나와요.
User: 진짜요? / 설마요.
AI: 진짜예요. 먹어봐요.
User: 먹어볼게요. / 알겠어요, 먹을게요.
## sim2: 새벽

AI: 지금 몇 시예요?
User: 새벽 두 시예요. / 벌써 두 시예요.
AI: 조금만 더 해요.
User: 뭔 소리예요. / 이상해요.
AI: 거의 다 됐어요.
User: 네, 할 수 있어요. / 조금만 더 해요.
AI: 잘 하고 있어요.
User: 감사해요, 태오 씨. / 힘이 나요, 감사해요.
## sim3: 완료

AI: 다 했어요? 끝났어요?
User: 네, 드디어요. / 끝났어요!
AI: 잘했어요. 수고했어요.
User: 같이 해서 괜찮았어요. / 태오 씨 덕분이에요.
AI: 이제 쉬어요.
User: 네, 감사해요. / 감사해요, 태오 씨.
AI: 내일 빵 또 사올게요.
User: 기대할게요. / 좋아요, 약속이에요!
