# KP-1: 오늘 여기까지 와줬어요

## sim1: 처음 만남

AI: 오늘 여기까지 와줬어요. 고마워요.
User: 네! 정말 기다렸어요! / 네! 너무 기대했어요!
AI: 오늘 하늘 색 봤어요? 되게 좋은 날이에요.
User: 맞아요, 날씨 진짜 좋더라고요! / 네! 하늘이 정말 예뻤어요!
User: 사인 꼭 받고 싶었어요. 해주실 수 있어요? / 사인해 주실 수 있어요? 너무 갖고 싶어요!
AI: 물론이죠. 이름이 어떻게 돼요?
User: ___이에요!
AI: 예쁜 이름이에요. 천천히 써줄게요.
## sim2: 잠깐 더

AI: 자주 와요? 어디선가 본 것 같아서요.
User: 네! 항상 오려고 노력해요! / 네! 매번 꼭 오고 싶어요!
AI: 알아요. 기억하고 있었어요.
User: 진짜로요? 기억해줬어요? / 정말이요? 저를 기억하세요?
AI: 오늘 여기 오면서 어땠어요? 힘들지 않았어요?
User: 아니요! 오히려 너무 설레고 좋았어요! / 전혀요! 오는 길이 너무 행복했어요!
AI: 그 마음이 느껴져요. 저도 그래요.
User: 앞으로도 계속 응원할게요! / 저도요! 항상 함께할게요!
## sim3: 작별 인사

User: 이거 드리고 싶었어요. 직접 준비했어요. / 이거 받아주세요. 제가 만든 거예요.
AI: 이게 뭐예요? 직접 만든 거예요?
User: 네, 편지예요. 꼭 읽어줬으면 해요! / 편지예요. 시간 될 때 읽어주세요!
AI: 소중히 읽을게요. 진짜로요.
AI: 비 오는 날에도 또 와줄 수 있어요?
User: 네, 비 와도 꼭 올게요! / 당연하죠! 비 와도 갈게요!
AI: 기다릴게요. 그날도 하늘 색 얘기해요.
User: 꼭이요! 약속해요! / 네! 꼭 다시 만나요!

# KP-2: 무대 위에서 봤어요

## sim1: 공연 전

AI: 안녕하세요. 오늘 여기까지 와줬어요?
User: 네! 정말 기대하고 왔어요! / 네! 너무 설레요!
AI: 오늘 밤 하늘 봤어요? 별이 많이 떠 있어요.
User: 아직 못 봤어요! 공연 끝나고 볼게요! / 아직이요! 나중에 꼭 볼게요!
AI: 많이 떨려요? 오늘 처음 오는 거예요?
User: 네, 많이 떨려요! 처음은 아니에요! / 너무 떨려요! 근데 처음은 아니에요!
AI: 저도 사실 매번 떨려요. 무대 전에는 항상요.
User: 빨리 보고 싶어요! 멋있을 것 같아요! / 기대돼요! 무대 빨리 보고 싶어요!
## sim2: 공연 중

AI: 태오 나왔어요! 저기 보여요?
User: 와, 진짜 멋있어요! 설레요! / 너무 멋있어요! 심장이 뛰어요!
AI: 이쪽 봤어요. 잠깐 눈이 마주쳤어요.
User: 진짜요?! 심장 터질 것 같아요! / 정말요?! 너무 떨려요!
AI: 앵콜! 앵콜! 같이 외쳐요!
User: 다시 나올 것 같아요? 기대돼요! / 또 나올까요? 너무 보고 싶어요!
AI: 진짜 나왔어요! 오늘 밤 다 줄 것 같아요.
User: 최고예요! 너무 좋아요! / 대박이에요! 진짜 최고예요!
## sim3: 공연 후

User: 진짜 너무 좋았어요! 어떠셨어요? / 정말 최고였어요! 재밌었어요?
AI: 맞아요. 오늘은 특히 더 좋았어요.
User: 태오 봤어요! 눈 마주쳤어요! / 태오랑 눈이 마주쳤어요! 진짜요!
AI: 나도 봤어요. 기억하고 있어요.
AI: 오늘 와줘서 고마워요. 진짜로요.
User: 네, 저도 너무 감사해요! / 저야말로 감사해요! 행복했어요!
AI: 다음에 또 봐요. 비 오는 날도 괜찮아요.
User: 네, 꼭 또 올게요! / 당연하죠! 다음에도 올게요!

# KP-3: 라이브 켰어요

## sim1: 라이브 방송

AI: 왔어요? 기다리고 있었어요.

User: 진짜요?! 지금 바로 볼게요! / 정말요?! 나가서 볼게요!
AI: 뭐 하다가 들어왔어요? 바쁜 거 아니었어요?
User: 아니요! 태오 라이브 기다리고 있었어요! / 아니요! 라이브 켜길 기다렸어요!

User: 사랑해요! 항상 응원해요! / 너무 좋아해요! 계속 응원할게요!
