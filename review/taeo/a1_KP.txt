# KP-1: 오늘 여기까지 와줬어요

## sim1: 처음 만남

AI: 오늘 여기까지 와줬어요. 고마워요.
User: 네! 정말 기다렸어요! / 너무 기대했어요!
AI: 오늘 날씨 좋아요. 하늘 봤어요?
User: 네! 하늘 진짜 예뻐요! / 맞아요, 날씨 좋아요!
AI: 사인 해드릴게요! 이름이 뭐예요?
User: ___이에요!
AI: 예쁜 이름이에요.
User: 감사해요! / 너무 좋아요!
## sim2: 잠깐 더
AI: 자주 와요?
User: 네! 항상 와요! / 네, 매번 오고 싶어요!
AI: 알아요. 기억해요.
User: 진짜요? / 저를 기억해요?
AI: 오늘 오면서 어땠어요?
User: 너무 좋았어요! / 설레고 행복했어요!
AI: 저도 좋아요.
User: 앞으로도 응원할게요! / 항상 함께할게요!

## sim3: 작별 인사

User: 이거 드리고 싶었어요. / 이거 받아주세요.
AI: 이게 뭐예요?
User: 편지예요. 꼭 읽어주세요! / 제가 쓴 거예요!
AI: 소중히 읽을게요.
User: 또 올게요! / 다음에 또 만나요!
AI: 기다릴게요.
User: 꼭이요! 약속해요! / 네! 꼭 다시 만나요!
AI: 사랑해요!

# KP-2: 무대 위에서 봤어요

## sim3: 공연 후

# KP-3: 라이브 켰어요

## sim1: 라이브 방송

AI: 왔어요? 기다렸어요.
User: 네! 바로 왔어요! / 알림 보고 바로 들어왔어요!
AI: 오늘 달 예뻐요. 봤어요?
User: 진짜요?! 지금 볼게요! / 정말요?! 나가서 볼게요!
AI: 뭐 하다 왔어요?
User: 라이브 기다렸어요! / 태오 보고 싶었어요!
AI: 고마워요. 기뻐요.
User: 사랑해요! / 항상 응원해요!
## sim2: 메시지

AI: 메시지 읽었어요. 다 읽었어요.
User: 진짜요?! / 정말이요?!
AI: 고마워요. 마음이 따뜻해요.
User: 진짜요? / 정말요?
AI: 오늘 공연 뭐가 제일 좋았어요?
User: 다 좋았어요! / 전부 좋았어요!
AI: 고마워요. 기분 좋아요.
User: 항상 응원할게요! / 계속 응원할게요!
## sim3: 잘 자요

User: 오늘 라이브 너무 좋았어요! / 오늘 방송 재밌었어요!

User: 잘 자요! 좋은 꿈 꿔요! / 잘 자요! 푹 쉬세요!
AI: 좋은 꿈 꿔요. 사랑해요!
