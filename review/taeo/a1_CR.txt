# CR-1: 오늘 하늘 색 진짜 좋다

## sim1: 하늘

AI: 오늘 하늘 진짜 예쁘지 않아요?
User: 어, 그러네요. 파래요. / 맞아요, 예뻐요!
AI: 저 여기 자주 오는 태오예요.
User: 저는 ___예요.
AI: 반가워요! 여기 자주 와요?
User: 네, 자주 와요! / 오늘 처음이에요.
AI: 저도 자주 와요. 하늘 보기 좋아요.
User: 맞아요, 좋아요! / 저도 좋아요.

## sim2: 자리

AI: 이 자리 좋죠? 빛이 좋아요.
User: 맞아요, 여기 좋아요. / 네, 따뜻해요.
AI: 뭐 해요? 일해요?
User: 네, 작업 중이에요. / 공부 중이에요.
AI: 멋져요. 나중에 보여줘요.
User: 네, 보여줄게요! / 좀 더 되면요!
AI: 기대할게요!
User: 감사해요! / 네!
## sim3: 하루 끝

AI: 오늘 잘 됐어요?
User: 그럭저럭이에요. / 좀 힘들었어요.
AI: 괜찮아요. 내일은 더 잘 될 거예요.
User: 그래요? / 감사해요.
AI: 네, 진짜예요!
User: 알겠어요! / 믿을게요.
AI: 내일 봐요!
User: 네, 내일 봐요! / 잘 가요!

# CR-2: 소리로 한번 생각해봐

## sim1: 막혔을 때

AI: 막혔어요? 표정이 그렇게 보여요.
User: 어, 좀 막혔어요. / 어떻게 알았어요?
AI: 표정 보면 알아요. 잠깐 쉬어요.
User: 쉬어도 돼요? / 알겠어요.
AI: 당연하죠. 잠깐 일어나요.
User: 어디 가요? / 좋아요.
AI: 물 마시고 와요.
User: 알겠어요! / 네, 가요.
## sim2: 함께

AI: 좀 나아졌어요?
User: 네, 좀 나아졌어요. / 아직 좀 어려워요.
AI: 천천히 해요. 잘 하고 있어요.
User: 진짜요? / 감사해요.
AI: 네, 진짜예요. 거의 다 됐어요.
User: 신기하게 됐어요! / 조금 됐어요.
AI: 잘했어요! 계속 해봐요.
User: 네, 감사해요! / 해볼게요.
## sim3: 계속

AI: 커피 식었어요. 마셔요.
User: 그러네요, 잊었어요. / 감사해요.
AI: 따뜻한 거 마셔야 해요.
User: 맞아요! / 네.
AI: 잘 될 것 같아요.
User: 진짜요? / 어떻게 알아요?
AI: 느낌이에요. 잘 하고 있어요.
User: 알겠어요, 믿을게요! / 감사해요.

# CR-3: 빵 먹으면 아이디어 나온다니까

## sim1: 야근

AI: 빵 사왔어요. 먹어요.
User: 어, 감사해요! / 어떻게 알고 왔어요?
AI: 야근하는 것 같아서요.
User: 맞아요, 힘들어요. / 어떻게 알았어요?
AI: 빵 먹으면 힘 나요. 먹어봐요.
User: 진짜요? / 알겠어요, 먹어볼게요.
AI: 네, 진짜예요!
User: 맛있어요! / 감사해요.
## sim2: 새벽

AI: 몇 시예요?
User: 새벽 두 시예요. / 진짜 늦었어요.
AI: 조금만 더요. 거의 다 왔어요.
User: 네, 할 수 있어요. / 조금만 더요.
AI: 잘 하고 있어요!
User: 감사해요. / 덕분에 버티고 있어요.
AI: 빵 또 있어요. 먹어요.
User: 감사해요! / 네, 먹을게요.
## sim3: 완료

AI: 다 했어요?
User: 네, 드디어 끝났어요! / 다 했어요!
AI: 잘했어요! 진짜 수고했어요.
User: 감사해요. 같이 있어줘서요. / 수고하셨어요.
AI: 이제 쉬어요.
User: 네, 쉴게요. / 감사해요.
AI: 내일 빵 또 사올게요!
User: 기대할게요! / 내일 봐요.
