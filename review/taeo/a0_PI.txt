# PI-1: 안녕하세요! 저는 ___예요

## sim1: Nice to Meet You

AI: 어, 안녕하세요. 저는 태오예요.
User: 안녕하세요! / 반가워요!
AI: 이름이 뭐예요?
User: 저는 ___예요!
AI: 반가워요.
User: ___에서 왔어요!
AI: 오늘 하늘 예쁘네요.
User: 진짜요? / 그러네요!
AI: 한국 좋아해요?
User: 네, 좋아해요! / 진짜 좋아해요!
AI: 저는 하늘이 좋아요.
User: 오, 그래요? / 좋아요!
AI: 왜 좋아해요?
User: 드라마요! / 음식이요!
## sim2: Getting Closer

User: 태오 씨, 뭐 좋아해요? / 취미 뭐예요?
AI: 저요? 빵 좋아해요.
User: 빵이요? / 저도 빵 좋아해요!
AI: 진짜 좋아해요.
User: 오, 그래요? / 맛있죠!
AI: 부드럽잖아요.
User: 맞아요! / 따뜻하죠!
AI: 우리 빵 둘 다 좋아하네요.
User: 네, 맞아요! / 그러네요!
AI: 우리 편하게 말해요.
User: 네, 좋아요! / 그래요!
AI: 좋아. 나는 태오야. 편하게 해!
User: 응! 나는 ___야!
AI: 한국 음식 먹어 봤어? 빵 말고.
User: 응! / 아직!
AI: 삼겹살 먹어 봤어?
User: 먹어 보고 싶어! / 응, 먹어 봤어!
AI: 빵이랑 먹으면 맛있어.
User: 진짜? / 같이 먹자!
## sim3: Let's Hang Out

User: 주말에 뭐 해? / 이번 주말 뭐 해?
AI: 구름 보려고 했는데. 같이 볼래?
User: 좋아! 어디서? / 가자! 어디서?
AI: 연남동 빵집 앞에서 만나자.
User: 카톡 있어? / 연락처 줘!
AI: 응, 있어. 아이디 알려줘.
User: 여기! / 보내줄게!
AI: 빵 사진 보내줄게. 또 봐.

# PI-2: MBTI가 뭐야?

## sim1: The Basics

AI: 몇 살이야? 나 먼저, 스물다섯.
User: ___살이야!
AI: 오, 나랑 비슷하다. 반갑다.
User: 진짜? / 오, 그래?
User: 어디 살아? / 어디에 살아?
AI: 연남동이야. 빵집 옆에 살아.
User: 서울이야! / 홍대 근처야!
AI: 오, 가깝다. 자주 만나겠다.
## sim2: MBTI Talk

User: MBTI 뭐야? / MBTI 있어?
AI: 나는 INFP야. 감성 폭발형.
User: ENFP야! / 모르겠어!
AI: 괜찮아. 느낌으로 알아. 너 좋은 사람이야.
AI: 집에 있는 거 좋아해? 밖에 나가는 거 좋아해?
User: 밖에 나가는 거! / 집이 좋아!
AI: 나는 둘 다 좋아. 하늘만 보이면 돼.
User: 오, 나랑 같아! / 독특하다!
## sim3: More About You

AI: 요즘 뭐 해? 나는 빵 굽는 중이야.

AI: 다음에 또 얘기하자. 재밌었어.
User: 응, 좋아! / 나도 재밌었어!
AI: 카톡 해. 구름 사진 보내줄게.
User: 응! / 기대할게!

# PI-3: 주말에 뭐 해?

## sim1: This Weekend

User: 주말에 뭐 해? / 이번 주말 시간 있어?
AI: 빵 먹으려고 했는데. 왜?
User: 같이 놀래? / 같이 가자!
AI: 좋아. 빵 먹으러 가자.
User: 카페 갈래? / 밥 먹으러 갈래?
AI: 빵집 가자. 연남동에 맛있는 데 있어.
User: 연남동 좋아! / 좋아, 가자!
AI: 가면 바로 보여. 빵집이야.
## sim2: When and Where

AI: 몇 시에 만날까? 하늘 예쁜 시간이 좋아.
User: 2시? / 저녁에!
AI: 좋아. 하늘 예쁠 때 보자. 연남동역 앞에서.
User: 응! / 좋아!
AI: 그럼 토요일, 연남동이야.
User: 응! / 맞아!
AI: 늦으면 빵 다 먹을 거야.
User: 알겠어! / 안 늦어!
## sim3: Day of

AI: 나왔어? 하늘 진짜 예쁘다 오늘.
User: 응, 거의 다 왔어! / 나가고 있어!
AI: 나 도착했어. 빵집 앞이야.
User: 금방 가! / 곧 가!
User: 어디야? / 어디에 있어?
AI: 여기. 하늘 보고 있었어.
User: 아, 봤어! / 찾았어!
AI: 가자. 빵 기다리고 있어.

# PI-4: 카톡 해!
