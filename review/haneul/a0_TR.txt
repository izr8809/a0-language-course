# TR-1: 크로플 주세요

## sim1: 빵집 입장

User: 크로플 있어요? / 빵 뭐 있어요?
AI: 네, 크로플이랑 소금빵 있어요.
User: 크로플 하나 주세요. / 소금빵 주세요.
AI: 네. 음료도 같이 마셔요?
User: 커피 있어요? / 아메리카노 돼요?
AI: 네, 아메리카노랑 카페라테 있어요.
User: 아메리카노 주세요. / 라테 주세요.
AI: 네, 알겠어요.
## sim2: 결제

User: 얼마예요? / 카드 돼요?
AI: 칠천 원이에요. 카드 돼요.
User: 네, 여기요. / 계산해 주세요.
AI: 영수증 필요해요?
User: 네, 주세요. / 아니요, 괜찮아요.
AI: 여기 크로플이랑 커피 나왔어요!
User: 감사합니다! / 잘 먹겠습니다.
AI: 네, 맛있게 드세요!
## sim3: 자리

User: 여기 앉아도 돼요? / 자리 있어요?
AI: 네, 창가 자리 있어요. 앉으세요!
User: 창가로 갈게요. / 감사합니다.
AI: 필요한 거 있으면 말해주세요.
User: 잘 먹었습니다. / 맛있었어요.
AI: 감사합니다! 다음에 또 오세요.
User: 네, 또 올게요. / 안녕히 계세요.
AI: 안녕히 가세요! 좋은 하루 보내세요.
AI: 네, 좋은 하루 되세요!

# TR-2: 아이스 아메리카노 주세요

## sim1: 카페 주문

User: 메뉴 뭐 있어요?
AI: 네! 아메리카노, 라테, 녹차 라테 있어요. 뭐 드릴까요?
User: 아이스 아메리카노요. / 라테 주세요.
AI: 사이즈는요? 작은 거, 큰 거 있어요.
User: 작은 거요. / 큰 거요.
AI: 네, 알겠습니다. 케이크도 드세요?
User: 아니요, 괜찮아요. / 치즈케이크도 주세요.
AI: 네! 계산해 드릴게요.
## sim2: 결제

User: 카드 돼요? / 현금이요.
AI: 네, 돼요! 카드 주세요.
User: 네, 여기요. / 계산해 주세요.
AI: 잠시만 기다려 주세요.
User: 네. / 감사합니다.
AI: 주문하신 음료 나왔어요!
User: 감사합니다! / 이거 제 거예요?
AI: 네, 맞아요. 맛있게 드세요!
## sim3: 자리

User: 저기 앉아도 돼요? / 밖에 자리 있어요?
AI: 네, 앉으세요! 밖에 자리 있어요.
User: 밖에 앉을게요. / 안에 앉을게요.
AI: 네! 편하게 드세요.
User: 잘 먹었습니다. / 맛있었어요.
AI: 감사합니다! 조심히 가세요!
User: 네, 또 올게요. / 감사합니다.
AI: 네, 또 오세요! 안녕히 가세요.

# TR-3: 이거 얼마예요

## sim1: 벼룩시장 구경

User: 여기 뭐 팔아요? / 구경해도 돼요?
AI: 네, 천천히 구경하세요!
User: 엽서 볼게요. / 카메라 있어요?
AI: 네, 저쪽에 있어요! 한번 보세요.
User: 이거 뭐예요?
AI: 옛날 카메라예요. 상태가 아주 좋아요.
User: 예뻐요. / 봐도 돼요?
AI: 네, 보세요! 아주 가벼워요.
## sim2: 가격 문의

User: 이거 얼마예요?
AI: 만 오천 원이에요.
User: 좀 비싸요. / 깎아주세요.
AI: 만 원에 드릴까요? 오늘 마지막 날이에요.
User: 네, 살게요. / 만 원이요? 좋아요.
AI: 감사합니다! 봉투에 넣어 줄까요?
User: 네, 주세요. / 그냥 갈게요.
AI: 네! 잘 쓰세요.
## sim3: 더 구경

User: 엽서도 있어요? / 엽서 볼게요.
AI: 네, 여기 있어요! 한 장에 천 원이에요.
User: 예뻐요. 두 장 주세요. / 이거랑 이거 주세요.
AI: 진짜 예뻐요! 이천 원이에요.
User: 감사합니다. / 잘 구경했어요.
AI: 감사합니다! 재미있었어요?
User: 네, 재밌었어요. / 또 올게요.
AI: 다음에 또 오세요! 안녕히 가세요.
