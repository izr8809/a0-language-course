# TR-1: 크로플 주세요

## sim1: 빵집 입장

User: 크로플 있어요? / 빵 종류 뭐 있어요?
AI: 네, 있어요! 크로플이랑 소금빵 있어요. 뭐 드릴까요?
User: 크로플 주세요. / 소금빵 주세요.
AI: 네! 따뜻하게 해드릴까요?
User: 음료도 있어요? / 커피 돼요?
AI: 네, 아메리카노랑 라테 있어요. 뭐 드릴까요?
User: 아이스 라테 주세요. / 아메리카노 주세요.
AI: 네, 알겠습니다. 사이즈는요?
## sim2: 결제

User: 카드 돼요? / 카드로 할게요.
AI: 네, 돼요. 여기 대주세요.
User: 감사합니다. / 네, 여기요.
AI: 영수증 필요하세요?
AI: 주문하신 거 나왔어요!
User: 감사합니다! / 여기요?
AI: 네, 여기요. 맛있게 드세요!
User: 감사합니다. / 잘 먹겠습니다.
## sim3: 자리

User: 감사합니다. / 창가로 갈게요.

AI: 네! 성함이 어떻게 되세요?

# TR-3: 이거 얼마예요

## sim1: 벼룩시장 구경

User: 여기 뭐 팔아요? / 구경해도 돼요?
AI: 네, 천천히 구경하세요! 엽서, 그릇, 옛날 카메라 다 있어요.
User: 엽서 볼게요. / 카메라 있어요?
AI: 네, 저쪽에 있어요! 한번 보세요.
User: 이거 뭐예요? / 이 카메라 뭐예요?
AI: 필름 카메라예요. 오래된 건데 상태 좋아요.
User: 예뻐요. / 만져봐도 돼요?
AI: 네, 만져보세요! 가벼워요.
## sim2: 가격 문의

User: 이거 얼마예요? / 카메라 얼마예요?
AI: 만 오천 원이에요. 상태 좋아서 괜찮은 가격이에요.
User: 좀 비싸요. / 깎아주세요.
AI: 만 원에 드릴까요? 오늘 마지막 날이에요.
User: 네, 살게요. / 만 원이요? 좋아요.
AI: 감사합니다! 봉투에 넣어드릴까요?
User: 네, 주세요. / 괜찮아요, 그냥 들고 갈게요.
AI: 네! 잘 쓰세요.
## sim3: 더 구경

User: 엽서도 있어요? / 엽서 볼게요.
AI: 네, 여기 있어요! 한 장에 천 원이에요.
User: 예뻐요. 두 장 주세요. / 이거랑 이거 주세요.
AI: 좋은 거 고르셨네요! 이천 원이에요.
User: 감사합니다. / 잘 구경했어요.
AI: 감사합니다! 재미있으셨어요?
User: 네, 재밌었어요. / 또 올게요.
AI: 다음에 또 오세요! 좋은 하루 되세요.
