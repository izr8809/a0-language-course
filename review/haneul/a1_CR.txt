# CR-1: 작업 의뢰

## sim1: 미팅 준비

AI: 오늘 두 시에 손님 와요.
User: 네, 알고 있어요! / 네, 오늘 와요!
AI: 이 사진 빛이 좋아요. 앞에 놓을까요?
User: 네, 앞에 놓을게요!
AI: 네. 조금 더 따뜻하면 좋겠어요.
User: 맞아요, 손님도 좋아할 거예요. / 네, 제가 제일 좋아하는 사진이에요.
AI: 손님도 이 빛을 좋아하면 좋겠어요.
User: 분명히 좋아할 거예요! / 네, 진짜 예뻐요!
## sim2: 클라이언트 소개

AI: 손님이 어떤 사진을 원해요?
User: 따뜻한 사진을 찍고 싶어 해요.
AI: 따뜻한 사진... 알겠어요. 밖에서 찍을까요?
User: 네, 공원에서 찍고 싶어 해요. / 밖에서 찍으면 좋을 거예요.
AI: 공원 나무 앞이 좋겠어요. 예뻐요.
User: 꽃이 많아서 좋아요. / 공원 나무 앞이 예뻐요.
AI: 네. 거기서 예쁜 빛을 찾을게요.
User: 거기서 찍으면 예쁠 거예요! / 저도 거기가 좋아요!
## sim3: 작업 조건 확인

AI: 시간이 얼마나 있어요?
User: 다음 주 금요일까지 해야 해요.
AI: 시간은 충분해요.
User: 네, 빨리 끝날 거예요. / 다행이에요!
AI: 사진은 두 번만 고친다고 말해 주세요.
User: 네, 알겠어요.
AI: 그럼 이메일 보내 주세요. 사진을 볼게요.
User: 네, 이메일 보낼게요.

# CR-2: 작업 진행

## sim1: 초안 공유

AI: 첫 번째 사진이에요. 볼래요?
User: 네, 지금 보고 있어요.
AI: 저는 이 사진이 제일 마음에 들어요.
User: 이 사진이 예뻐요. 저도 좋아요! / 저도 이 사진이 좋아요!
AI: 사진이 어때요? 너무 어두워요?
User: 아니요, 따뜻하고 예뻐요! / 네, 조금 어둡지만 멋있어요!
AI: 다행이에요. 고치고 싶은 곳이 있어요?
User: 하늘을 밝게 해 주세요! / 고칠 곳 없어요. 완벽해요!
## sim2: 피드백 받기

AI: 손님한테 연락이 왔어요. 뭐라고 해요?
User: 뒤에 나무가 많아서 안 예쁘다고 해요. 
AI: 나무가 많아요. 지울까요?
User: 네, 지워주세요.
AI: 컴퓨터로 고치는 게 좋아요.
User: 네, 부탁해요.
AI: 네. 나무만 지울게요.
User: 네, 예쁘게 지워 주세요!
## sim3: 수정 완료

AI: 다 고쳤어요. 볼래요?
User: 네, 지금 보고 있어요! / 나무를 지워서 깔끔해요!
AI: 조금 더 밝게 했어요. 빛이 예뻐요.
User: 네, 밝아서 더 예뻐요! / 정말 수고했어요!
AI: 이제 마음에 들어요?
User: 네, 아주 마음에 들어요! / 네, 손님도 좋아할 거예요!
AI: 좋아요. 이제 손님한테 보낼게요.
User: 네, 이메일로 보내요!

# CR-3: 납품 완료

## sim1: 최종 파일 정리

AI: 마지막 사진 정리하고 있어요. 거의 끝났어요.
User: 수고했어요! 예쁜 사진이 많아요. / 일이 끝나서 기분이 좋아요!
AI: 네. 큰 사진, 작은 사진 다 보낼게요.
User: 좋아할 거예요!
AI: 모두 스물다섯 장이에요. 많아요?
User: 아니요, 충분해요!
AI: 빛이 다 예뻐서 다 보낼게요.
User: 손님이 정말 좋아할 거예요. / 정말 수고했어요!
## sim2: 파일 보내기

AI: 손님한테 이메일 보냈어요.
User: 네, 확인했어요!
AI: 사진이 잘 열려요?
User: 네, 잘 열려요! / 네, 잘 보여요!
AI: 다행이에요. 사진이 예뻐요.
User: 사진이 깔끔해서 좋아요! / 손님이 빨리 보면 좋겠어요.
AI: 네. 손님한테 연락 오면 말해 주세요.
User: 네, 연락 오면 말할게요!
## sim3: 마무리 및 정산

AI: 영수증도 보냈어요. 끝났어요.
User: 네, 확인했어요. 좋아요! / 네, 방금 봤어요!
AI: 고마워요. 이번 작업 좋았어요.
User: 저도 같이 일해서 재미있었어요! / 저도 즐거웠어요!
AI: 다음에도 같이 해요.
User: 네, 다음에도 같이 하고 싶어요! / 다음에도 불러 주세요!
AI: 네. 오늘 푹 쉬어요.
User: 오늘 수고했어요!
