# PI-1: 요즘 뭐 하세요?

## sim1: Catching Up

AI: 어, 안녕하세요!
User: 하늘 씨! 반가워요! / 네, 안녕하세요!
AI: 잘 지냈어요?
User: 네, 잘 지냈어요! / 바빴어요!
AI: 저는 요즘 사진 많이 찍어요.
User: 어떤 사진이요? / 멋있어요!
AI: 카페 사진이요.
User: 보여 주세요! / 와, 좋아요!
## sim2: Getting Closer

User: 하늘 씨는 주말에 뭐 해요? / 주말에 보통 뭐 해요?
AI: 보통 사진 찍으러 나가는데, 혹시 사진 좋아해요?
User: 네, 좋아해요! / 어디서 찍어요?
AI: 망원동이요.
User: 거기 분위기 좋죠! / 가 보고 싶어요!
AI: 우리 이제 많이 친해졌죠?
User: 네, 맞아요! / 그런 것 같아요!
AI: 말 놓을까요?
User: 네, 좋아요! / 그래요!
AI: 좋아! 나는 하늘이야. 뭐라고 불러줄까?
User: 하늘아! / 하늘이!
## sim3: Making Plans

AI: 서울에서 어디 좋아해?
User: 한강 좋아해! / 카페 좋아해!
AI: 나는 망원 한강 좋아해.
User: 거기 예뻐? / 같이 가고 싶어!
AI: 다음에 같이 사진 찍으러 갈래?
User: 좋아! 언제? / 가고 싶어!
AI: 이번 주말 어때?
User: 좋아! / 주말 좋아!
AI: 카톡 아직 있지?
User: 응, 있어! / 당연하지!
AI: 좋아. 시간 정해서 연락할게.
User: 응! 기다릴게! / 좋아! 연락 줘!

# PI-2: 우리 밥 먹으러 가자

# PI-3: 너 때문에 한국어 배워

## sim1: Getting Personal

AI: 한국어 왜 배워? 궁금했어.
User: 한국 친구 사귀고 싶어서! / 드라마 알아듣고 싶어서!
AI: 그렇구나. 지금도 잘하고 있어.
User: 아직 멀었어. / 진짜? 고마워!
AI: 한국에서 힘든 거 없어?
User: 가끔 외로워. / 말이 잘 안 통할 때.
AI: 힘들면 나한테 말해. 듣고 있을게.
User: 고마워. / 응, 그럴게.
## sim2: Encouragement

AI: 한국어 처음 배울 때 어땠어?
User: 진짜 어려웠어! / 하나도 몰랐어.
AI: 근데 지금 이만큼 왔잖아.
User: 그래? 고마워! / 아직 많이 부족해.
User: 너 때문에 한국어 더 열심히 하고 싶어. / 너랑 더 많이 이야기하고 싶어.
AI: 진짜? 그 말 좋다.
User: 진짜야! / 정말이야!
AI: 나도 너 때문에 영어 배우고 싶어.
## sim3: Real Friends

AI: 우리 알게 된 지 얼마나 됐지?
User: 한 달? / 벌써 두 달!
AI: 벌써 그렇게 됐네.
User: 맞아! / 시간 빠르다!
AI: 앞으로도 자주 보자.
User: 당연하지! / 나도 그러고 싶어!
AI: 다음 주에 뭐 하자.
User: 좋아! 연락해! / 응! 기대된다!
