# KD-3: 오해와 해명

## sim1: 오해 발생

AI: 그 사진 내가 찍어준 건데 왜 지운 거야?
User: 지운 게 아니에요, 다른 앨범에 옮겼어요. / 안 지웠어요, 다른 곳에 저장했어요.
AI: 왜 말을 안 했어, 그냥 말하면 됐잖아.
User: 말할 틈이 없었어요, 정말 미안해요. / 말할 기회가 없었어요, 진짜 미안해요.
User: 오해한 것 같아서 말하고 싶었어요. / 오해가 있는 것 같아서 설명하고 싶었어요.
AI: 오해가 맞을 수도 있잖아, 내가 어떻게 알아.
User: 직접 확인해봐요, 지금 보여줄 수 있어요. / 직접 봐요, 지금 바로 보여줄게요.
AI: 그래 알겠어, 지금 바로 보여줘봐.
## sim2: 해명

AI: 그래서 왜 따로 모아뒀어, 이유가 뭔데?
User: 소중해서요, 지우고 싶지 않았어요. / 중요해서요, 없애고 싶지 않았어요.
AI: 소중하다고 하면 다야, 좀 더 말해봐.
User: 같이 찍은 게 다 거기 있어요, 봐봐요. / 우리 사진 전부 거기 있어요, 확인해봐요.
User: 이제 믿어줄 수 있어요, 제발요. / 이제 믿어줘요, 부탁이에요.
AI: 미리 말했으면 이렇게 안 됐을 텐데.
User: 맞아요, 다음엔 꼭 먼저 말할게요. / 그래요, 다음엔 반드시 먼저 얘기할게요.
AI: 알겠어, 나도 너무 빨리 오해한 것 같아.
## sim3: 화해

AI: 다음엔 그냥 얘기해줘, 혼자 오해하기 싫어.
User: 그렇게 할게요, 진심으로 약속해요. / 네, 그럴게요, 진짜 약속할게요.
AI: 우리 오늘 사진 찍을까, 기념으로.
User: 좋아요, 같이 찍어요. / 그래요, 우리 사진 찍어요.
User: 화 다 풀렸어요, 이제? / 이제 기분 괜찮아요?
AI: 반쯤은, 나머지는 사진 찍으면서 풀게.
User: 그게 무슨 소리예요, 진짜로. / 무슨 말이에요, 정말로.
AI: 이상한 소리인 건 알아, 근데 그게 나야.

# KD-1: 감정 폭발 후 화해

## sim1: 감정 폭발

AI: 더 이상 참을 수가 없어, 왜 매번 이래?
User: 내가 뭘 잘못했는지 말해줘요. / 뭘 잘못한 건지 알려줘요.
AI: 그게 문제야, 본인은 모른다는 게.
User: 그러니까 말해줘요, 알고 싶어요. / 그래서 얘기해줘요, 듣고 싶어요.
User: 화내는 거 이해해요, 그래도 들어줘요. / 화난 거 알아요, 그래도 한번 들어봐요.
AI: 지금은 아무 말도 듣기 싫어.
User: 그래도 한마디만 할게요, 들어줘요. / 한 가지만 말할게요, 잠깐만요.
AI: 그래 말해봐, 딱 한마디만 들을게.
## sim2: 냉각 시간

AI: 그 말이 왜 이렇게 걸리는지 알아?
User: 왜 그렇게 생각하는지 말해줘요. / 왜 그런 느낌이 드는지 얘기해줘요.
AI: 항상 뒤에 있는 것 같아서, 네 우선순위에서.
User: 그런 거 아니에요, 진짜로요. / 절대 그런 게 아니에요, 정말로요.
User: 제일 중요한 사람이에요, 그거 알죠? / 가장 소중한 사람이에요, 알고 있죠?
AI: 그 말 듣고 싶었던 것 같기도 해.
User: 미리 말했어야 했는데, 정말 미안해요. / 먼저 얘기했어야 했어요, 진짜 미안해요.
AI: 나도 폭발하면 안 됐는데, 미안해.
## sim3: 화해

AI: 우리 좀 이상한 것 같아, 싸우고 또 이러고.
User: 이상한 거 맞아요, 근데 괜찮아요. / 이상하긴 해요, 그래도 좋아요.
AI: 왜 괜찮아, 힘들지 않아?
User: 같이니까 괜찮아요, 그래서요. / 함께라서 괜찮아요, 그거면 돼요.
User: 다음엔 폭발하기 전에 말해줘요. / 다음엔 화나기 전에 얘기해줘요.
AI: 그게 잘 안 되더라고, 근데 노력할게.
User: 나도 더 잘 챙길게요, 약속해요. / 저도 더 신경 쓸게요, 약속할게요.
AI: 그래, 우리 잘 해보자.

# KD-2: 재회와 새 시작

## sim1: 재회

AI: 여기 자주 오는 곳인데 또 마주치네.
User: 우연이에요, 진짜로요. / 그냥 우연히 온 거예요, 정말요.
AI: 우연이든 아니든, 어쨌든 여기 왔잖아.
User: 그러네요, 여기서 또 만났네요. / 맞아요, 또 여기서 보게 됐네요.
User: 잘 지냈어요, 요즘 어떻게 지냈어요? / 잘 지내고 있었어요? 요즘 어때요?
AI: 그냥저냥, 사진은 많이 찍었어.
User: 보여줘요, 진짜 보고 싶어요. / 사진 보고 싶어요, 진짜로요.
AI: 그건 다음에, 지금은 그냥 여기 있자.
## sim2: 서로 확인

AI: 나 보고 싶었어, 솔직하게 말해봐.
User: 보고 싶었어요, 많이요. / 많이 보고 싶었어요, 진짜로요.
AI: 나도, 이상하게 자꾸 생각이 났어.
User: 이상한 거 아니에요, 그냥 마음이에요. / 이상한 게 아니에요, 그게 마음인 거예요.
User: 다시 시작할 수 있을까요, 우리? / 우리 다시 해볼 수 있을까요?
AI: 처음부터 다시? 아니면 이어서 할까?
User: 이어서 가고 싶어요, 그게 좋아요. / 이어서 하고 싶어요, 그게 더 좋아요.
AI: 그래, 그게 더 나을 것 같아.
## sim3: 새 시작

AI: 오늘 사진 찍어도 돼, 이 순간 남기고 싶어.
User: 같이 찍어요, 기념으로요. / 우리 같이 찍어요, 오늘 기념으로.
AI: 기념이라고 하니까 좀 웃기긴 하다.
User: 웃겨도 괜찮아요, 남기고 싶어요. / 웃겨도 상관없어요, 기록하고 싶어요.
User: 이번엔 더 잘할 수 있어요, 진짜로요. / 이번엔 더 잘하고 싶어요, 정말로요.
AI: 나도, 근데 덜 뜬금없이 말하긴 힘들 것 같아.
User: 그러지 마요, 그게 좋은 거예요. / 그렇게 하지 마요, 그게 매력이에요.
AI: 그래? 그럼 그냥 나대로 살게.
