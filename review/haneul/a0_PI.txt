# PI-1: 안녕하세요! 저는 ___예요

## sim1: Nice to Meet You

AI: 안녕하세요. 저는 하늘이에요.
User: 안녕하세요! / 반가워요!
AI: 이름이 뭐예요?
User: 저는 ___예요!
AI: 예쁜 이름이에요.
User: 감사해요! / ___에서 왔어요!
AI: 한국 좋아해요?
User: 네, 좋아해요! / 진짜 좋아해요!
AI: 저도 좋아해요.
User: 진짜요? / 오, 좋아요!
AI: 왜 좋아해요?
User: 드라마요! / 음식이요!
## sim2: Getting Closer

User: 하늘 씨, 뭐 좋아해요? / 취미 있어요?
AI: 사진 찍는 거 좋아해요.
User: 저도 사진이요! / 음악 좋아해요!
AI: 같이 찍을래요?
User: 좋아요! / 네!
AI: 우리 비슷한 거 많네요.
User: 네, 맞아요! / 그러네요!
AI: 말 놓을까요?
User: 네, 좋아요! / 그래요!
AI: 좋아! 나는 하늘이야. 편하게 해!
User: 응! 나는 ___야!
AI: 한국 음식 먹어 봤어?
User: 응! / 아직!
AI: 크로플 좋아해.
User: 먹어 보고 싶어! / 나도 좋아해!
AI: 같이 먹으러 가자!
User: 좋아! / 어디가 맛있어?
## sim3: Let's Hang Out

User: 주말에 뭐 해? / 주말에 시간 있어?
AI: 음... 예쁜 카페 가고 싶어.
User: 좋아! / 나도!
AI: 같이 갈래?
User: 응! 어디 갈까? / 좋아! 어디?
AI: 망원동 카페 가자!
User: 좋아! / 거기 가고 싶어!
User: 카톡 있어? / 카톡 해!
AI: 응, 있어! 아이디 보내 줘!
User: 여기! / 보낼게!
AI: 연락할게. 또 보자!

# PI-2: MBTI가 뭐야?

## sim1: The Basics

AI: 몇 살이야?

AI: 나는 24살이야. 나이는 숫자일 뿐이야. 뭐 하는 사람이야?
User: 진짜? / 오, 그래?
User: 어디 살아? / 집 어디야?
AI: 망원동이야. 동네가 좋아. 너는?
User: 서울이야! / 홍대 근처야!
AI: 오, 가깝다! 자주 만날 수 있겠다. 놀러 와!
## sim2: MBTI Talk

User: MBTI 뭐야? / MBTI 알아?

User: ENFP야! / 모르겠어!
AI: 괜찮아. 나는 INFP야. 조용한 편이야. 너는 어때?
AI: 집에 있는 거 좋아해? 밖에 나가는 거 좋아해?
User: 밖에 나가는 거! / 집이 좋아!
AI: 나는 둘 다 좋아해. 기분에 따라 달라.
User: 오, 나랑 비슷해! / 나는 좀 달라!
## sim3: More About You

User: 응, 좋아! / 그래!
AI: 카톡 해. 사진 보내 줄게!
User: 응! / 기대된다!
