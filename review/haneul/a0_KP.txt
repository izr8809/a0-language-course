# KP-1: 팬미팅

## sim1: 사인회

AI: 오셨어요. 기다렸어요.
User: 네, 왔어요. / 기다렸어요.
AI: 잘 오셨어요. 오늘 좋은 날이에요. 기분 어때요?
User: 감사해요. / 좋은 날이에요.
User: 노래 좋아요. / 무대 좋았어요.
AI: 고마워요. 진심이에요?
User: 오래 기다렸어요. / 꿈같아요.
AI: 알아요. 고마워요. 행복해요. 어떤 노래 좋았어요?
## sim2: 포토 존

AI: 어떻게 찍을까요? 재밌게 찍어요.
User: 같이 찍어요. / 사진 찍어도 돼요?
AI: 좋아요. 여기 분위기 좋아요. 어떤 포즈 할까요?
User: 감사해요. / 좋아요.
User: 잘 나왔어요. / 예뻐요.
AI: 진짜요? 보여주세요.
User: 예뻐요. / 간직할게요.
AI: 다행이에요. 잘 나왔어요. 저도 보고 싶어요.
## sim3: 마무리

AI: 벌써 가세요? 아쉬워요.
User: 네, 가야 돼요. / 아쉬워요.
AI: 잘 가세요. 조심하세요. 다음에 또 봐요.
User: 또 올게요. / 꼭 올게요.
User: 좋았어요. / 행복했어요.
AI: 저도 좋았어요. 재밌었어요. 또 올 거죠?
User: 다음에 봐요. / 기다릴게요.
AI: 꼭 다시 와요. 기다릴게요.

# KP-2: 콘서트 백스테이지

## sim1: 백스테이지 입장

AI: 어떻게 들어오셨어요? 신기해요.
User: 이벤트로요. / 당첨됐어요.
AI: 그렇군요. 운 좋으시네요. 기대하셨어요?
User: 당첨됐어요. / 놀랐어요.
User: 공연 좋았어요. / 감동받았어요.
AI: 고마워요. 진짜 기뻐요. 어디가 좋았어요?
User: 감동받았어요. / 최고였어요.
AI: 그 말 들으니 좋아요. 더 열심히 할게요.
## sim2: 대기실

AI: 뭐 마실래요? 물 있어요.
User: 물 주세요. / 감사해요.
AI: 여기요. 차가워서 좋아요. 목 마르죠?
User: 감사해요. / 고마워요.
User: 긴장했어요? / 떨렸어요?
AI: 조금 떨렸어요. 무대 좋아요.
User: 안 그래 보였어요. / 멋있었어요.
AI: 원래 떨려도 웃어요. 그게 제 비밀이에요.
## sim3: 공연 후

AI: 어떠셨어요? 무대 좋았죠?
User: 너무 좋았어요. / 최고였어요.
AI: 저도 행복했어요. 최고였어요. 다음에도 오실 거예요?
User: 또 올게요. / 기대돼요.
User: 다음 공연 있어요? / 언제 또 해요?
AI: 곧 있어요. 기대돼요.
User: 기대돼요. / 꼭 갈게요.
AI: 꼭 오세요. 더 좋을 거예요.

# KP-3: 라이브 방송

## sim1: 방송 시작

AI: 지금 보세요? 잘 됐어요.
User: 네, 봐요. / 기다렸어요.
AI: 오늘 기분 좋아요. 여러분은요?
User: 기다렸어요. / 저도 좋아요.
User: 안녕하세요. / 하늘 씨!
AI: 안녕하세요. 반가워요.
User: 오늘 예뻐요. / 분위기 좋아요.
AI: 고마워요. 기분 좋아요. 뭐 하고 싶어요?
## sim2: 방송 중

AI: 질문 있어요? 뭐든 물어봐요.
User: 취미가 뭐예요? / 평소에 뭐 해요?
AI: 사진 찍어요. 예쁜 거 찍어요. 여러분은요?
User: 저도요. / 보여줘요.
User: 노래 해줘요. / 노래 듣고 싶어요.
AI: 조금 있다가 할게요.
User: 기다릴게요. / 좋아요.
AI: 금방 해 줄게요. 기다려요.
## sim3: 방송 마무리

AI: 이제 끝나요. 아쉬워요.
User: 벌써요? / 아쉬워요.
AI: 또 할게요. 걱정 마세요. 언제 또 볼까요?
User: 기다릴게요. / 꼭 해줘요.
User: 오늘 재밌었어요. / 행복했어요.
AI: 저도 재밌었어요. 행복했어요. 다음에 또 해요.
User: 또 해요. / 기다릴게요.
AI: 꼭 할게요. 또 봐요.
