# KP-1: 팬미팅

## sim1: 사인회

User: ___이에요. / 팬이에요!
AI: 와, 만나서 반가워요. 밥 먹었어요?
User: 네, 먹었어요. / 아직 안 먹었어요.
AI: 우리 노래 좋아해요?
User: 네, 진짜 좋아해요! / 매일 들어요.
AI: 고마워요! 자, 사인 여기 있어요.
User: 감사합니다!
## sim2: 포토 존

User: 사진 찍어도 돼요?
AI: 네, 좋아요! 이리 와요.
User: 감사합니다! / 떨려요.
AI: 하나, 둘, 셋! 김치!
User: 와, 잘 나왔어요. / 예뻐요.
AI: 진짜요? 저도 보여 주세요.
User: 여기요. / 진짜 잘 나왔어요.
AI: 진짜 잘 나왔어요! 오늘 고마워요.
## sim3: 마무리

AI: 벌써 끝났어요. 아쉬워요.
User: 저도 아쉬워요. / 벌써 가요?
AI: 네, 이제 가야 돼요.
User: 오늘 진짜 행복했어요. / 최고였어요.
AI: 저도 진짜 행복했어요. 우리 또 만나요!
User: 네, 또 와요!
AI: 조심히 가요! 안녕!
User: 안녕히 가세요! / 잘 가요!

# KP-2: 콘서트 백스테이지

## sim1: 백스테이지 입장

AI: 안녕하세요! 이벤트로 왔어요?
User: 네! 진짜 떨려요. / 네, 맞아요!
AI: 축하해요! 오늘 공연 재미있었어요?
User: 네, 최고였어요! / 감동했어요.
AI: 다행이에요. 어디가 제일 좋았어요?
User: 노래가 좋았어요. / 춤이 멋있었어요.
AI: 진짜 고마워요. 저 너무 기분 좋아요.
User: 저도 행복해요.
## sim2: 대기실

AI: 물 마셔요. 여기 있어요.
User: 감사합니다. / 고마워요.
AI: 저 오늘 무대에서 진짜 많이 떨었어요.
User: 진짜요? / 몰랐어요.
AI: 진짜예요. 팬들 보니까 괜찮았어요.
User: 와, 감동이에요. / 최고예요.
AI: 오늘 와줘서 진짜 고마워요.
User: 제가 더 고마워요. / 행복해요.
## sim3: 공연 후

User: 공연 또 해요?
AI: 다음 달에 또 해요. 올 거예요?
User: 네, 가요! / 꼭 가요!
AI: 진짜죠? 저 다 기억해요.
User: 약속해요! / 잊지 마세요!
AI: 네, 약속해요. 조심히 가요.
User: 안녕히 계세요. / 수고하셨어요!
AI: 네! 다음에 또 봐요!

# KP-3: 라이브 방송

## sim1: 방송 시작

AI: 안녕하세요! 제 목소리 잘 들려요?
User: 네, 잘 들려요! / 안녕하세요!
AI: 밥 먹었어요? 저는 방금 먹었어요.
User: 저도 먹었어요. / 아직 안 먹었어요.
AI: 오늘 제 옷 어때요? 예뻐요?
User: 진짜 멋있어요! / 예뻐요!
AI: 고마워요. 오늘 기분 진짜 좋아요.
User: 저도 기분 좋아요! / 다행이에요.
## sim2: 방송 중

User: 취미가 뭐예요? / 평소에 뭐 해요?
AI: 요즘 사진 많이 찍어요.
User: 보여 주세요! / 궁금해요!
AI: 네, 나중에 꼭 사진 올릴게요.
User: 노래해 주세요! / 노래 듣고 싶어요.
AI: 무슨 노래 듣고 싶어요?
User: 데뷔곡이요! / 다 좋아요!
AI: 알겠어요. 지금 조금 노래할게요.
## sim3: 방송 마무리

AI: 앗, 벌써 시간이 이렇게 됐어요.
User: 벌써요? / 가지 마요.
AI: 아쉽지만 저 이제 가야 돼요.
User: 오늘 진짜 재미있었어요. / 행복했어요.
AI: 저도 재미있었어요. 늦었으니까 빨리 자요!
User: 네, 잘 자요! / 좋은 꿈 꿔요!
AI: 안녕! 다음에 또 만나요!
User: 안녕! / 또 만나요!
