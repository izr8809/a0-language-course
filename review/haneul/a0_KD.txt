# KD-1: 우연한 만남

## sim1: 망원동 골목에서

AI: 저기 봐봐. 그림 있어.
User: 뭐요? / 어디요?
AI: 저 벽 그림. 예쁘다. 오래된 게 좋아.
User: 예쁘네요. / 신기하다.
User: 자주 와요? / 처음 봤어요.
AI: 가끔 와. 여기 좋아.
User: 왜요? / 좋아요?
AI: 골목 분위기 좋아. 조용해. 너도 여기 좋아?
## sim2: 공원 벤치에서

AI: 여기 앉아도 돼? 옆자리.
User: 네. / 앉으세요.
AI: 고마워. 바람 좋다. 여기 자주 와?
User: 카메라 있어요? / 사진 찍어요?
User: 이름 뭐예요? / 뭐라고 불러요?
AI: 하늘이야. 이름 예쁘지?
User: 예쁜 이름이다. / 좋은 이름이다.
AI: 예쁘지? 마음에 들어. 너 이름은?
## sim3: 골목 끝에서

AI: 나 여기까지야. 집 가야 해.
User: 아, 그래요? / 벌써요?
AI: 오늘 재밌었어. 또 만나자.
User: 잘 가요. / 조심히 가요.
User: 또 봐요. / 다음에 만나요.
AI: 또 볼 수 있을까?
User: 왜요? / 당연하지.
AI: 궁금해서 물어봤어.

# KD-2: 감정 고백

## sim1: 옥상에서

AI: 춥지 않아? 바람 분다.
User: 괜찮아요. / 조금요.
AI: 별 보여. 오늘 예쁘다. 너도 봐.
User: 뭐 생각해요? / 무슨 생각이에요?
User: 저 좋아요? / 좋아해요?
AI: 좋아해. 처음부터 알았어.
User: 언제부터요? / 진짜요?
AI: 그날 골목에서부터. 기억나지?
## sim2: 한강 벤치에서

AI: 어색해? 나도 좀 떨려.
User: 조금요. / 나도요.
AI: 나도 떨려. 괜찮아. 천천히 하자.
User: 솔직하다. / 좋다.
User: 무서워요? / 괜찮아요?
AI: 무서워. 근데 좋아.
User: 나도요. / 같이 있으니까 괜찮아요.
AI: 같이 무서우면 괜찮아.
## sim3: 빗속에서

AI: 우산 없어? 비 온다.
User: 없어요. / 가방에 없어요.
AI: 나도 없어. 비 좋다. 좀 맞자.
User: 어떡해요. / 비 온다.
User: 가자. / 뛰자.
AI: 어디 갈까? 같이 가자.
User: 아무 데나. / 카페 가자.
AI: 좋아. 비 맞으면서 가자.

# KD-3: 이별/재회

## sim1: 이별 장면

AI: 오늘 날씨 이상해. 흐리다.
User: 왜요? / 무슨 느낌이요?
AI: 느낌이 안 좋아. 너는 어때?
User: 무슨 느낌이요? / 괜찮아요?
User: 힘들어요? / 괜찮아요?
AI: 힘들어. 많이 힘들어.
User: 나도요. / 많이요.
AI: 알아. 나도 다 알아. 그래서 더 힘들어.
## sim2: 혼자 있는 시간

AI: 보고 싶다. 진짜 많이.
User: 저도요. / 나도요.
AI: 그래? 다행이다. 나만 그런 줄 알았어.
User: 진짜로요. / 많이요.
User: 잊었어요? / 잊어요?
AI: 아직 못 잊었어.
User: 나도 아직이에요. / 못 잊었어요.
AI: 다행이다. 잊지 마.
## sim3: 재회 장면

AI: 또 왔어. 여기 좋지?
User: 하늘아. / 하늘이다.
AI: 왔구나. 보고 싶었어. 진짜 많이.
User: 보고 싶었어요. / 나도요.
User: 어떻게 지냈어요? / 잘 지냈어요?
AI: 사진 많이 찍었어.
User: 나는 힘들었어요. / 보고 싶었어요.
AI: 나도 힘들었어. 가지 마.
