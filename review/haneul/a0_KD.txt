# KD-1: 우연한 만남

## sim1: 망원동 골목에서

AI: 저기 봐요. 벽에 그림 있어요.
User: 네? / 어디요?
AI: 저기요. 그림이 참 예뻐요.
User: 네, 예쁘네요. / 신기해요.
AI: 저는 여기 자주 와요. 조용해요.
User: 저는 처음 왔어요. / 분위기 좋아요.
AI: 여기 진짜 좋아요. 같이 봐요.
User: 네, 좋아요. / 고마워요.
## sim2: 공원 벤치에서

AI: 저 여기 앉아도 돼요?
User: 네, 앉으세요. / 네, 괜찮아요.
AI: 고마워요. 바람이 시원해요.
User: 네, 좋아요. / 그러네요.
AI: 저는 하늘이에요. 이름이 뭐예요?
User: 저는 ___이에요.
AI: 예쁜 이름이에요! 반가워요!
User: 고마워요. / 반가워요!
## sim3: 골목 끝에서

AI: 저는 이제 집에 가요.
User: 아, 그래요? / 벌써 가요?
AI: 네. 오늘 만나서 재미있었어요.
User: 저도 재미있었어요.
AI: 우리 다음에 또 만나요!
User: 네, 좋아요. / 다음에 봐요.
AI: 네! 그럼 안녕히 가세요!
User: 안녕히 가세요! / 잘 가요!

# KD-2: 감정 고백

## sim1: 옥상에서

AI: 춥지 않아요? 바람 많이 불어요.
User: 괜찮아요. / 조금 추워요.
AI: 하늘 봐요. 별이 진짜 많아요.
User: 네, 예뻐요. / 진짜 많아요.
AI: 저 할 말 있어요. 많이 좋아해요.
User: 진짜요? / 언제부터요?
AI: 골목에서 처음 봤을 때부터요.
User: 저도 좋아해요.
## sim2: 한강 벤치에서

AI: 저 지금 조금 떨려요.
User: 저도 떨려요.
AI: 우리 이제 만나요. 어때요?
User: 네, 좋아요. / 알겠어요.
AI: 진짜요? 저 너무 기분 좋아요!
User: 저도 기분 좋아요.
AI: 이제 손 잡아도 돼요?
User: 네, 잡아요!
## sim3: 빗속에서

AI: 앗, 비 와요. 우산 있어요?
User: 아니요, 없어요.
AI: 저도 없어요. 우리 그냥 뛰어요!
User: 어디로 가요?
AI: 저기 카페로 가요! 제 손 잡아요!
User: 알겠어요! / 빨리 뛰어요!
AI: 비 맞으니까 시원해요! 재미있어요!
User: 네! 재미있어요! / 감기 조심해요!

# KD-3: 이별/재회

## sim1: 이별 장면

AI: 우리 이제 그만 만나요. 미안해요.
User: 왜요? / 무슨 일이에요?
AI: 요즘 저 너무 힘들어요.
User: 헤어지기 싫어요.
AI: 저도 마음이 아파요.
User: 진짜 끝이에요? / 이러지 마요.
AI: 네. 저 먼저 갈게요. 잘 지내요.
User: 안 돼요... / 잘 가요.
## sim2: 애틋한 전화

AI: 여보세요... 지금 자요?
User: 아니요. / 왜 전화했어요?
AI: 미안해요. 목소리 듣고 싶었어요.
User: 우리 헤어졌잖아요. / 끊을게요.
AI: 알아요. 미안해요.
User: 저도 힘들어요. / 늦었어요.
AI: 내일 우리 처음 만난 골목으로 올래요?
User: 생각해볼게요. / 알겠어요.
## sim3: 재회 장면

AI: 진짜 왔어요. 고마워요.
User: 많이 기다렸어요?
AI: 아니요, 진짜 미안해요.
User: 저도 힘들었어요. / 보고 싶었어요.
AI: 이제 안 떠나요.
User: 진짜예요? / 약속해요.
AI: 네, 약속해요. 진짜 사랑해요.
User: 저도 사랑해요.
