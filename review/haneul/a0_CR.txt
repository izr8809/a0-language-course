# CR-1: 작업실에서

## sim1: 첫 만남

AI: 오늘 날씨 좋다.
User: 네? / 안녕하세요?
AI: 창가 좋다. 따뜻하다. 여기 자주 와?
User: 아, 그러네요. / 따뜻하네요.
User: 여기 앉아도 돼요? / 옆에 앉아도 돼요?
AI: 앉아. 여기 좋아.
User: 고마워요. / 감사해요.
AI: 조용히 해. 집중할 거야. 뭐 하러 왔어?
## sim2: 작업 중

AI: 카메라 있어? 궁금해.
User: 없어요. / 핸드폰이요.
AI: 핸드폰도 괜찮아. 사진 찍어?
User: 핸드폰만요. / 사진은 찍어요.
User: 뭐 해요? / 뭐 그려요?
AI: 그림 그려. 풍경.
User: 뭐 그려요? / 보여 주세요.
AI: 오늘 본 거 그려. 보여줄까?
## sim3: 인사

AI: 이름 뭐야? 알고 싶어.
User: ___.
AI: 하늘이야. 이름 예쁘지?
User: 예쁜 이름이다. / 좋은 이름이다.
User: 내일도 와요? / 자주 와요?
AI: 아마 올 거야.
User: 또 봐요.
AI: 내일도 좋겠다.

# CR-2: 작업 중 대화

## sim1: 사진

AI: 이거 봐. 어때?
User: 예뻐. / 분위기 좋다.
AI: 아침에 찍었어. 분위기 좋았어. 같이 갈래?
User: 어디서 찍었어? / 나도 보고 싶어.
User: 나도 찍어 줘? / 나도 찍고 싶어.
AI: 좋아. 거기 서 봐.
User: 고마워. / 잘 찍어!
AI: 움직이지 마. 잘 나온다. 어떻게 나왔는지 볼래?
## sim2: 작업물

AI: 이거 뭐야? 보여줘.
User: 내 작업이야. / 이거 해봤어.
AI: 좋다. 마음에 들어. 언제부터 했어?
User: 정말? / 고마워.
User: 하늘 거 봐도 돼? / 하늘 작업도 보여 줘.
AI: 봐. 아직 다 안 했어.
User: 멋있다. / 분위기 있다.
AI: 아직 멀었어. 더 그릴 거야. 어디가 좋아?
## sim3: 쉬는 시간

AI: 커피 마셔? 내가 줄게.
User: 어. / 좋아.
AI: 잠깐 기다려. 금방 돼. 뭐 넣을까?
User: 고마워. / 나도 줄까?
User: 잠깐 쉬자. / 쉬고 싶다.
AI: 나도 쉬고 있었어.
User: 나도. / 오늘 많이 했다.
AI: 바람 좋다. 창문 열자.

# CR-3: 작업 마무리

## sim1: 정리

AI: 오늘 끝이야? 벌써 어둡다.
User: 응, 끝냈어. / 거의 다 했어.
AI: 나도 끝났어. 오늘 어땠어?
User: 오늘 많이 했다. / 보람 있다.
User: 짐 챙겨? / 갈 거야?
AI: 챙기고 있어. 천천히.
User: 나도 챙길게. / 나도 정리할게.
AI: 서두르지 마. 천천히 하자. 같이 나갈까?
## sim2: 오늘 감상

AI: 오늘 어땠어? 좋았어?
User: 좋았어. / 재밌었어.
AI: 나도 좋았어. 분위기 좋았어. 뭐가 제일 좋았어?
User: 왜 좋았어? / 뭐가 좋았어?
User: 여기 자주 와? / 매일 와?
AI: 거의 매일 와. 좋아해.
User: 나도 올게. / 나도 자주 올게.
AI: 같이 있으면 좋겠다.
## sim3: 다음 만남

AI: 잘 가. 오늘 재밌었어.
User: 응, 잘 가. / 나도 갈게.
AI: 내일도 꼭 와. 약속이야?
User: 올게. / 내일 봐.
User: 내일 몇 시야? / 몇 시에 올 거야?
AI: 낮에 올 거야. 오후쯤.
User: 알겠어. / 나도 올게.
AI: 또 보자. 기다릴게.
