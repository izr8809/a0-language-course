# PI-1: 안녕하세요! 저는 ___이에요

## sim1: Nice to Meet You

AI: 안녕하세요! 저는 민지예요!
User: 안녕하세요! / 반가워요!
AI: 이름이 뭐예요?
User: 저는 ___이에요!
AI: 반가워요! 어디에서 왔어요?
User: ___에서 왔어요!
AI: 와, 멀리서 왔네요! 한국 좋아해요?
User: 네, 좋아해요! / 진짜 좋아해요!
AI: 한국 왜 좋아해요?
User: 음식이요! / 쇼핑이요!
AI: 오, 저도요! 쇼핑 자주 해요?
User: 네, 자주요! / 가끔요!
## sim2: Getting Closer

User: 민지 씨, 취미 뭐예요? / 뭐 좋아해요?
AI: 저요? 쇼핑이랑 맛집이요!
User: 저도요! / 오, 좋아요!
AI: 진짜요? 우리 진짜 잘 맞아요!
AI: 나이가 어떻게 돼요?
User: ___살이에요!
AI: 오, 비슷해요! 편하게 말해요!
User: 네, 좋아요! / 네!
AI: 좋아! 나는 민지야, 편하게 해!
User: 응! 나는 ___야!
AI: 반가워! 한국 음식 좋아해?
User: 응, 좋아해! / 진짜 좋아해!
## sim3: Let's Hang Out

AI: 주말에 뭐 해? 같이 놀래?
User: 좋아! 뭐 할까? / 어디 갈까?
AI: 홍대 가자! 맛집 알아!
User: 좋아! / 가자!
User: 카톡 있어? / 카톡 해!
AI: 응, 있어! 아이디 알려줘!
User: 여기! / 보낼게!
AI: 저장했어! 연락할게!

# PI-2: 같이 쇼핑 가자!

## sim1: 홍대 쇼핑

AI: 오늘 뭐 살 거야? 나 옷 사고 싶어!
User: 나도 옷! / 나는 구경만!
AI: 여기 예쁜 거 많아! 들어가 보자!
User: 좋아! / 어디?
AI: 이거 어때? 예쁘지?
User: 예쁘다! / 별로야.
AI: 진짜? 나는 이거 좋아! 너는 뭐 좋아해?
User: 이거! / 저거가 더 예뻐!
## sim2: 스타일

AI: 평소에 뭐 입어? 나는 원피스!
User: 나는 청바지! / 나도 원피스!
AI: 오, 그래? 이거 입어 봐!
User: 이거? / 진짜?
AI: 어때? 잘 어울려?
User: 괜찮아? / 좀 큰 것 같아.
AI: 완전 예뻐! 이거 사!
User: 살까? / 다른 거 볼래.
## sim3: 더 알아가기

AI: 요즘 뭐 빠졌어? 나 마라탕!
User: 나는 드라마! / 나도 마라탕!
AI: 오, 진짜? 나한테도 알려줘!
User: 응, 추천해 줄게! / 다음에 같이 보자!
AI: 오늘 진짜 재밌었어! 또 오자!
User: 응, 또 오자! / 나도 재밌었어!
AI: 다음엔 카페도 가자! 내가 맛집 알아!
User: 좋아! 기대돼! / 나도 기대돼!

# PI-3: 오늘 뭐 먹지?

## sim1: 뭐 먹을까?

AI: 배고파! 오늘 뭐 먹을까?
User: 떡볶이? / 마라탕?
AI: 오, 둘 다 좋아! 뭐가 더 좋아?
User: 네가 골라! / 떡볶이!
AI: 맛집 알아? 나 하나 알아!
User: 어디? / 진짜?
AI: 홍대에 맛있는 데 있어! 거기 갈래?
User: 좋아! 가자! / 다른 데는?
## sim2: 약속 정하기

AI: 몇 시에 만날까? 나 6시 좋아!
User: 6시 좋아! / 7시는?
AI: 알겠어! 홍대역 앞에서 볼까?
User: 응! / 어디가 좋아?
AI: 오늘 저녁, 홍대역!
User: 응, 기대돼! / 배고파, 빨리 가자!
AI: 나도! 늦지 마!
User: 안 늦어! / 먼저 가 있어!
## sim3: 만나자!

AI: 나왔어? 나 거의 다 왔어!

User: 금방 가! / 나도 배고파!
User: 민지야! 어디야? / 다 왔어!

User: 아, 봤어! 가자! / 여기야! 가자!
AI: 가자가자! 배고파 죽겠어!

# PI-4: 사진 보내줘!
