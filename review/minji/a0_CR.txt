# CR-1: 안녕하십니까

## sim1: Morning

User: 안녕하십니까! / 안녕하세요!
AI: 안녕하세요! 오늘 날씨 좋죠?
User: 네, 좋습니다! / 정말 좋아요!
AI: 오늘도 화이팅이에요!
AI: 오늘 회의 있어요. 알고 있죠?
User: 네, 알고 있습니다. / 네, 알겠습니다.
AI: 좋아요. 10시에 회의실이에요.
User: 네, 알겠습니다. / 10시요, 네!
## sim2: During the Day

AI: 점심 먹었어요?
User: 네, 먹었습니다. / 아직입니다.
AI: 같이 먹을래요?
User: 네, 좋습니다. / 감사합니다.
AI: 이거 좀 봐 주실 수 있어요?
User: 네, 보겠습니다. / 네, 알겠습니다.
AI: 감사합니다!
User: 아닙니다. / 별말씀을요.
## sim3: Leaving

AI: 오늘 수고 많으셨습니다!
User: 수고하셨습니다! / 고생하셨습니다!
User: 먼저 가보겠습니다. / 먼저 퇴근하겠습니다.
AI: 네, 조심히 가세요!
User: 내일 뵙겠습니다! / 내일 봐요!
AI: 네, 내일 봐요!
User: 안녕히 가세요! / 내일 봐요!
AI: 안녕히 가세요!

# CR-2: 회의 준비해 주세요

## sim1: 준비

AI: 오늘 3시에 회의 있어요. 준비 좀 해 주세요.
User: 네, 알겠습니다. / 뭘 준비할까요?
AI: 이거 다섯 장 복사해 주세요.
User: 네, 바로 하겠습니다. / 네, 지금 할게요.
AI: 다 됐어요?
User: 네, 다 했습니다. / 거의 다 됐습니다.
AI: 잘했어요! 회의실에 물도 놓아 주세요.
User: 네! / 몇 개 놓을까요?
## sim2: 회의

AI: 자료 나눠 주세요.
User: 여기 있습니다. / 네, 잠깐만요.
AI: 감사합니다. 의견 있으세요?
User: 좋은 것 같습니다. / 한 가지 있습니다.
AI: 좋아요! 이것도 정리해 줄 수 있어요?
User: 네, 하겠습니다. / 언제까지요?
AI: 다음 주 월요일까지요.
User: 네, 준비하겠습니다. / 네, 알겠습니다.
## sim3: 칭찬

AI: 오늘 회의 준비 잘해 줬어요!
User: 감사합니다! / 부족하지 않았나요?
AI: 아니에요, 잘했어요! 내일도 부탁해요!
User: 네, 열심히 하겠습니다! / 감사합니다!
User: 오늘 수고 많으셨습니다! / 먼저 가보겠습니다!
AI: 수고했어요! 내일 봐요!
User: 네, 내일 뵙겠습니다! / 안녕히 계세요!
AI: 조심히 가세요!

# CR-3: 죄송합니다, 늦었습니다

## sim1: 지각

User: 죄송합니다, 늦었습니다. / 늦어서 죄송합니다.
AI: 괜찮아요. 무슨 일 있었어요?
User: 지하철이 늦었습니다. / 길이 막혔습니다.
AI: 알겠어요. 다음부터 조심하세요!
User: 다음부터 일찍 나오겠습니다. / 다시는 안 늦겠습니다.
AI: 알겠어요. 오늘 할 일 확인했어요?
User: 네, 확인했습니다. / 지금 확인하겠습니다.
AI: 좋아요, 시작하세요!
## sim2: 실수

User: 저기, 죄송한데 실수가 있었습니다. / 죄송합니다, 실수했습니다.
AI: 어떤 실수요?
User: 이메일을 잘못 보냈습니다. / 파일을 잘못 보냈습니다.
AI: 아, 괜찮아요. 다시 보내 주세요.
User: 네, 바로 수정하겠습니다. / 지금 다시 보내겠습니다.
AI: 네, 빨리 부탁해요!
User: 다 했습니다. 확인해 주세요. / 수정했습니다!
AI: 네, 이번엔 맞아요! 잘했어요!
## sim3: 마무리

AI: 실수는 누구나 해요. 걱정하지 마세요!
User: 감사합니다. / 다음엔 더 조심하겠습니다.
AI: 처음이니까 그럴 수 있어요! 잘하고 있어요!
User: 정말요? 힘이 나요! / 감사합니다!
AI: 오늘 수고했어요! 내일 봐요!
User: 네, 수고하셨습니다! / 감사합니다!
User: 내일 뵙겠습니다! / 안녕히 가세요!
AI: 내일도 화이팅이에요!
