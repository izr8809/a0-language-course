# TR-1: 강남 스파/뷰티샵

## sim1: 스파 도착

User: 여기 예뻐요! 마음에 들어요. / 같이 와서 좋아요.
AI: 다행이에요. 제가 좋아하는 스파예요!
User: 몸이 피곤해요. 마사지 받고 싶어요! / 어떤 마사지가 좋아요?
AI: 아로마 마사지가 좋아요. 추천해요!
User: 예약 안 했어요. 괜찮아요? / 바로 받을 수 있어요?
AI: 네! 당일 예약 돼요.
User: 다행이에요! / 지금 받고 싶어요.
AI: 좋아요. 카운터에 가요!
## sim2: 서비스 안내

User: 마사지 종류가 많아요. 뭐가 좋아요? / 아로마 마사지 하고 싶어요.
AI: 아로마 마사지 좋아요. 짧은 거랑 긴 거 있어요.
User: 어떤 게 좋아요? / 길게 받고 싶어요.
AI: 긴 게 인기 많아요. 그걸로 해요!
User: 네, 푹 쉬고 싶어요.
AI: 좋아요. 피로가 풀릴 거예요!
User: 빨리 받고 싶어요! / 기대돼요.
AI: 지금 들어가요!
## sim3: 스파 후

User: 마사지가 시원해요. 기분이 좋아요!
AI: 다행이에요. 얼굴이 밝아졌어요!
User: 또 오고 싶어요! / 좋은 곳 더 알려 주세요.
AI: 네! 강남에 좋은 곳 많아요.
User: 피부 관리 받고 싶어요! / 피부 관리도 같이 가요.
AI: 좋아요. 찬성이에요!
User: 다음에 제가 낼게요! / 또 같이 가요.
AI: 아니에요. 같이 가면 재미있어요!

# TR-2: 강남 고급 호텔

## sim1: 호텔 도착

User: 호텔이 크고 멋있어요! / 고급스러워서 기분이 좋아요.
AI: 예쁘죠! 제가 좋아하는 호텔이에요.
User: 뭐가 제일 좋아요? / 다 구경하고 싶어요.
AI: 수영장이 제일 예뻐요!
User: 짐이 무거워요. 방에 가고 싶어요. / 체크인 어디서 해요?
AI: 저기 프론트 보여요? 같이 가요!
User: 제가 먼저 말할까요? / 뭐라고 해요?
AI: 제가 예약했어요. 제 이름만 말하면 돼요!
## sim2: 체크인

User: 아침에 조식 먹고 싶어요! / 아침 먹으면 기분이 좋을 거예요.
AI: 조식 맛있어요. 7시부터 먹어요!
User: 룸서비스도 돼요? / 방에서 먹을 수 있어요?
AI: 네! 돼요. 좋죠!
User: 방이 크고 예뻐요! / 창밖 풍경이 예뻐요.
AI: 다행이에요. 오늘 푹 쉬어요!
User: 예약해줘서 고마워요! / 덕분에 푹 쉴 거예요.
AI: 별말씀을요. 즐겁게 놀아요!
## sim3: 호텔 탐색

User: 짐 정리했어요. 수영장 가요! / 수영장 돈 더 내요?
AI: 조금만 더 내요. 수영복 입고 가요!
User: 사진 많이 찍고 싶어요! / 네, 바로 갈게요.
AI: 좋아요. 예쁘게 찍어줄게요!
User: 수영하고 저녁 먹고 싶어요! / 맛있는 식당 가요.
AI: 일 층 레스토랑이 맛있어요. 거기 가요!
User: 기대돼요. 배고파요! / 완벽할 거예요.
AI: 그럼 수영장 가요!

# TR-3: 강남 갤러리/전시회

## sim1: 갤러리 도착

User: 갤러리가 조용하고 멋있어요! / 처음 와서 신기해요.
AI: 강남에서 유명한 갤러리예요. 좋아할 것 같아요!
User: 빨리 보고 싶어요! / 지금 무슨 전시예요?
AI: 오늘 사진 전시를 해요. 멋있어요!
User: 표는 제가 살게요!
AI: 제가 벌써 샀어요. 그냥 들어가요!
User: 고마워요. / 다음에 밥 살게요.
AI: 밥 좋아요. 들어가요!
## sim2: 전시 안내

User: 안에 사진이 많아요? / 기대돼요!
AI: 네! 일 층은 사진이고 이 층은 설치 미술이에요.
User: 사진 찍고 싶어요! / 이 층부터 가요!
AI: 좋아요. 이 층이 인기 많아요!
User: 이 층 먼저 갈게요! / 천천히 구경해요.
AI: 좋아요. 따라갈게요!
User: 기분이 좋아요! / 조용해서 보기 좋아요.
AI: 천천히 봐요!
## sim3: 작품 감상

User: 이 사진 색깔이 예뻐요! / 이 작품 어때요?
AI: 따뜻하죠. 마음이 편해요!
User: 이 사진이 제일 좋아요! / 기분이 좋아요.
AI: 저도 이거 좋아요. 신기해요!
User: 좋은 곳 알려줘서 고마워요! / 같이 구경해서 재미있어요.
AI: 웃어줘서 저도 행복해요!
User: 다음에도 데려가 주세요! / 재미있는 곳 찾으면 같이 가요.
AI: 네! 다음에도 예쁜 곳 가요!
