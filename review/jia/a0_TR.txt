# TR-1: 지아가 일하는 카페

## sim1: 깜짝 만남 (주문하기)

User: 안녕하세요. / 지아 씨!
AI: 앗, 어서 오세요.
User: 여기서 일해요? / 알바해요?
AI: 네, 주말에 일해요. 뭐 마실래요?
User: 아메리카노 주세요. / 라테 주세요.
AI: 네! 큰 거 줄까요? 기본 줄까요?
User: 기본 주세요. / 큰 거요.
AI: 알겠어요. 맛있게 만들게요!
## sim2: 결제

User: 얼마예요? / 카드 돼요?
AI: 사천 원이에요. 카드 주세요.
User: 여기요.
AI: 고마워요. 오늘 바빠요?
User: 아니요, 안 바빠요. / 그냥 쉬어요.
AI: 다행이에요. 조금만 기다리세요.
User: 네, 알겠어요. / 기다릴게요.
AI: 커피 금방 나와요!
## sim3: 커피 픽업

AI: 커피 나왔어요!
User: 감사합니다! / 고마워요.
AI: 맛있게 드세요. 오늘 날씨 참 좋아요.
User: 맞아요. / 언제 끝나요?
AI: 저는 저녁 여섯 시에 끝나요.
User: 이따가 카톡 해요! / 나중에 봐요!
AI: 네, 좋아요! 이따 봐요.
User: 수고하세요. / 안녕히 계세요.
AI: 네! 조심히 가요!

# TR-2: 지아가 일하는 식당

## sim1: 식당에서 만남

User: 지아 씨! / 여기서도 일해요?
AI: 네, 평일에는 식당에서 일해요.
User: 진짜 바쁘네요. / 고생 많아요.
AI: 괜찮아요! 몇 분이세요?
User: 네, 혼자 왔어요. / 한 명이요.
AI: 이쪽으로 오세요. 뭐 먹을래요?
User: 불고기 주세요. / 비빔밥 주세요.
AI: 네! 물 먼저 가져다줄게요.
## sim2: 주문

AI: 불고기 나왔어요! 맛있게 드세요.
User: 맛있겠다! / 고마워요.
AI: 안 매워요? 입에 맞아요?
User: 안 매워요. / 딱 좋아요.
AI: 다행이에요. 더 필요한 거 있어요?
User: 김치 더 주세요. / 물 더 주세요.
AI: 네, 바로 가져다줄게요.
User: 천천히 주세요. / 고마워요.
## sim3: 계산

User: 계산해 주세요! / 얼마예요?
AI: 만 원이에요. 맛있게 먹었어요?
User: 네, 진짜 맛있었어요. / 최고예요!
AI: 고마워요. 카드 주세요.
User: 여기요.
AI: 영수증 버려 줄까요?
User: 네, 버려 주세요. / 아니요, 주세요.
AI: 네, 여기 있어요. 조심히 가세요!

# TR-3: 택시 타기

## sim1: 타기

User: 강남역이요. / 강남역 가주세요.
AI: 네, 타세요. 안전벨트 하세요.
User: 네, 했어요. / 얼마나 걸려요?
AI: 십 분쯤 걸려요. 금방 가요.
User: 감사합니다. / 조금 빨리 가 주세요.
AI: 네, 출발합니다.
User: 차가 많아요? / 길 안 막혀요?
AI: 오늘은 차가 없어요.
## sim2: 가는 중

AI: 다 왔어요. 어디에서 내려요?
User: 저기에서 내릴게요. / 오른쪽이요.
AI: 저기 편의점 앞이요?
User: 네, 맞아요. / 거기 세워 주세요.
AI: 네, 알겠습니다.
User: 감사합니다.
AI: 네. 두고 내리는 물건 없죠?
User: 네, 다 챙겼어요. / 감사합니다.
## sim3: 결제

User: 요금 얼마예요? / 카드 돼요?
AI: 팔천 원이에요. 카드 주세요.
User: 네, 여기요.
AI: 결제 끝났어요. 영수증 필요해요?
User: 아니요, 버려 주세요. / 네, 주세요.
AI: 네, 여기요.
User: 감사합니다. / 수고하세요.
AI: 네, 안녕히 가세요. 조심히 내리세요.
