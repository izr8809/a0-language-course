# CR-1: 첫날이에요

## sim1: 출근

User: 안녕하세요. 오늘부터 인턴입니다. / 안녕하세요. 새로 왔습니다.
AI: 지아예요. 반갑습니다. 따라오세요.
User: 잘 부탁드려요.
AI: 네. 이쪽으로 오세요.
User: 여기 앉아요?
AI: 네, 여기 앉으세요. 이 자리예요.
User: 감사합니다! / 네, 알겠습니다.
AI: 필요한 거 바로 말하세요.
## sim2: 오전

AI: 이거 읽어보세요. 업무 자료예요.
User: 다 읽어요? / 지금 읽어요?
AI: 네. 모르는 거 질문하세요.
User: 감사합니다! / 네, 읽을게요!
AI: 점심 먹었어요?
User: 아직 안 먹었어요.
AI: 같이 가요. 점심시간이에요.
User: 네!
## sim3: 퇴근

User: 이제 집에 가요? / 지금 퇴근해요?
AI: 네, 수고했어요. 먼저 가세요.
User: 오늘 감사했습니다! / 수고하셨습니다!
AI: 내일 9시 출근이에요. 늦지 마세요.
User: 내일 봐요!
AI: 조심히 가세요.
User: 안녕히 계세요!
AI: 네, 푹 쉬세요.

# CR-2: 보고서 써주세요

## sim1: 첫 업무

AI: 이 일 해 주세요. 금요일까지예요.
User: 제가요? / 네, 알겠습니다!
AI: 아주 쉬워요. 한번 해보세요.
User: 네, 해보겠습니다! / 질문 있습니다.
AI: 네, 말하세요.
User: 이 부분 뭐예요?
AI: 아, 이거요? 이렇게 하세요.
User: 알겠습니다. / 감사합니다.
## sim2: 피드백

User: 팀장님, 다 했습니다. / 팀장님, 끝났습니다.
AI: 다 했어요? 잠깐만요.
User: 괜찮아요? / 다 맞아요?
AI: 음, 여기 조금 고치세요.
User: 여기요? / 이 부분이요?
AI: 네. 나머지는 다 괜찮아요.
User: 알겠습니다! / 네, 다시 할게요!
AI: 네, 천천히 하세요.
## sim3: 인정

User: 팀장님, 다 고쳤어요. / 다시 했습니다.
AI: 볼게요. 음, 아주 좋아요.
User: 진짜요? / 다행입니다!
AI: 네, 진짜 잘했어요. 수고했어요.
User: 감사합니다!
AI: 다음에도 똑같이 하세요.
User: 앞으로도 잘 부탁드려요! / 열심히 하겠습니다!
AI: 네. 내일도 힘내세요.
