# KP-1: 팬미팅 (사인회)

## sim1: 드디어 차례

User: 드디어 만났어요! 기뻐요. / 오래 기다렸어요! 행복해요.
AI: 반가워요! 기다렸죠? 미안해요.
User: 아니요. 기분이 좋아요! / 하나도 안 힘들어요.
AI: 저도 오늘 행복해요!
User: 여기에 사인해 주세요! / 이 앨범에 써 주세요.
AI: 네! 하트도 그려 줄게요.
User: 고마워요. 매일 볼 거예요! / 예쁘게 그려 주세요.
AI: 완성했어요. 예쁘죠?
## sim2: 사인 중 대화

User: 새 노래가 좋아요. 매일 들어요! / 새 노래를 매일 듣고 싶어요.
AI: 진짜요? 감동이에요.
User: 노래가 신나요. / 노래가 예뻐요.
AI: 열심히 했어요. 고마워요!
User: 힘들 때 노래를 들어요. 힘이 나요!
AI: 저도 힘이 나요. 파이팅!
User: 계속 응원할게요! / 항상 여기 있을게요.
AI: 네! 멋있는 무대 보여줄게요.
## sim3: 팬미팅 마무리

User: 오늘이 꿈 같아요! / 오늘 행복해요.
AI: 저도 오늘 즐거웠어요!
User: 다음에 또 올게요! / 다음에도 오고 싶어요.
AI: 약속해요. 다음에 또 만나요!
User: 같이 사진 찍어요! / 사진 찍고 싶어요.
AI: 네! 하나, 둘, 셋. 김치!
User: 고마워요. 안 잊을 거예요! / 사진이 예뻐요.
AI: 저도 안 잊을게요. 잘 가요!

# KP-2: 콘서트 백스테이지

## sim1: 백스테이지 입장

User: 여기 와서 신기해요! / 기분이 좋아요.
AI: 와줘서 고마워요. 반가워요!
User: 오늘 공연이 떨려요! / 무대 보고 싶어요.
AI: 저도 두근두근해요!
User: 떨리면 어떻게 해요? / 지금 긴장해요?
AI: 무대에 가면 안 떨려요. 재미있어요!
User: 멋있어요. 다치지 마요. / 무대가 기대돼요.
AI: 응원 덕분이에요. 고마워요!
## sim2: 백스테이지 대화

User: 밥 잘 먹어요? / 잘 지내요?
AI: 네! 맛있는 거 많이 먹어요.
User: 무리하지 마요. / 건강이 중요해요.
AI: 고마워요. 감동이에요!
User: 새 앨범은 언제 나와요? / 새 노래 듣고 싶어요.
AI: 비밀이에요. 곧 나와요!
User: 기대돼요! / 어떤 노래예요?
AI: 아주 신나는 노래예요. 기대해요!
## sim3: 공연 직전

User: 무대에 가면 기분이 어때요? / 소리가 들려요. 떨려요.
AI: 빨리 나가고 싶어요!
User: 제가 크게 응원할게요! / 재미있게 볼게요.
AI: 좋아요. 같이 신나게 놀아요!
User: 오늘 무대도 최고예요! / 빨리 보여 주세요.
AI: 네! 완벽하게 할게요.
User: 조심해요. 파이팅! / 재미있게 볼게요. 파이팅!
AI: 네! 이따 만나요. 안녕!

# KP-3: 라이브 방송

## sim1: 방송 시작

User: 방송 켜서 반가워요! / 기다렸어요. 왔어요.
AI: 반가워요! 밥 먹었어요?
User: 오늘 무슨 이야기 해요? / 재미있는 이야기 해요.
AI: 얼굴 보고 싶어서 켰어요!
User: 많이 바빴어요? / 피곤하지 않아요?
AI: 조금 바빴어요. 지금은 좋아요!
User: 밥 많이 먹어요! / 건강해야 해요.
AI: 네! 매일 잘 먹어요.
## sim2: 방송 중 소통

User: 어떤 계절이 좋아요? / 제일 좋아하는 계절이 뭐예요?
AI: 가을이요! 하늘이 예뻐요.
User: 저도 가을이 좋아요! / 가을 날씨가 좋아요.
AI: 맞아요. 걷기 좋아요!
User: 티비에 언제 나와요? / 빨리 보고 싶어요.
AI: 곧 나와요. 기다려요!
User: 기대돼요! / 매일 응원할게요.
AI: 고마워요. 힘이 나요!
## sim3: 방송 마무리

User: 벌써 끝나요? 아쉬워요! / 더 보고 싶어요.
AI: 저도 아쉬워요!
User: 다음에도 방송해요! / 매일 보고 싶어요.
AI: 네! 다음에 더 길게 할게요.
User: 오늘 재미있었어요! / 즐거웠어요.
AI: 저도 행복했어요. 사랑해요!
User: 푹 쉬어요! / 내일도 파이팅!
AI: 네! 예쁜 꿈 꿔요. 안녕!
