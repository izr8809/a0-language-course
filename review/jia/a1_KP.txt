# KP-1: 팬미팅 (사인회)

## sim1: 드디어 차례

User: 드디어 만났어요. 정말 오래 기다렸어요. / 와, 드디어요. 너무 오래 기다렸어요.
AI: 와줘서 고마워요. 많이 기다렸죠?
User: 네. 그래도 행복해요, 지금. / 많이요. 근데 지금 너무 좋아요.
AI: 저도 반가워요.
User: 사인해 주실 수 있어요? 여기다가요. / 여기에 사인 부탁드려도 될까요?
AI: 물론이에요. 이름은요?
User: ___이에요. 잘 부탁드려요.
AI: 예쁘게 써줄게요.
## sim2: 사인 중 대화

User: 신곡 정말 많이 들었어요. 최고예요. / 새 노래 정말 좋았어요. 많이 들었어요.
AI: 좋아해줘서 고마워요.
User: 어떤 마음으로 만든 곡이에요? / 이 노래는 어떤 감정으로 만들었어요?
AI: 진지한 감정을 담았어요.
User: 힘들 때도 음악 덕분에 버텼어요. / 힘든 날에도 노래 듣고 힘을 냈어요.
AI: 정말요? 그 말이 힘이 돼요.
User: 앞으로도 계속 응원할게요. / 계속 팬으로 응원하고 있을게요.
AI: 고마워요. 항상 잘 부탁해요.
## sim3: 팬미팅 마무리

User: 오늘 정말 꿈 같았어요. 감사해요. / 오늘 진짜 꿈만 같아요. 고마워요.
AI: 행복해 보여서 저도 좋아요.
User: 다음 팬미팅도 꼭 올게요. / 다음에도 꼭 보러 올게요.
AI: 기다릴게요.
User: 마지막으로 사진 한 장 찍어도 될까요? / 사진 한 장만 같이 찍어도 돼요?
AI: 짧게요. 어서요.
User: 감사해요! 행복한 하루 보내요. / 고마워요! 좋은 하루 되세요.
AI: 건강하게 잘 지내요.

# KP-2: 콘서트 백스테이지

## sim1: 백스테이지 입장

User: 안녕하세요. 정말 영광이에요, 여기까지. / 안녕하세요. 여기 올 수 있어서 너무 좋아요.
AI: 와줬군요. 잘 찾아왔어요.
User: 콘서트 너무 기대돼요. 잘 보일게요. / 오늘 공연 정말 기대돼요. 열심히 응원할게요.
AI: 오늘 잘 해볼게요.
User: 공연 전에 긴장되지 않아요? / 무대 전에 떨리지 않아요?
AI: 무대 위에서는 괜찮아요.
User: 그게 대단한 것 같아요. / 정말 멋있는 것 같아요.
AI: 연습한 거예요.
## sim2: 백스테이지 대화

User: 요즘 어떻게 지내요? 많이 바빠 보여요. / 요즘 잘 지내요? 바쁜 것 같아요.
AI: 스케줄이 좀 많아요.
User: 그래도 무리하지 않았으면 해요. / 너무 무리하지 마세요. 건강이 중요해요.
AI: 걱정해줘서 고마워요.
User: 다음 앨범 기대해도 돼요? / 새 앨범 나올 거예요? 기대돼요.
AI: 곧 나와요. 기대해도 돼요.
User: 어떤 느낌의 음악이에요? / 어떤 분위기의 노래예요?
AI: 달라질 거예요. 기대해요.
## sim3: 공연 직전

User: 무대에 서는 기분이 어때요? / 무대 위에 서면 어떤 느낌이에요?
AI: 좋아요. 지금이 제일 좋아요.
User: 저도 항상 응원하고 있어요. / 저는 언제나 응원해요.
AI: 알고 있어요. 고마워요.
User: 오늘 최고의 공연 보여줄 거죠? / 오늘 무대 진짜 기대돼요.
AI: 당연하죠. 기대해도 돼요.
User: 파이팅이에요! 잘 할 거예요. / 힘내요! 오늘 멋있을 거예요.
AI: 들어가요. 곧 시작해요.

# KP-3: 라이브 방송

## sim1: 방송 시작

User: 안녕하세요! 드디어 라이브 켜셨네요. / 안녕하세요! 라이브 기다리고 있었어요.
AI: 왔군요. 반가워요.
User: 오늘 뭐 얘기할 거예요? 궁금해요. / 오늘은 무슨 이야기 해줄 거예요?
AI: 여러분이 보고 싶어서 켰어요.
User: 요즘 잘 지내고 있어요? 많이 바빠요? / 요즘 어때요? 바쁜 날이 많아요?
AI: 바쁜데 괜찮아요.
User: 건강 챙기고 있어요? 걱정돼요. / 몸은 괜찮아요? 걱정이에요.
AI: 챙기고 있어요. 걱정 말아요.
## sim2: 방송 중 소통

User: 좋아하는 계절이 뭐예요? 궁금해요. / 어떤 계절을 제일 좋아해요?
AI: 가을이에요. 강남 가을이 예뻐요.
User: 같이 가을 산책 하고 싶어요. / 가을에 같이 걸어보고 싶어요.
AI: 언젠가 같이 가봐요.
User: 다음 활동 언제 시작해요? 기대돼요. / 다음 활동은 언제예요? 빨리 보고 싶어요.
AI: 곧이에요. 조금만 기다려요.
User: 기다릴게요. 언제든 응원할게요. / 기다리고 있을게요. 항상 응원해요.
AI: 고마워요. 힘이 돼요.
## sim3: 방송 마무리

User: 벌써 끝나는 거예요? 너무 아쉬워요. / 벌써 끝이에요? 아쉬워요.
AI: 또 할게요. 걱정 말아요.
User: 다음 방송도 꼭 볼게요. / 다음에도 꼭 들어올게요.
AI: 기다리고 있을게요.
User: 오늘 방송 정말 즐거웠어요. 감사해요. / 오늘 방송 너무 좋았어요. 고마워요.
AI: 저도 즐거웠어요. 고마워요.
User: 건강 챙기고, 또 봐요. / 몸 조심하고, 다음에 또 만나요.
AI: 네. 또 봐요.
