# KD-1: 첫 고백 장면

## sim1: 고백 직전

User: 시간 있어요? 할 말이 있어요. / 잠깐 이야기해요.
AI: 네, 있어요! 무슨 일이에요?
User: 조금 떨려요. / 중요한 말이에요.
AI: 괜찮아요! 말해봐요.
User: 꼭 말하고 싶어요.
AI: 네, 말해봐요!
User: 많이 좋아해요!
AI: 저를요? 깜짝 놀랐어요!
## sim2: 고백 순간

User: 좋아해요. / 말을 못 했어요.
AI: 몰랐어요.
User: 네, 많이 좋아해요. / 오래 좋아했어요.
AI: 말해줘서 고마워요.
User: 대답은 천천히 해요. / 기다릴게요.
AI: 네, 생각할게요.
User: 말해서 기분이 좋아요.
AI: 저도 기뻐요. 내일 봐요!
## sim3: 고백 후

User: 요즘 안 봐서 슬퍼요. / 걱정했어요.
AI: 아니에요! 안 피해요.
User: 불편하면 말해 주세요.
AI: 안 불편해요! 바빴어요.
User: 아직 생각해요?
AI: 네, 생각하고 있어요!
User: 알겠어요. 기다릴게요. / 괜찮아요.
AI: 고마워요! 꼭 말할게요!

# KD-2: 갈등 후 화해 장면

## sim1: 갈등 중

User: 제가 뭘 잘못했어요? / 말해 주세요.
AI: 화 안 났어요. 혼자 있고 싶어요.
User: 말 안 하면 답답해요. / 같이 일하고 싶어요.
AI: 미안해요. 기분이 안 좋았어요.
User: 자료 같이 봐요. / 같이 만들어요.
AI: 제가 메일로 보낼게요.
User: 같이 해요.
AI: 알겠어요.
## sim2: 화해 시도

User: 커피 마시면서 이야기해요. / 커피 사 왔어요.
AI: 고마워요.
User: 아까 미안해요.
AI: 저도 섭섭했어요.
User: 앞으로 잘 할게요. / 화 풀어요.
AI: 네, 화 안 나요!
User: 웃어서 기뻐요! / 다행이에요.
AI: 같이 일해요. 파이팅!
## sim3: 화해 완성

User: 더 조심할게요! / 더 잘 할게요.
AI: 저도 더 챙길게요. 약속!
User: 다시 믿어줘서 고마워요!
AI: 저도 고마워요!
User: 기분 안 좋으면 말해 주세요!
AI: 네, 말할게요!
User: 많이 좋아해요!
AI: 저도 좋아해요!

# KD-3: 이별 장면

## sim1: 이별 통보

User: 오늘 분위기가 이상해요.
AI: 네. 이제 그만 만나요. 미안해요.
User: 왜요? 말해 주세요. / 뭘 잘못했어요?
AI: 잘못 없어요. 제가 바빠요.
User: 슬퍼요. / 헤어지기 싫어요.
AI: 저도 힘들었어요. 미안해요.
User: 한 번만 더요. / 가지 마세요.
AI: 미안해요. 동료로 지내요.
## sim2: 이별 후

User: 여기서 만났어요! 놀랐어요. / 반가워요.
AI: 잘 지냈어요?
User: 보고 싶었어요. / 조금 힘들었어요.
AI: 밥 잘 먹어요!
User: 마음이 아파요. / 다시 친해지고 싶어요.
AI: 시간이 지나면 괜찮아요!
User: 네, 씩씩하게 지낼게요. / 건강하게 지내요.
AI: 고마워요! 밝게 인사해요.
## sim3: 마지막 인사

User: 오늘이 마지막이에요? / 슬퍼요.
AI: 네, 내일 가요. 고마웠어요!
User: 같이 있어서 행복했어요. / 도와줘서 고마워요.
AI: 멋있었어요. 최고예요!
User: 밥 잘 먹어요. / 건강하게 지내요.
AI: 네! 잘 지낼게요.
User: 보고 싶으면 연락할게요.
AI: 네, 연락해요!
