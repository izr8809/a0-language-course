# KD-3: 첫 고백 장면

## sim1: 고백 직전

User: 잠깐 시간 있어요? 드릴 말씀이 있어요. / 지금 잠깐 괜찮아요? 할 말이 있어요.
AI: 지금 바빠요. 뭔 일이에요?
User: 중요한 말이에요. 꼭 들어줬으면 해요. / 꼭 해야 할 말이에요. 들어줄 수 있어요?
AI: 5분은 있어요. 말해봐요.
User: 어떻게 시작해야 할지 잘 모르겠어요. / 뭐라고 말해야 할지 잘 모르겠어요.
AI: 그냥 솔직하게 말해요.
User: 이게 제 진심인 말이에요. / 진심으로 하는 말이에요.
AI: 듣고 있어요.
## sim2: 고백 순간

User: 좋아해요. 꽤 오래 전부터 그랬어요. / 좋아했어요. 많이, 오래전부터요.
AI: …그게 무슨 말이에요.
User: 진지한 말이에요. 싫으면 솔직히 말해줘요. / 장난 아니에요. 싫으면 말해도 돼요.
AI: 갑작스러워요.
User: 대답 안 해도 괜찮아요. 기다릴게요. / 지금 대답 안 해도 돼요. 천천히 해요.
AI: 시간을 좀 줘요.
User: 알겠어요. 서두르지 않을게요. / 네, 괜찮아요. 급하지 않아요.
AI: …고마워요.
## sim3: 고백 후

User: 요즘 피하는 것 같아서 신경 쓰였어요. / 요즘 저를 피하는 것 같아서 걱정됐어요.
AI: 그런 거 아니에요.
User: 불편하면 솔직히 말해줘도 돼요. / 힘들면 그렇다고 말해줘도 괜찮아요.
AI: 불편한 거 아니에요.
User: 그럼 아직 생각 중인 거예요? / 아직 결정 못 한 거예요?
AI: 네. 아직 정리 중이에요.
User: 기다릴게요. 천천히 해요. / 괜찮아요. 시간 걸려도 돼요.
AI: …알겠어요.

# KD-1: 갈등 후 화해 장면

## sim1: 갈등 중

User: 제가 뭘 잘못한 건지 알고 싶어요. / 제가 뭘 잘못했는지 말해줘요.
AI: 지금 그 얘기 하고 싶지 않아요.
User: 계속 이러면 같이 일하기 어렵지 않아요? / 이렇게 계속 가면 일하기 힘들지 않아요?
AI: 알아요. 그래서 참고 있어요.
User: 오늘 회의 자료 같이 검토해야 해요. / 회의 자료 같이 봐야 해요, 오늘.
AI: 자료는 제가 보낼게요.
User: 같이 확인하는 게 낫지 않아요? / 둘이 같이 보는 게 더 좋지 않아요?
AI: …잠깐만 기다려요.
## sim2: 화해 시도

User: 커피 마시면서 얘기할 수 있을까요? / 커피 한잔 하면서 이야기해요.
AI: 뭘 하려는 건데요.
User: 오해가 있는 것 같아서 풀고 싶어요. / 오해가 생긴 것 같아요. 이야기하고 싶어요.
AI: 오해라고 생각하는 거예요?
User: 그때 제 말이 상처가 됐죠. 미안해요. / 제 말 때문에 기분 나빴죠. 죄송해요.
AI: 솔직히 기분 나빴어요.
User: 진심으로 사과하고 싶어요. / 정말 미안하다고 말하고 싶었어요.
AI: …알겠어요.
## sim3: 화해 완성

User: 앞으로는 조심할게요. 믿어줘요. / 다음부터는 더 신경 쓸게요. 믿어요.
AI: 한 번 더 믿어볼게요.
User: 감사해요. 정말 다행이에요. / 고마워요. 진짜 다행이에요.
AI: 다음엔 먼저 말해줘요.
User: 다시 잘 지낼 수 있는 거죠? / 우리 다시 괜찮아진 거예요?
AI: 그러려고 여기 나온 거잖아요.
User: 맞아요. 고마워요, 진짜로요. / 그렇죠. 정말 고마워요.
AI: 커피 마셔요.

# KD-2: 이별 장면

## sim1: 이별 통보

User: 무슨 말 하려는지 느낌이 와요. / 뭔 말 할 건지 대충 알 것 같아요.
AI: 이제 그만하는 게 맞는 것 같아요.
User: 이유라도 알려줘요. / 왜인지 말해줄 수 있어요?
AI: 이유는 없어요. 그냥 끝이에요.
User: 이렇게 갑자기 끝낼 수는 없어요. / 갑자기 이러면 안 되죠.
AI: 갑작스럽지 않아요. 오래 생각했어요.
User: 저한테 기회를 한 번만 줘요. / 한 번만 더 기회를 주세요.
AI: …미안해요.
## sim2: 이별 후

User: 여기서 보게 될 줄 몰랐어요. / 여기서 만날 줄은 몰랐어요.
AI: 저도요. 잘 지냈어요?
User: 그냥 그랬어요. 솔직히 말하면요. / 솔직히 쉽지 않았어요.
AI: …그랬군요.
User: 아직 좀 어색한 게 어쩔 수 없네요. / 아직 어색한 건 어쩔 수 없죠.
AI: 시간이 필요한 거예요.
User: 맞아요. 천천히 가요. / 그렇죠. 서두르지 말아요.
AI: 잘 지내요.
## sim3: 마지막 인사

User: 오늘이 마지막 날이죠? 진짜로요. / 정말 오늘이 마지막이에요?
AI: 네. 오늘로 끝이에요.
User: 그동안 정말 고마웠어요. / 지금까지 정말 감사했어요.
AI: 저도요.
User: 잘 지내야 해요. 꼭이요. / 꼭 건강하게 잘 지내요.
AI: 그쪽도 잘 지내요.
User: 연락할게요. 안 받아도 돼요. / 가끔 연락할게요. 부담 갖지 말아요.
AI: …받을게요.
