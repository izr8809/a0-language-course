# KD-1: 처음 만났어요

## sim1: 첫 인사 (First Greeting)

User: 안녕하세요. 처음 뵈어요.
AI: 지아예요. 반갑습니다~
User: 잘 부탁드려요. / 반갑습니다.
AI: 커피 좋아해요?
User: 네, 좋아해요. / 진짜 좋아해요.
AI: 저도 좋아해요. 여기 카페 자주 와요?
User: 네, 자주 와요. / 아니요, 가끔 와요.
AI: 좋아요! 우리 같이 커피 마셔요.
## sim2: 어색한 시간 (Awkward Time)

User: 뭐 마셔요? / 뭐 좋아해요?
AI: 저는 아메리카노요. 어떤 거 마셔요?
User: 저도 아메리카노요. / 저는 라테요.
AI: 오, 저랑 똑같아요!
User: 네, 신기해요. / 안 추워요?
AI: 저는 괜찮아요. 춥지 않아요.
User: 다행이에요. / 감기 조심해요.
AI: 고마워요.
## sim3: 묘한 느낌 (Strange Feeling)

User: 오늘 재밌었어요. / 오늘 좋았어요.
AI: 진짜요? 저도 참 좋았어요.
User: 네. / 저도요.
AI: 나중에 꼭 카톡 해요!
User: 또 만나요? / 다음에 또 봐요?
AI: 좋아요! 우리 또 만나요.
User: 네, 연락해요. / 카톡 해요.
AI: 꼭이요!

# KD-2: …걱정했어요

## sim1: 지아가 이상해 (Something's Off)

User: 괜찮아요? / 무슨 일이에요?
AI: 괜찮아요. 왜요?
User: 진짜요? / 정말요?
AI: 진짜요. 걱정 마세요.
User: 말해요. / 얘기해요.
AI: 별일 아니에요.
User: 말해요. / 듣고 싶어요.
AI: 나중에요. 지금은 괜찮아요.
## sim2: 감정 (Feelings)

AI: 왜 연락 안 했어요?
User: 미안해요. / 정말 미안해요.
AI: 저 많이 기다렸어요.
User: 몰랐어요. 미안해요. / 바빴어요.
AI: 저 혼자 많이 걱정했어요.
User: 화났어요? / 기분 안 좋아요?
AI: 화 안 났어요. 그냥 걱정했어요.
User: 알겠어요. / 미안해요.
## sim3: 수습 (Aftermath)

User: 괜찮아요? / 지금 어때요?
AI: 네, 괜찮아요.
User: 진짜 괜찮아요? / 정말요?
AI: 조금 외로웠어요.
User: 같이 있어요. / 안 가요.
AI: 가지 마세요. 좀 더 있어요.
User: 네, 같이 있어요.
AI: 고마워요. 진짜 좋아요.

# KD-3: …좋아해요

## sim1: 말 못하는 지아 (She Can't Say It)

User: 왜 말 없어요? / 왜 조용해요?
AI: 말 없는 거 아니에요.
User: 무슨 일이에요? / 무슨 생각해요?
AI: 그냥 생각해요.
User: 할 말 있어요? / 말해요.
AI: 네, 있어요. 근데 조금 부끄러워요.
User: 괜찮아요. / 편하게 말해요.
AI: 요즘 자꾸 생각나요.
## sim2: 돌려서 (Indirect)

AI: 좋아해요. 진짜 많이요.
User: 네? / 뭐라고요?
AI: 다 들었죠? 다시 안 말해요.
User: 저도 좋아해요.
AI: 진짜요? 장난 아니죠?
User: 네. / 진짜예요.
AI: 다행이에요. 진짜 다행이에요.
User: 저도 다행이에요. / 고마워요.
## sim3: 결국엔 (Finally)

User: 우리 만나요? / 우리 연인이에요?
AI: 당연하죠! 왜 물어봐요?
User: 진짜요? / 정말요?
AI: 진짜요. 이제 도망 못 가요.
User: 손 잡아요? / 손 잡아도 돼요?
AI: 앗, 손이 차가워요!
User: 미안해요. / 밖이 추워요.
AI: 제가 잡을게요. 이제 따뜻하죠?
