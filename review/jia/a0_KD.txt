# KD-1: 처음 만났어요

## sim1: 첫 인사 (First Greeting)

User: 안녕하세요. 처음 뵙겠습니다. / 안녕하세요. 반가워요.
AI: 지아예요. 반갑습니다~
User: 잘 부탁드려요. / 반갑습니다.
AI: 커피 좋아해요?
User: 여기 자주 와요? / 여기 많이 와요?
AI: 가끔요. 여기 좋아해요?
User: 저도 자주 와요. / 네, 좋아해요.
AI: 그럼 또 마주칠 수도 있겠네요~
## sim2: 어색한 시간 (Awkward Time)

User: 뭐 마셔요? / 뭐 드실래요?
AI: 아메리카노요~ 뭐 드실래요?
User: 저도 아메리카노요. / 라테 마실게요.
AI: 오, 취향이 비슷하네요!
AI: 안 추워요?
User: 괜찮아요. / 안 추워요.
AI: 거짓말~ 추운 거 다 보여요.
User: …감사해요. / 고마워요.
## sim3: 묘한 느낌 (Strange Feeling)

User: 오늘 좋았어요. / 오늘 즐거웠어요.
AI: 진짜요? 저도 좋았어요~
User: 네. / 진짜요.
AI: 도착하면 카톡 주세요!
User: 또 만나요? / 다음에 또 봐요?
AI: 좋아요! 기대할게요~
User: 연락할게요. / 카톡 할게요.
AI: 꼭이요! 안 하면 제가 먼저 할 거예요~

# KD-2: …걱정했어요

## sim1: 지아가 이상해 (Something's Off)

User: 괜찮아요? / 무슨 일이에요?
AI: 괜찮아요~ 왜요?
User: 진짜요? / 정말요?
AI: 진짜요. 걱정 마세요~
User: 말해주세요. / 얘기해요.
AI: 별거 아니에요~
User: 말해주세요. / 듣고 싶어요.
AI: 나중에요. 지금은 괜찮아요.
## sim2: 감정 (Feelings)

AI: 연락 왜 안 했어요?
User: …죄송해요. / 미안해요.
AI: 기다렸어요. 많이.
User: 몰랐어요. 죄송해요. / 정말 미안해요.
User: 화났어요? / 기분 안 좋아요?
AI: 화 안 났어요.
User: 그럼 뭔데요. / 그럼 왜요?
AI: 걱정했어요. 다음엔 연락해 주세요.
## sim3: 수습 (Aftermath)

User: 괜찮아요? / 지금 어때요?
AI: 괜찮아요~ 당신은요?
User: 솔직하게요. / 진짜로요.
AI: …조금 외로웠어요.
User: 있을게요. / 안 가요.
AI: 가지 마세요. 좀 더 있어요.
User: 있을게요. / 네, 같이 있어요.
AI: 같이 있어요. 그것만으로 괜찮아요.

# KD-3: …좋아해요

## sim1: 말 못하는 지아 (She Can't Say It)

User: 왜 말 없어요? / 왜 조용해요?
AI: 말 없는 거 아니에요.
User: 뭔데요. / 무슨 생각해요?
AI: 생각 중이에요.
User: 할 말 있어요? / 뭐 말하고 싶어요?
AI: 있어요. 근데 좀 부끄러워요.
User: 말해주세요. / 괜찮아요, 해봐요.
AI: 요즘… 자꾸 생각나요.
## sim2: 돌려서 (Indirect)

AI: 좋아해요. 많이.
User: …네? / 뭐라고요?
AI: 들었잖아요~ 다시 안 말할 거예요.
User: 진짜요? / 정말요?
User: 저도요. / 저도 좋아해요.
AI: 진짜요? 놀리는 거 아니죠?
User: 네. / 진짜예요.
AI: …다행이에요. 진짜 다행이에요.
## sim3: 결국엔 (Finally)

User: 우리 사귀는 거예요? / 우리 연인이에요?
AI: 당연하죠~ 왜 물어봐요?
User: 진짜요? / 정말요?
AI: 진짜요. 이제 도망 못 해요~
User: 손 잡아도 돼요? / 손 잡을래요?
AI: 손 차가워요! 왜 이렇게 차가워요~
User: 손 차가워요. / 미안해요, 추워서요.
AI: 제가 잡아줄게요. 따뜻하죠?
