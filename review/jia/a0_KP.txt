# KP-1: 팬미팅

## sim1: 줄 서기

User: 안녕하세요. / 반가워요!
AI: 어서 오세요! 반가워요.
User: 오래 기다렸어요. / 정말 보고 싶었어요!
AI: 진짜요? 고마워요!
User: 이거 받으세요. / 선물이에요!
AI: 와! 감사해요. 예뻐요!
User: 팬이에요. / 좋아해요!
AI: 고마워요!
## sim2: 사인

User: 사인해 줘요. / 사인 부탁해요!
AI: 어디에 할까요?
User: 여기요. / 이 책이요.
AI: 알겠어요. 잠깐만요!
User: 이름 써 주세요. / 예쁘게 써 줘요!
AI: 네, 예쁘게 쓸게요.
User: 감사해요. / 고마워요!
AI: 여기 있어요. 오늘 고마워요!
## sim3: 포토타임

User: 사진 찍어요? / 같이 찍어요!
AI: 좋아요! 같이 찍어요.
User: 옆에 갈게요.
AI: 네! 이리 오세요.
User: 고마워요. / 감사해요!
AI: 잘 나왔어요! 예뻐요.
User: 또 올게요. / 다음에 또 봐요!
AI: 꼭 오세요! 기다릴게요.

# KP-2: 콘서트

## sim1: 콘서트 전

User: 콘서트 기대할게요. / 많이 기다렸어요!
AI: 잘 할게요! 재밌게 봐요.
User: 응원해요. / 화이팅!
AI: 고마워요! 힘이 나요.
User: 어떤 곡 해요? / 무슨 노래 해요?
AI: 비밀이에요. 조금만 기다려요!
User: 좋아요. / 빨리 보고 싶어요!
AI: 네, 재미있게 보세요! 고마워요.
## sim2: 콘서트 중

User: 여기요! / 여기 봐요!
AI: 보여요! 아주 잘 보여요.
User: 최고예요! / 너무 좋아요!
AI: 고마워요! 더 열심히 할게요.
User: 멋있어요! / 힘내요!
AI: 네, 약속해요!
User: 사랑해요. / 최고예요!
AI: 저도요! 진짜 사랑해요.
## sim3: 콘서트 후

User: 최고였어요. / 너무 좋았어요!
AI: 다행이에요! 오늘 즐거웠어요?
User: 네, 진짜 즐거웠어요! / 네, 재미있었어요!
AI: 저도 즐거웠어요. 다음에 또 와요!
User: 네, 또 올게요. / 꼭 올게요!
AI: 꼭이에요! 기다릴게요.
User: 잘 쉬어요!
AI: 네! 조심히 가세요.

# KP-3: 라이브 방송

## sim1: 라이브 시작

User: 왔어요! / 안녕하세요!
AI: 어서 오세요! 반가워요.
User: 기다렸어요. / 보고 싶었어요!
AI: 저도 기다렸어요!
User: 밥 먹었어요? / 오늘 뭐 했어요?
AI: 저는 잘 지냈어요! 여러분은요?
User: 저도 잘 지냈어요! / 아주 좋아요!
AI: 다행이에요. 오늘 기분 최고예요!
## sim2: 질문 시간

User: 요즘 어때요? / 요즘 바빠요?
AI: 바빠요. 그래도 재밌어요!
User: 안 피곤해요?
AI: 괜찮아요! 하나도 안 피곤해요.
User: 밥 먹었어요?
AI: 아직 안 먹었어요. 배고파요!
User: 밥 먹어요. / 빨리 드세요!
AI: 이따가 먹을게요. 고마워요!
## sim3: 라이브 마무리

User: 벌써 끝나요?
AI: 네, 이제 끝나요. 아쉬워요.
User: 아쉬워요. / 더 하고 싶어요!
AI: 저도요! 우리 다음에 또 해요.
User: 다음에 또 해요. / 또 해줘요!
AI: 네, 약속해요!
User: 잘 쉬어요. / 푹 쉬어요!
AI: 고마워요! 잘 자요.
