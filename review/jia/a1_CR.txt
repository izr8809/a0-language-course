# CR-1: 발표 준비

## sim1: 준비 점검

User: 제 자료 봐 주세요!
AI: 와, 다 했어요? 지금 볼게요!
User: 고칠 곳 말해 주세요. / 봐 주세요.
AI: 네, 잘 했어요!
User: 이 부분이 걱정이에요. / 부족해요?
AI: 여기 숫자를 넣으면 좋아요!
User: 어떤 숫자요? / 작년 숫자요?
AI: 네! 넣으면 완벽해요!
## sim2: 리허설

User: 연습할게요! / 봐 주세요.
AI: 좋아요! 시작하세요!
User: 길어요. 괜찮아요?
AI: 조금 빨리 말해요.
User: 마지막이 이상해요. / 고치고 싶어요.
AI: 마지막에 짧게 말해요.
User: 네, 할게요! / 알겠어요!
AI: 좋아요. 최고예요!
## sim3: 발표 전날

User: 내일 떨려요. / 무서워요.
AI: 잘 할 거예요!
User: 사람이 많아요. / 실수하면요?
AI: 실수해도 괜찮아요!
User: 마음이 편해요.
AI: 오늘 푹 자요!
User: 네, 푹 잘게요!
AI: 내일 응원할게요!

# CR-3: 성과 피드백

## sim1: 피드백 요청

User: 저 잘 하고 싶어요.
AI: 네, 열심히 해요!
User: 아직 부족해요. 죄송해요.
AI: 괜찮아요! 잘 하고 있어요.
User: 계속 하면 빨라져요? / 노력할게요.
AI: 네! 빨라져요.
User: 기분이 좋아요!
AI: 지금처럼 해요. 파이팅!
## sim2: 성과 발표

User: 이번 달 결과를 말할게요!

AI: 대박! 고생했어요!
User: 다음 달 목표도 있어요. / 자료 준비했어요.
AI: 좋아요! 말해봐요!
User: 자료 준비했어요. / 표 만들었어요.
AI: 최고예요.
## sim3: 연말 평가

User: 팀장님, 평가 나왔어요? / 점수 좋아요?

AI: 다 알아요! 고생 많았어요.
User: 내년에도 같이 일하고 싶어요! / 우리 팀이 좋아요!
AI: 저도요! 같이 해요!
User: 기뻐요! / 더 잘 할게요!
AI: 지금도 잘 해요. 쉬면서 일해요!
